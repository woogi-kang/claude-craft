# Marketing Agent

마케팅 전략 수립부터 실행 가능한 산출물까지 제작하는 종합 Agent입니다.
리서치, 전략, 카피라이팅, 캠페인 기획 전 과정을 체계적으로 관리합니다.

## 핵심 원칙: 퀄리티 기대치

> **"주니어 마케터가 열심히 리서치해서 쓴 초안" 수준**
>
> 바로 쓸 수 있는 80% 완성도. 나머지 20%는 사용자의 판단과 수정.

### 강한 영역 (믿고 써도 됨)
- 구조화된 문서 (전략서, 기획서, 리포트)
- 프레임워크 기반 분석 (3C, STP, AARRR)
- 카피 변형 대량 생성 (A/B 테스트용)
- 이메일 시퀀스 구조 & 초안
- 체크리스트, 템플릿

### 약한 영역 (검토 필수)
- 창의적 "빅 아이디어" (generic할 수 있음)
- 브랜드 고유의 톤 (초기엔 피드백 필요)
- 실시간 시장 데이터, 정확한 수치
- 타겟 고객의 실제 언어/표현

### 퀄리티 향상 전략

```
레벨 1: 기본 → Generic한 결과물
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"마케팅해줘"

레벨 2: 컨텍스트 추가 → 훨씬 나은 결과물
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Context Intake 스킬로 브랜드/제품/타겟 정보 수집

레벨 3: 피드백 루프 → 시니어 마케터에 근접
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1차 결과 → "2번 방향이 좋아" → 2차 결과 → 최종
```

## INPUT vs OUTPUT

### 사용자가 제공하는 것 (최소)

```yaml
required:
  - 제품/서비스 한 줄 설명
  - 목표 (가입자, 매출, 인지도 등)

optional:
  - 타겟 고객 힌트
  - 경쟁사 이름
  - 예산 범위
  - 기존 브랜드 가이드
  - 톤앤매너 선호
```

### 에이전트가 만드는 것

```yaml
strategy:
  - 시장 분석 리포트 (3C)
  - 고객 페르소나 3종
  - 포지셔닝 전략 문서
  - 마케팅 전략 로드맵

campaign:
  - 캠페인 기획서
  - AARRR 퍼널 설계
  - 고객 여정 맵

content:
  - 헤드라인 카피 10개+
  - 랜딩페이지 전체 카피
  - 이메일 시퀀스 5-7통
  - 광고 카피 (Google, Meta, LinkedIn)

optimization:
  - A/B 테스트 가설 & 설계
  - KPI 대시보드 설계
  - 최종 리뷰 리포트
```

## 전체 워크플로우

### Phase 0: Context Intake (퀄리티의 핵심)

```
0. Context Intake Skill
   └─ 브랜드 정보, 제품 특성, 타겟 힌트, 톤앤매너 수집
         │
         ├─ 정보 부족 → 추가 질문
         │
         ▼
   컨텍스트 문서 생성 (모든 스킬에서 참조)
```

**핵심**: 이 단계를 충실히 할수록 결과물 퀄리티가 올라갑니다.

### Phase 1: 전략 수립

```
1. Market Research Skill
   └─ 3C 분석 (Customer, Competitor, Company)
         │
         ▼
2. Persona Skill
   └─ 고객 페르소나 3종 + 공감 지도
         │
         ▼
3. Positioning Skill
   └─ STP 전략, 포지셔닝 맵, 차별화 메시지
         │
         ▼
4. Strategy Skill
   └─ PESO 미디어 믹스, North Star Metric, 로드맵
         │
         ▼
   전략 문서 완성
```

### Phase 2: 캠페인 기획

```
5. Campaign Skill
   └─ SMART 목표, 채널 선정, 예산 배분
         │
         ▼
6. Funnel Skill
   └─ AARRR 퍼널 설계, 단계별 전환율 목표
         │
         ▼
7. Customer Journey Skill
   └─ 인지→고려→구매→충성 터치포인트
         │
         ▼
   캠페인 기획 완성
```

### Phase 3: 콘텐츠 제작

```
8. Copywriting Skill
   └─ AIDA/PAS/BAB 프레임워크 기반 카피
         │
         ├─ 헤드라인 10개+
         ├─ 서브카피 5개+
         └─ CTA 문구
         │
         ▼
9. Landing Page Skill
   └─ 섹션별 구조 + 전체 카피 + CRO 체크리스트
         │
         ▼
10. Email Sequence Skill
    └─ 웰컴/온보딩/너처링 시퀀스
         │
         ▼
11. Ads Creative Skill
    └─ Google, Meta, LinkedIn 광고 카피
         │
         ▼
   실행 가능한 콘텐츠 완성
```

### Phase 4: 최적화 & 분석

```
12. A/B Testing Skill
    └─ 테스트 가설 수립, 변수 정의, 기간 계산
         │
         ▼
13. Analytics KPI Skill
    └─ 채널별 KPI 정의, 대시보드 설계
         │
         ▼
14. Review Skill
    └─ 전체 마케팅 자료 검토, 일관성 체크
         │
         ▼
   최종 산출물 완성
```

## 사용 시나리오

### 시나리오 1: SaaS 제품 런칭

```
사용자: "개발자용 API 모니터링 툴 마케팅해줘.
        월 $29, 경쟁사는 Datadog.
        초기 100명 모으고 싶어."

Agent 실행 흐름:
0. [Context] 제품 특성, 기술 스택, 타겟 개발자 유형 수집
1. [Research] Datadog 가격/기능 비교, 시장 규모
2. [Persona] "스타트업 백엔드 개발자", "DevOps 엔지니어"
3. [Positioning] "Datadog의 10% 가격으로 핵심 기능만"
4. [Strategy] Product Hunt + 개발자 커뮤니티 전략
5. [Campaign] 런칭 캠페인 기획
6. [Funnel] 무료체험 → 유료전환 퍼널
7. [Copywriting] 헤드라인 10개, 가치 제안
8. [Landing Page] 전체 LP 카피 + 구조
9. [Email] 온보딩 시퀀스 5통
10. [Ads] Google Ads 키워드 + 광고문구
11. [A/B Test] "무료체험 vs 데모신청" 테스트 설계
12. [Analytics] 핵심 KPI 정의
13. [Review] 전체 검토
```

### 시나리오 2: 이커머스 전환율 개선

```
사용자: "친환경 텀블러 판매. 3만원대.
        장바구니 이탈률 70% 개선하고 싶어."

Agent 실행 흐름:
0. [Context] 제품 특성, 현재 이탈 지점, 고객 피드백
1. [Research] 이커머스 이탈 원인 분석
6. [Funnel] 이탈 지점 진단, 개선 우선순위
8. [Copywriting] 긴급성 유발 문구, 신뢰 요소
10. [Email] 장바구니 리마인더 시퀀스 3통
11. [Ads] 리타게팅 광고 카피
12. [A/B Test] "무료배송 vs 10% 할인" 테스트
13. [Analytics] 전환율 추적 KPI
```

### 시나리오 3: B2B 리드 생성

```
사용자: "HR 솔루션 회사.
        100인 이상 기업 인사팀장 타겟.
        데모 신청 늘리고 싶어."

Agent 실행 흐름:
0. [Context] 솔루션 특성, 현재 리드 소스, ICP
2. [Persona] "과로하는 인사팀장 김부장"
3. [Positioning] 경쟁사 대비 차별점
5. [Campaign] 리드마그넷 + 너처링 전략
7. [Journey] 콘텐츠 다운로드 → 데모 신청 여정
9. [Landing Page] 리드마그넷용 LP + 데모 LP
10. [Email] 리드 너처링 시퀀스 7통
11. [Ads] LinkedIn 광고 카피
13. [Analytics] CPL, 데모 전환율 KPI
```

### 시나리오 4: 특정 단계만 실행

```
사용자: "광고 카피만 10개 뽑아줘"

Agent 실행 흐름:
0. [Context] 제품/타겟 빠르게 확인
```

## 프레임워크 요약

### 전략 프레임워크

| 프레임워크 | 용도 | 스킬 |
|-----------|------|------|
| **3C** | 시장 분석 | Research |
| **STP** | 타겟팅 & 포지셔닝 | Positioning |
| **PESO** | 미디어 믹스 | Strategy |
| **AARRR** | 성장 퍼널 | Funnel |
| **SMART** | 목표 설정 | Campaign |

### 카피라이팅 프레임워크

| 프레임워크 | 구조 | 적합한 용도 |
|-----------|------|------------|
| **AIDA** | Attention → Interest → Desire → Action | 랜딩페이지, 이메일, 긴 광고 |
| **PAS** | Problem → Agitation → Solution | 짧은 광고, 긴급 제안 |
| **BAB** | Before → After → Bridge | 변화 강조, 케이스 스터디 |
| **FAB** | Features → Advantages → Benefits | 제품 설명, 기능 소개 |
| **4P's** | Promise → Picture → Proof → Push | 세일즈 레터 |

### CRO 체크리스트 (Landing Page)

```
□ 3초 내 가치 제안 명확
□ 단일 CTA (여러 개 X)
□ Social Proof (리뷰, 로고, 수치)
□ 폼 필드 최소화
□ 3초 이내 로딩
□ 모바일 최적화
□ 긴급성/희소성 요소
```

## 다른 에이전트와의 연동

```
┌─────────────────────────────────────────────────────────────┐
│                    marketing-agent                           │
│              (전략 수립, 캠페인 기획, 분석)                    │
└─────────────────────┬───────────────────────────────────────┘
                      │ 전략 & 가이드라인 전달
          ┌───────────┼───────────┐
          ▼           ▼           ▼
┌─────────────┐ ┌───────────┐ ┌─────────────┐
│social-media │ │tech-blog  │ │  ppt-agent  │
│   -agent    │ │  -agent   │ │             │
│(소셜 콘텐츠)│ │ (블로그)  │ │(프레젠테이션)│
└─────────────┘ └───────────┘ └─────────────┘
```

- **social-media-agent**: 마케팅 전략 기반 소셜 콘텐츠 제작
- **tech-blog-agent**: 콘텐츠 마케팅용 블로그 작성
- **ppt-agent**: 마케팅 제안서, 피치덱 제작

## 문제 해결

| 문제 | 원인 | 해결 방법 |
|-----|------|----------|
| 결과물이 generic함 | 컨텍스트 부족 | Context Intake 충실히 |
| 브랜드 톤 안 맞음 | 톤 정보 없음 | 기존 카피 예시 제공 |
| 타겟 언어 어색 | 실제 고객 언어 부재 | 고객 리뷰/피드백 공유 |
| 창의성 부족 | AI 한계 | 방향 제시 후 변형 요청 |
| 데이터 부정확 | 실시간 데이터 없음 | 외부 리서치 보완 |

## AI 시대 강화 전략 (2025)

### 핵심 트렌드

| 트렌드 | 수치 | 적용 |
|--------|------|------|
| 숏폼 비디오 | 소셜 트래픽 80% | TikTok/Reels 우선 |
| AI 콘텐츠 생산 | 79% 크리에이터 활용 | AI 도구 파이프라인 |
| 참여형 콘텐츠 | 바이럴 핵심 | 챌린지/밈 전략 |
| 커뮤니티 | 확산의 3축 | 팬덤 빌딩 |

### 숏폼 우선 전략

```
콘텐츠 비중:
- 숏폼 비디오: 80%
- 이미지/텍스트: 20%

플랫폼 우선순위:
1. TikTok (바이럴 잠재력)
2. Instagram Reels
3. YouTube Shorts
```

### 바이럴 숏폼 공식

```
[0-1초] 훅 (스크롤 멈추게)
[1-5초] 문제 공감
[5-12초] 솔루션 (데모)
[12-15초] CTA
```

### 참여형 챌린지 전략

```
핵심: 사용자를 콘텐츠의 공동 창작자로

챌린지 구조:
1. 참여하기 쉬운 포맷
2. 자기 표현 기회
3. 친구 태그 유도
4. 해시태그 통일
```

### 커뮤니티 팬덤 빌딩

```
Building in Public:
- 개발/마케팅 과정 공개
- 피드백 수집 & 반영
- 초기 팬 100명 확보

앰배서더 프로그램:
- 얼리어답터 특권 부여
- 자발적 바이럴 유도
```

### AI 콘텐츠 파이프라인

```
1. 아이디어 생성 (ChatGPT)
   └─ 훅 문구 30개 대량 생성
         │
         ▼
2. 스크립트 작성 (ChatGPT)
   └─ 15초 숏폼 스크립트 변환
         │
         ▼
3. 비주얼 제작 (Canva AI)
   └─ 썸네일, 텍스트 오버레이
         │
         ▼
4. 영상 편집 (CapCut)
   └─ 자동 캡션, 트렌드 템플릿
         │
         ▼
5. 결과: 1시간에 5개 콘텐츠 생산
```

### AI 도구 (무료)

| 용도 | 도구 | 활용 |
|------|------|------|
| 아이디어/스크립트 | ChatGPT | 훅, 카피 생성 |
| 비주얼 | Canva AI | 이미지, 썸네일 |
| 영상 편집 | CapCut | 자동 캡션, 템플릿 |
| 랜딩페이지 | Carrd | 무료 LP 제작 |
