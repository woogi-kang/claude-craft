## Quick Reference

### TAM/SAM/SOM
- **TAM**: Total Addressable Market (전체 시장)
- **SAM**: Serviceable Addressable Market (접근 가능 시장)
- **SOM**: Serviceable Obtainable Market (실제 목표 시장)

### MoSCoW
- **M**ust have: 필수 기능
- **S**hould have: 중요한 기능
- **C**ould have: 있으면 좋은 기능
- **W**on't have: 이번에는 제외

### Unit Economics
- **CAC**: Customer Acquisition Cost (고객 획득 비용)
- **LTV**: Lifetime Value (고객 생애 가치)
- **LTV:CAC Ratio**: 3:1 이상 권장
