## Framework Selection Guide

```
문제 정의 필요 → Problem-Solution Fit
시장 규모 필요 → TAM/SAM/SOM
경쟁 분석 필요 → Feature Matrix + Positioning Map
비즈니스 검증 → Lean Canvas + Unit Economics
기능 우선순위 → MoSCoW
목표 설정 필요 → OKR + North Star Metric
```
