## Search Strategy

To find procedures for a Japanese concern:

1. Match the user's input against BOTH formal and casual variants in the concern table above
   - "毛穴やばい" → 毛穴 → tag_id 3
   - "ほうれい線消したい" → ほうれい線 → tag_id 16
   - "フェイスラインやばい" → たるみ → tag_id 2
2. Search `dermatology_procedures.json` for procedures containing that tag_id in their `tag_ids` array
3. Sort results by `grade` (1 first = most popular)
4. Fetch top 3-5 procedure details from `dermatology_procedure_details_complete.json`
5. Present in Japanese with treatment-translation.md names
6. Use casual tone (常体 base) matching the @ask.nandemo voice
