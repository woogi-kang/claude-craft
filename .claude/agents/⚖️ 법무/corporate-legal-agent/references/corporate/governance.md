# Corporate Governance Reference

## 기관 구조

```yaml
corporate_structure:
  주주총회:
    role: "최고의사결정기관"
    exclusive_matters:
      - "정관 변경"
      - "이사/감사 선임/해임"
      - "재무제표 승인"
      - "합병/분할"
      - "해산"

  이사회:
    role: "업무집행 의사결정"
    composition: "이사 3인 이상"
    optional: "자본금 10억 미만 + 이사 2인 이하"

  대표이사:
    role: "회사 대표, 업무 집행"
    selection: "이사회 결의"

  감사:
    role: "이사 직무 감시"
    optional: "자본금 10억 미만"
```

## 의결 요건 매트릭스

| 사항 | 결정기관 | 정족수 | 의결 |
|------|----------|--------|------|
| 재무제표 승인 | 주주총회 | 1/4 출석 | 과반수 |
| 이사 선임 | 주주총회 | 1/4 출석 | 과반수 |
| 정관 변경 | 주주총회 | 1/3 출석 | 2/3 이상 |
| 합병/분할 | 주주총회 | 1/3 출석 | 2/3 이상 |
| 이사 해임 | 주주총회 | - | 발행주식 2/3 |
| 대표이사 선임 | 이사회 | 과반수 출석 | 과반수 |
| 신주 발행 | 이사회 | 과반수 출석 | 과반수 |

## 임원 임기

```yaml
officer_terms:
  이사:
    maximum: "3년"
    typical: "정관으로 정함"
    renewal: "주주총회 선임"

  감사:
    maximum: "3년"
    typical: "정관으로 정함"
    renewal: "주주총회 선임"

  대표이사:
    term: "이사 임기 내"
    note: "이사 자격 상실 시 대표이사 자격도 상실"
```

## 등기사항

```yaml
registration_matters:
  필수_등기:
    - "상호"
    - "본점 소재지"
    - "목적"
    - "자본금"
    - "발행주식 총수"
    - "이사 성명"
    - "대표이사 성명/주소"
    - "감사 성명 (설치 시)"

  변경_등기:
    deadline: "14일 이내"
    penalty: "최대 100만원 과태료"
```
