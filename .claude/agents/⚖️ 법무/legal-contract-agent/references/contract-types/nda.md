# NDA (비밀유지계약) Reference

## 개요

NDA(Non-Disclosure Agreement)는 비밀정보의 보호를 위한 계약으로, 사업 협력, 투자 논의, 기술 교류 등의 상황에서 활용됩니다.

---

## NDA 유형

| 유형 | 설명 | 사용 상황 |
|------|------|----------|
| **일방향 NDA** | 한 쪽만 정보 공개 | 투자 피칭, 기술 판매 |
| **상호 NDA** | 양쪽 모두 정보 공개 | 합작투자, 기술 제휴 |
| **다자간 NDA** | 3개 이상 당사자 | 컨소시엄, 다자간 협력 |

---

## 핵심 검토 항목

### 1. 비밀정보 정의 (Definition of Confidential Information)

```
체크포인트:
□ 비밀정보 범위가 명확한가?
□ 구두 정보 포함 여부 및 확인 절차
□ 파생 정보(derivative information) 포함 여부
□ 비밀 표시(marking) 요건

위험 신호:
- "모든 정보" 포괄적 정의 → 범위 제한 필요
- 구두 정보 확인 기간 없음 → 30일 이내 서면 확인 권장
```

### 2. 예외 조항 (Exceptions)

```
표준 예외 사항:
- 공개 정보 (publicly available)
- 사전 보유 정보 (prior knowledge)
- 독자 개발 정보 (independently developed)
- 제3자로부터 적법하게 취득한 정보
- 법적 공개 의무

체크포인트:
□ 표준 예외 조항이 모두 포함되어 있는가?
□ 역설계(reverse engineering) 예외 여부
□ 법적 공개 시 사전 통지 의무
```

### 3. 의무 기간 (Duration of Obligations)

```
일반적 기준:
- 영업비밀: 영구 또는 비밀성 상실 시까지
- 일반 비밀정보: 3-5년

체크포인트:
□ 계약 기간과 의무 기간 구분
□ 영업비밀에 대한 별도 규정
□ 계약 종료 후 의무 존속 조항
```

### 4. 사용 제한 (Use Restrictions)

```
체크포인트:
□ 허용 목적이 명확히 정의되어 있는가?
□ 내부 공유 범위 (need-to-know basis)
□ 복제/복사 제한
□ 정보 반환/파기 의무
```

### 5. 손해배상 조항 (Remedies)

```
체크포인트:
□ 금전적 손해배상 한도
□ 금지명령(injunctive relief) 조항
□ 간접손해 배제 여부
□ 손해배상 산정 기준
```

## 협상 포지션별 전략

### 정보 제공자 (Discloser) - 강한 포지션

```
우선순위:
1. 비밀정보 범위 최대화
2. 의무 기간 장기화 (5년+)
3. 강력한 구제수단 확보
4. 잔존 지식(residual knowledge) 조항 제외

주장 포인트:
- 영업비밀은 영구 보호
- 파생 정보 포함
- 위반 시 즉시 금지명령 가능
```

### 정보 수령자 (Recipient) - 약한 포지션

```
우선순위:
1. 비밀정보 범위 제한
2. 의무 기간 합리화 (3년 이내)
3. 손해배상 한도 설정
4. 예외 조항 확대

주장 포인트:
- 비밀 표시된 정보만 해당
- 잔존 지식 조항 포함
- 간접손해 배제
```

## NDA 체크리스트

### 정보 제공자용

```
□ 비밀정보 정의가 충분히 포괄적인가?
□ 파생 정보가 포함되어 있는가?
□ 의무 기간이 적절한가 (최소 5년)?
□ 반환/파기 의무가 명확한가?
□ 금지명령 조항이 있는가?
□ 준거법/관할이 유리한가?
```

### 정보 수령자용

```
□ 비밀정보 범위가 합리적인가?
□ 표준 예외 조항이 모두 있는가?
□ 잔존 지식 조항이 있는가?
□ 의무 기간이 과도하지 않은가?
□ 손해배상 한도가 있는가?
□ 간접손해가 배제되어 있는가?
```

## 일반적인 위험 조항

| 조항 | 위험 수준 | 설명 |
|------|----------|------|
| 무제한 손해배상 | Critical | 손해배상 한도 협상 필요 |
| 5년 초과 의무기간 | High | 합리적 기간으로 조정 |
| 역설계 금지 | Medium | 업계 관행에 따라 판단 |
| 일방적 준거법 | Medium | 중립 관할권 협상 |
| 잔존지식 조항 없음 | Medium | 수령자 측에서 추가 요청 |

## 표준 문구 예시

### 비밀정보 정의

```
"비밀정보"란 공개 당사자가 수령 당사자에게 서면, 구두 또는
기타 방식으로 공개하는 모든 정보로서, 공개 시 "비밀" 또는
유사한 문구로 표시되거나, 구두 공개의 경우 공개 후 30일
이내에 서면으로 비밀임을 확인한 정보를 의미합니다.
```

### 예외 조항

```
다음 각 호에 해당하는 정보는 비밀정보에서 제외됩니다:
(a) 공개 시점에 이미 공개된 정보
(b) 수령자의 귀책 없이 공개된 정보
(c) 공개 이전에 수령자가 적법하게 보유한 정보
(d) 제3자로부터 비밀유지 의무 없이 적법하게 취득한 정보
(e) 수령자가 비밀정보를 사용하지 않고 독자적으로 개발한 정보
```
