# Content Quality Eval Preset

콘텐츠(블로그, 기획서, 마케팅 카피 등)의 품질을 평가하는 루브릭.
`eval-harness` 스킬 및 `review-orchestrator`에서 콘텐츠 리뷰 시 사용.

## 평가 축 (가중치)

### Clarity (30%)

명확하고 이해하기 쉬운가?

| 점수 | 기준 |
|------|------|
| 9-10 | 복잡한 개념도 쉽게 전달, 논리적 흐름 완벽 |
| 7-8 | 대부분 명확, 일부 문장 재구성 필요 |
| 5-6 | 핵심 메시지는 전달되나 구조 개선 필요 |
| 3-4 | 혼란스러운 구조, 핵심 메시지 불분명 |
| 1-2 | 이해 불가, 논리적 흐름 부재 |

### Completeness (25%)

필요한 정보가 모두 포함되었는가?

| 점수 | 기준 |
|------|------|
| 9-10 | 모든 관점 커버, 예상 질문에 선제 답변 |
| 7-8 | 핵심 내용 포함, 사소한 누락 1-2곳 |
| 5-6 | 기본 내용은 있으나 깊이 부족 |
| 3-4 | 중요한 섹션 누락 |
| 1-2 | 초안 수준, 대부분 미완성 |

### Accuracy (25%)

사실에 기반하고 기술적으로 정확한가?

| 점수 | 기준 |
|------|------|
| 9-10 | 모든 주장에 근거, 최신 정보, 기술적 정확 |
| 7-8 | 대부분 정확, 사소한 부정확 1-2곳 |
| 5-6 | 핵심은 맞으나 세부 사항 부정확 |
| 3-4 | 오해를 유발할 수 있는 부정확한 내용 |
| 1-2 | 사실과 다른 주장, 오래된 정보 |

### Engagement (20%)

독자의 관심을 끌고 유지하는가?

| 점수 | 기준 |
|------|------|
| 9-10 | 강력한 도입부, 독자 참여 유도, 기억에 남는 마무리 |
| 7-8 | 흥미로운 전개, 적절한 사례/비유 |
| 5-6 | 읽을 만하지만 특별히 끌리지 않음 |
| 3-4 | 건조하고 기계적, AI 생성 느낌 |
| 1-2 | 지루하거나 반복적 |

## 합격 기준

```
가중 평균 = (Clarity × 0.30) + (Completeness × 0.25) + (Accuracy × 0.25) + (Engagement × 0.20)

PASS: 가중 평균 ≥ 7.0
FAIL: Accuracy < 5 (사실 오류)
FAIL: Clarity < 4 (이해 불가)
```

## 출력 형식

```markdown
## Content Quality Evaluation

| Axis | Score | Weight | Weighted |
|------|-------|--------|----------|
| Clarity | 8 | 30% | 2.40 |
| Completeness | 7 | 25% | 1.75 |
| Accuracy | 9 | 25% | 2.25 |
| Engagement | 6 | 20% | 1.20 |
| **Total** | | | **7.60** |

**Result**: PASS
```
