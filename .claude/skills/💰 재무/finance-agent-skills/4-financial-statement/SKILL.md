# Financial Statement Skill

월간/분기/연간 재무제표를 자동 생성하는 스킬입니다.

## 핵심 원칙

- **정확성**: 원장 데이터 기반 자동 집계
- **표준 준수**: DART 공시 형식 참조
- **시각화**: 차트 포함 PDF 생성

## 주요 지표 자동 계산

```yaml
metrics:
  profitability:
    - gross_margin: (매출총이익 / 매출액) * 100
    - operating_margin: (영업이익 / 매출액) * 100
    - net_margin: (당기순이익 / 매출액) * 100

  liquidity:
    - current_ratio: 유동자산 / 유동부채
    - quick_ratio: (유동자산 - 재고자산) / 유동부채

  efficiency:
    - receivables_turnover: 매출액 / 매출채권
    - payables_turnover: 매출원가 / 매입채무

  growth:
    - revenue_growth: (당월매출 - 전월매출) / 전월매출 * 100
    - profit_growth: (당월순이익 - 전월순이익) / 전월순이익 * 100
```

## 품질 체크

```yaml
validation_rules:
  # 기본 검증
  - 차변총액 == 대변총액
  - 자산 == 부채 + 자본
  - 기말현금 == 기초현금 + 현금증감

  # 논리 검증
  - 매출총이익 <= 매출액
  - 영업이익 <= 매출총이익
  - 당기순이익 <= 영업이익 + 영업외수익

  # 이상치 검증
  - 마진율 급변 (±20% 이상) → 경고
  - 비용 카테고리 급변 (±50% 이상) → 검토 요청
```
