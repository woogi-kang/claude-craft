# Logo Styles Quick Reference

배치 생성 시 사용되는 20가지 스타일과 프롬프트 패턴.

## Style Categories

### 1. Minimalist
심플한 기하학 도형, 여백 활용. **적합:** 테크, SaaS, 전문직
```
minimalist {subject} logo, simple geometric shapes, clean lines, white space, single color
```

### 2. Modern
그라데이션, 세련된 곡선. **적합:** 스타트업, 앱
```
modern {subject} logo, sleek gradient, tech-forward, smooth curves, innovative
```

### 3. Geometric
정밀한 기하학 패턴. **적합:** 건축, 테크
```
geometric {subject} logo, abstract patterns, mathematical precision, symmetrical
```

### 4. Gradient
컬러 트랜지션. **적합:** 디지털, 앱
```
gradient {subject} logo, vibrant color transitions, modern digital feel, smooth flow
```

### 5. Abstract
비구상, 상징적 표현. **적합:** 컨설팅, 대기업
```
abstract {subject} logo, conceptual, symbolic, non-literal, artistic interpretation
```

### 6. Lettermark
이니셜/모노그램. **적합:** 개인 브랜드, 럭셔리
```
letter "{letter}" logo, monogram, typographic, distinctive character, clean
```

### 7. Negative Space
네거티브 스페이스 활용. **적합:** 크리에이티브
```
negative space {subject} logo, clever white space, hidden meaning, dual imagery
```

### 8. Line Art
단일 선, 연속. **적합:** 미니멀, 아트
```
line art {subject} logo, single stroke, continuous line, elegant simplicity
```

### 9. 3D / Isometric
입체감, 깊이. **적합:** 게임, 테크
```
3D {subject} logo, dimensional, depth, soft shadows, isometric perspective
```

### 10. Vintage / Retro
레트로 뱃지 스타일. **적합:** 카페, 수제 브랜드
```
vintage {subject} logo, retro badge, heritage feel, warm earth tones, distressed
```

### 11. Emblem
문장, 실드 형태. **적합:** 스포츠, 전통
```
{subject} emblem logo, badge, crest, enclosed design, traditional, authoritative
```

### 12. Mascot
캐릭터, 마스코트. **적합:** 푸드, 게임, 키즈
```
cute {character} mascot logo, friendly, cartoon, big eyes, memorable
```

### 13. Hand-drawn
손그림, 스케치. **적합:** 아티산, 크리에이티브
```
hand-drawn {subject} logo, sketch style, artistic strokes, organic lines, ink
```

### 14. Luxury
고급, 골드 악센트. **적합:** 패션, 주얼리
```
luxury {subject} logo, elegant, gold accents, refined, premium, serif typography
```

### 15. Flat
플랫 디자인, 단색. **적합:** 앱, 웹
```
flat design {subject} logo, solid colors, no gradients, clean sharp edges
```

### 16. Pixel Art
8비트 레트로. **적합:** 게임, 인디
```
pixel art {subject} logo, 8-bit retro, blocky pixels, sharp edges, no anti-aliasing
```

### 17. Monoline
단일 선 두께. **적합:** 모던, 미니멀
```
monoline {subject} logo, single weight stroke, continuous path, uniform thickness
```

### 18. Wordmark
브랜드명 타이포. **적합:** 미디어, 패션
```
"{name}" wordmark logo, custom typography, distinctive lettering, brand name
```

### 19. Circular
원형 구도. **적합:** 스탬프, 뱃지
```
circular {subject} logo, round badge, contained within circle, balanced
```

### 20. Organic
자연스러운 곡선. **적합:** 에코, 헬스
```
organic {subject} logo, natural flowing lines, leaf elements, earthy tones
```

## Color Psychology

| 컬러 | 의미 | 업종 |
|------|------|------|
| Blue | 신뢰, 전문성 | 금융, 테크, 헬스케어 |
| Green | 성장, 자연 | 에코, 헬스, 교육 |
| Red | 에너지, 열정 | 푸드, 엔터, 스포츠 |
| Orange | 창의, 친근 | 크리에이티브, 키즈 |
| Purple | 럭셔리, 창의 | 패션, 뷰티 |
| Yellow | 낙관, 따뜻함 | 푸드, 교육 |
| Black | 프리미엄, 우아 | 럭셔리, 패션 |
| Gold | 고급, 전통 | 주얼리, 금융 |
