# 대한민국 근로기준법 - 근로시간/휴가 관련 조항

> 최종 조사일: 2026-03-24
> 근거법령: 근로기준법 (법률 제20520호, 2025.2.23. 시행)

---

## 1. 법정 근로시간 (제50조)

### 조문 내용
> 1주 간의 근로시간은 휴게시간을 제외하고 40시간을 초과할 수 없다.
>
> 1일의 근로시간은 휴게시간을 제외하고 8시간을 초과할 수 없다.

### 핵심 요소
| 구분 | 기준 |
|------|------|
| 1주 법정 근로시간 | 40시간 (휴게시간 제외) |
| 1일 법정 근로시간 | 8시간 (휴게시간 제외) |
| "1주"의 정의 | 휴일을 포함한 7일 (2018년 개정) |

### 실무적 의미
- 소정근로시간: 법정 근로시간 범위 내에서 근로자와 사용자가 합의한 근로시간
- 주 40시간을 초과하는 근로는 연장근로에 해당하여 가산임금 지급 필요
- 15세 이상 18세 미만 연소근로자: 1일 7시간, 1주 35시간 (제69조)
- 위반 시: 2년 이하 징역 또는 2천만원 이하 벌금 (제110조)

---

## 2. 연장근로의 제한 (제53조)

### 조문 내용
> 당사자 간에 합의하면 1주 간에 12시간을 한도로 제50조의 근로시간을 연장할 수 있다.

### 주 52시간제 구조
```
주 최대 근로시간 = 법정 근로시간(40시간) + 연장근로 한도(12시간) = 52시간
```

### 적용 범위별 시행일
| 사업장 규모 | 시행일 |
|------------|--------|
| 300인 이상 | 2018.7.1 |
| 50~299인 | 2020.1.1 |
| 5~49인 | 2021.7.1 |

### 특별연장근로 (제53조 제4항)
- 특별한 사정이 있으면 고용노동부장관의 인가와 근로자 동의를 받아 추가 연장 가능
- 사유: 재난/재해, 인명 보호, 시설 긴급 보수 등
- 1주 간 8시간을 초과하지 못함 (인가 기간: 최대 90일 이내, 연장 가능)

### 2023년 대법원 판결 (중요 변경)
- 연장근로 한도 위반 여부는 **1주 단위**로 판단
- 1일 8시간 초과 여부가 아닌, 1주 40시간 초과 합계가 12시간을 넘는지로 판단
- 고용노동부가 이에 맞춰 행정해석을 변경함

### 위반 시 벌칙
- 2년 이하 징역 또는 2천만원 이하 벌금 (제110조)

---

## 3. 휴게시간 (제54조)

### 조문 내용
> 사용자는 근로시간이 4시간인 경우에는 30분 이상, 8시간인 경우에는 1시간 이상의 휴게시간을 근로시간 도중에 주어야 한다.
>
> 휴게시간은 근로자가 자유롭게 이용할 수 있다.

### 핵심 원칙
| 근로시간 | 최소 휴게시간 |
|---------|-------------|
| 4시간 | 30분 이상 |
| 8시간 | 1시간 이상 |

### 실무적 의미
- 휴게시간은 근로시간에 포함되지 않음 (무급)
- "근로시간 도중에" 부여해야 함 (출퇴근 전후 불가)
- 근로자가 자유롭게 이용할 수 있어야 함 (대기시간과 구별)
- 일제히 부여 원칙, 다만 업무 특성상 개별 부여도 가능
- 실질적으로 사용자의 지휘감독 하에 있으면 근로시간으로 인정

---

## 4. 휴일 (제55조)

### 조문 내용
> 사용자는 근로자에게 1주에 평균 1회 이상의 유급휴일을 보장하여야 한다.
>
> 사용자는 근로자에게 대통령령으로 정하는 휴일을 유급으로 보장하여야 한다.

### 주휴일 (제55조 제1항)

#### 주휴일 부여 요건
- 1주 동안의 소정근로일을 개근한 근로자
- 1주 소정근로시간이 15시간 이상인 근로자

#### 주휴수당 계산
```
주휴수당 = 1일 소정근로시간 x 시간급 통상임금

예시: 주 40시간(1일 8시간) 근무, 시급 10,000원
주휴수당 = 8시간 x 10,000원 = 80,000원
```

### 관공서 공휴일 (제55조 제2항, 시행령 제30조)
- 2022년부터 5인 이상 사업장에 적용 확대 완료
- 관공서의 공휴일에 관한 규정에 따른 공휴일
- 대체공휴일 포함

### 근로자의 날 (5월 1일)
- 근로자의 날 제정에 관한 법률에 의한 법정 유급휴일
- 근로기준법상 휴일과 동일한 효력
- 근로 시 휴일근로 가산임금 지급 의무

## 5. 가산임금 (제56조) - 연장/야간/휴일 근로

### 조문 내용

#### 제56조 제1항 (연장근로)
사용자는 연장근로(제53조, 제59조 및 제69조 단서에 따라 연장된 시간의 근로)에 대하여는 통상임금의 100분의 50 이상을 가산하여 근로자에게 지급하여야 한다.

#### 제56조 제2항 (휴일근로)
사용자는 휴일근로에 대하여는 다음 각 호의 기준에 따른 금액 이상을 가산하여 근로자에게 지급하여야 한다.
1. 8시간 이내의 휴일근로: 통상임금의 100분의 50
2. 8시간을 초과한 휴일근로: 통상임금의 100분의 100

#### 제56조 제3항 (야간근로)
사용자는 야간근로(오후 10시부터 다음 날 오전 6시 사이의 근로)에 대하여는 통상임금의 100분의 50 이상을 가산하여 근로자에게 지급하여야 한다.

### 가산임금 요약표

| 근로 유형 | 가산율 | 산정 기준 |
|-----------|--------|-----------|
| 연장근로 (주 40시간 초과) | +50% | 통상임금 |
| 야간근로 (22시~06시) | +50% | 통상임금 |
| 휴일근로 (8시간 이내) | +50% | 통상임금 |
| 휴일근로 (8시간 초과) | +100% | 통상임금 |

### 중복 가산 예시
```
휴일 연장근로 (8시간 초과분): 통상임금 + 휴일가산(100%) + 연장가산(50%) = 250%
휴일 야간근로 (8시간 이내): 통상임금 + 휴일가산(50%) + 야간가산(50%) = 200%
연장 야간근로: 통상임금 + 연장가산(50%) + 야간가산(50%) = 200%
```

### 위반 시 벌칙
- 3년 이하 징역 또는 3천만원 이하 벌금 (제109조 제1항)
- 반의사불벌죄 (피해자가 처벌 불원 시 공소 불가)

## 6. 연차유급휴가 (제60조)

### 발생 요건 및 일수

#### 기본 연차 (제60조 제1항)
- **요건**: 1년간 80% 이상 출근
- **일수**: 15일의 유급휴가

#### 입사 1년 미만 / 80% 미만 출근자 (제60조 제2항)
- **요건**: 1개월 개근
- **일수**: 1일의 유급휴가 (최대 11일)

#### 장기근속 가산휴가 (제60조 제4항)
- **요건**: 3년 이상 계속 근로
- **가산**: 최초 1년 초과 매 2년마다 1일 가산
- **상한**: 가산휴가 포함 총 25일

### 연차 일수 계산표

| 근속연수 | 연차 일수 | 비고 |
|---------|----------|------|
| 1년 미만 | 최대 11일 | 매월 개근 시 1일씩 |
| 1년 | 15일 | 기본 연차 |
| 2년 | 15일 | |
| 3년 | 16일 | +1일 가산 시작 |
| 4년 | 16일 | |
| 5년 | 17일 | |
| 6년 | 17일 | |
| ... | ... | 매 2년마다 +1일 |
| 21년 | 25일 | 상한 도달 |
| 22년 이상 | 25일 | 상한 유지 |

### 출근으로 간주하는 기간 (제60조 제6항)
- 업무상 부상/질병으로 휴업한 기간
- 임신 중 여성의 출산전후휴가 기간 (제74조)
- 육아휴직 기간 (남녀고용평등법 제19조)

### 연차 사용 및 미사용 수당
- **시기지정권**: 근로자가 청구한 시기에 부여 원칙
- **시기변경권**: 사업 운영에 막대한 지장이 있는 경우 사용자가 시기 변경 가능
- **미사용 수당**: 연차 사용기간 내 미사용 시 통상임금 또는 평균임금으로 보상
- **소멸시효**: 연차휴가청구권 1년, 미사용 연차수당 청구권 3년

## 7. 연차유급휴가 사용촉진 (제61조)

### 조문 내용
사용자가 아래 절차를 모두 이행한 경우, 근로자가 미사용한 연차에 대해 보상 의무가 면제된다.

### 사용촉진 절차 (2단계)

#### 1단계: 미사용 휴가 통보 및 사용 촉구
- **시기**: 연차 사용기간 만료 6개월 전 기준, 10일 이내
- **내용**: 근로자별 미사용 휴가 일수를 알리고, 사용 시기를 정하여 통보하도록 서면 촉구
- **방법**: 반드시 서면으로 촉구

#### 2단계: 사용자의 시기 지정
- **시기**: 1단계 촉구 후 10일 이내에 근로자가 미통보 시
- **내용**: 사용자가 미사용 휴가의 사용 시기를 정하여 근로자에게 서면 통보
- **조건**: 근로자 대표와의 서면 합의가 있으면 변경 가능

### 실무적 의미
- 사용촉진 절차를 완료하면 미사용 연차수당 지급 의무가 면제됨
- 절차의 하자(서면 미비, 기한 미준수 등)가 있으면 촉진 효과 불인정
- 1년 미만 근로자의 연차(제60조 제2항)에 대해서도 별도 사용촉진 절차 적용 (제61조 제2항)

### 사용촉진 미이행 시
- 미사용 연차에 대한 수당 지급 의무 존속
- 미사용 연차수당 = 미사용 일수 x 1일 통상임금

## 8. 초과근무 수당 미지급 시 대응방법

### 초과근무 수당의 범위
- 연장근로수당 (주 40시간 초과분의 50% 가산)
- 야간근로수당 (22시~06시 근로의 50% 가산)
- 휴일근로수당 (8시간 이내 50%, 8시간 초과 100% 가산)

### 증거 확보 방법
| 증거 유형 | 확보 방법 |
|-----------|----------|
| 근로시간 기록 | 출퇴근 기록, 전자출입 로그, PC 로그온/오프 기록 |
| 업무 지시 기록 | 카카오톡/이메일 등 메신저 대화 캡처 |
| 급여명세서 | 매월 수령한 급여명세서 보관 |
| 근로계약서 | 소정근로시간, 임금 구성항목 확인 |
| 동료 증언 | 같은 부서 동료의 진술 |

### 단계별 대응

#### 1단계: 회사에 직접 청구
- 미지급 초과근무수당 내역 정리
- 서면(내용증명)으로 지급 요청
- 합리적 기한 설정 (통상 14일)

#### 2단계: 고용노동부 진정
- **신고처**: 사업장 소재지 관할 지방고용노동관서
- **신고 방법**: 방문, 우편, 온라인 (고용노동부 민원마당, 국번없이 1350)
- **처리**: 근로감독관 조사 -> 사실 확인 -> 시정 지시
- **주의**: 소멸시효 중단 효과 없음 (3년 시효 주의)

#### 3단계: 형사고소
- 가산임금 미지급은 근로기준법 제56조 위반
- 3년 이하 징역 또는 3천만원 이하 벌금 (제109조)
- 반의사불벌죄: 합의 시 처벌 불가 -> 합의 압박 수단으로 활용

#### 4단계: 민사소송
- **소액사건심판**: 3,000만원 이하 청구 시 (1회 변론, 신속 처리)
- **일반 민사소송**: 3,000만원 초과 시
- **청구 내용**: 미지급 수당 + 지연이자(연 20%) + 소송비용
- **시효**: 3년 (각 임금 지급기일부터 기산)

### 포괄임금제 관련 주의사항
- 포괄임금제라 하더라도 실제 연장근로시간이 약정 시간을 초과하면 추가 수당 청구 가능
- 근로시간 산정이 어려운 업무가 아닌 경우 포괄임금제 자체가 무효가 될 수 있음
- 대법원 판례: 포괄임금계약이 근로자에게 불이익하면 무효 (실제 근로시간 기준으로 재산정)

## 9. 탄력적 근로시간제 / 선택적 근로시간제 (참고)

### 탄력적 근로시간제 (제51조, 제51조의2)
| 유형 | 단위기간 | 요건 | 주 최대 근로시간 |
|------|---------|------|----------------|
| 2주 단위 | 2주 이내 | 취업규칙 | 48시간 |
| 3개월 단위 | 3개월 이내 | 근로자대표 서면합의 | 52시간 |
| 6개월 단위 | 3개월 초과 6개월 이내 | 근로자대표 서면합의 + 고용노동부 신고 | 52시간 |

### 선택적 근로시간제 (제52조)
- 1개월 이내 정산기간 설정
- 근로자가 근로시간 배치를 자율적으로 결정
- 정산기간 평균 주 40시간 이내

## 10. 5인 미만 사업장 적용 특례

5인 미만 사업장에는 근로기준법의 일부 조항이 적용되지 않는다.

| 조항 | 5인 미만 사업장 적용 여부 |
|------|------------------------|
| 법정 근로시간 (제50조) | 적용 제외 |
| 연장근로 제한 (제53조) | 적용 제외 |
| 연장/야간/휴일 가산임금 (제56조) | **적용 제외** |
| 연차유급휴가 (제60조) | 적용 제외 |
| 주휴일 (제55조 제1항) | **적용** |
| 임금 지급 원칙 (제43조) | **적용** |
| 해고 제한 (제23조) | 적용 제외 |
