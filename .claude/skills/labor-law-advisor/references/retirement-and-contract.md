# 퇴직금 및 근로계약 - 근로자퇴직급여보장법, 근로기준법

> 근거법률: 근로자퇴직급여 보장법 제4조, 제8조, 제9조, 제44조 / 근로기준법 제17조, 제20조, 제21조
> 최종 확인: 2026-03-24

---

## Part 1: 퇴직금

### 1. 퇴직급여제도의 설정

#### 제4조 (퇴직급여제도의 설정)

##### 제1항
사용자는 퇴직하는 근로자에게 급여를 지급하기 위하여 퇴직급여제도 중 하나 이상의 제도를 설정하여야 한다.

##### 제1항 단서 - 적용 제외
**4주간을 평균하여 1주간의 소정근로시간이 15시간 미만인 근로자**에 대하여는 적용하지 아니한다.

#### 퇴직금 지급요건 정리

| 요건 | 내용 | 비고 |
|------|------|------|
| **계속근로기간** | 1년 이상 | 입사일~퇴직일 기준 |
| **주당 소정근로시간** | 15시간 이상 (4주 평균) | 15시간 미만은 적용 제외 |
| **적용 대상** | 상시근로자 1인 이상 사업장 | 정규직, 계약직, 파트타임 불문 |

#### 실무적 의미
- **1년 미만 근로자**: 퇴직금 지급의무 없음 (단, 취업규칙·근로계약으로 지급 가능)
- **단시간 근로자**: 4주 평균 주 15시간 이상이면 퇴직금 대상
- **수습기간**: 계속근로기간에 포함
- **휴직기간**: 원칙적으로 계속근로기간에 포함 (개별 판단 필요)

### 2. 퇴직금 계산

#### 제8조 (퇴직금제도)

##### 제1항
퇴직금제도를 설정하려는 사용자는 계속근로기간 **1년에 대하여 30일분 이상의 평균임금**을 퇴직금으로 퇴직하는 근로자에게 지급할 수 있는 제도를 설정하여야 한다.

#### 퇴직금 계산 공식

```
퇴직금 = 1일 평균임금 x 30일 x (총 계속근로기간 / 365일)
```

#### 평균임금 계산 (근로기준법 제2조 제1항 제6호)

```
평균임금 = 산정사유 발생일 이전 3개월간 임금 총액 / 그 기간의 총일수
```

| 항목 | 포함 여부 | 예시 |
|------|-----------|------|
| 기본급 | O | 월급, 일급, 시급 |
| 정기상여금 | O | 매월/분기/반기 정기상여금 |
| 연차수당 | O | 미사용 연차수당 |
| 식대·교통비 (정기·일률·고정) | O | 매월 고정 지급되는 경우 |
| 연장·야간·휴일근로수당 | O | 실제 지급된 금액 |
| 실비변상적 금품 | X | 출장비, 차량유지비 (실비) |
| 일시적·우발적 금품 | X | 경조사비, 체력단련비 (비정기) |

#### 최저 보장: 통상임금
- 평균임금이 **통상임금보다 낮은 경우**, 통상임금을 평균임금으로 적용

#### 계산 예시

```
[조건]
- 근무기간: 3년 6개월 (1,278일)
- 최근 3개월 임금 총액: 12,000,000원
- 최근 3개월 총일수: 91일

[계산]
1일 평균임금 = 12,000,000 / 91 = 131,868원
퇴직금 = 131,868 x 30 x (1,278 / 365) = 13,847,917원
```

### 3. 퇴직금 지급기한

#### 제9조 (퇴직금의 지급)

##### 제1항
사용자는 근로자가 퇴직한 경우에는 그 지급사유가 발생한 날부터 **14일 이내**에 퇴직금을 지급하여야 한다. 다만, **특별한 사정이 있는 경우**에는 당사자 간의 합의에 의하여 지급기일을 연장할 수 있다.

##### 제2항 - 지급 방법
퇴직금은 근로자가 지정한 **개인형퇴직연금제도(IRP)의 계정**으로 이전하는 방법으로 지급하여야 한다.

##### 제2항 단서 - IRP 이전 예외
다음의 경우에는 근로자에게 직접 지급할 수 있다:
- 근로자가 **55세 이후**에 퇴직하여 급여를 받는 경우
- 급여가 **고용노동부장관이 정하여 고시하는 금액 이하**인 경우 (300만원 이하)
- 그 밖에 대통령령으로 정하는 사유

#### 실무적 의미

| 상황 | 처리 |
|------|------|
| 정상 퇴직 | 퇴직 후 14일 이내 IRP 이전 |
| 55세 이후 퇴직 | 현금 직접 수령 가능 |
| 300만원 이하 소액 | 현금 직접 수령 가능 |
| 지급 지연 합의 | 당사자 합의 시 기한 연장 가능 (서면 합의 권장) |
| 14일 초과 미지급 | 지연이자 발생 (연 20%) + 형사처벌 가능 |

#### 지연이자 (근로기준법 제37조 준용)
퇴직금을 기한 내에 지급하지 않은 경우, 미지급 퇴직금에 대하여 **연 20%의 지연이자** 가산

### 4. 미지급 벌칙

#### 제44조 (벌칙)

| 위반 행위 | 벌칙 |
|-----------|------|
| 제9조 제1항을 위반하여 퇴직금을 지급하지 아니한 자 | **3년 이하의 징역 또는 3천만원 이하의 벌금** |

#### 반의사불벌죄
- 피해자의 **명시적인 의사에 반하여** 공소를 제기할 수 없음
- 즉, 근로자가 처벌을 원하지 않으면 형사소추 불가

#### 구제절차

```
1단계: 내용증명 발송
  - 사업주에게 퇴직금 지급 요구 (14일 기한 명시)

2단계: 고용노동부 진정
  - 관할 지방고용노동관서에 임금(퇴직금) 체불 진정
  - 근로감독관이 사실조사 후 시정지도

3단계: 노동위원회 구제신청
  - 퇴직금 미지급이 부당해고와 연계된 경우

4단계: 민사소송
  - 임금(퇴직금) 청구소송 (소액사건심판 활용 가능)
  - 소멸시효: 3년 (퇴직일로부터)

5단계: 형사고소
  - 고용노동부 진정 후에도 미지급 시 형사고소
  - 3년 이하 징역 또는 3천만원 이하 벌금
```

## Part 2: 근로계약

### 5. 근로조건의 명시

#### 근로기준법 제17조 (근로조건의 명시)

##### 제1항
사용자는 근로계약을 체결할 때에 근로자에게 다음 각 호의 사항을 **명시**하여야 한다. 근로계약 체결 후 다음 각 호의 사항을 **변경하는 경우**에도 또한 같다.

#### 명시사항

| 구분 | 항목 | 서면 명시 의무 |
|------|------|:---:|
| 필수 | 임금 | O |
| 필수 | 소정근로시간 | O |
| 필수 | 휴일 | O |
| 필수 | 연차 유급휴가 | O |
| 필수 | 취업의 장소와 종사하여야 할 업무에 관한 사항 | O |
| 필수 | 근로계약기간 (기간제의 경우) | O |
| 추가 | 취업규칙에서 정한 사항 | - |
| 추가 | 기숙사 규칙에서 정한 사항 (기숙사 입주 시) | - |

##### 제2항 - 서면 교부
사용자는 임금의 구성항목·계산방법·지급방법, 소정근로시간, 휴일, 연차 유급휴가에 관한 사항이 명시된 **서면(근로계약서)**을 근로자에게 **교부**하여야 한다.

#### 위반 시 벌칙

| 위반 행위 | 벌칙 |
|-----------|------|
| 서면 명시·교부 의무 위반 (제17조 제2항) | **500만원 이하의 벌금** (제114조) |

#### 실무적 의미
- 근로계약서 **미교부** 자체가 위법 (내용 불문)
- 근로조건 **변경 시**에도 서면으로 명시·교부 필요
- 구두 약정도 유효하나, 분쟁 시 입증 곤란
- 전자문서로 교부도 가능

### 6. 위약 예정의 금지

#### 근로기준법 제20조 (위약 예정의 금지)

사용자는 근로계약 불이행에 대한 **위약금 또는 손해배상액을 예정하는 계약**을 체결하지 못한다.

#### 금지되는 계약 예시

| 유형 | 예시 | 위법 여부 |
|------|------|-----------|
| 퇴사 위약금 | "1년 이내 퇴사 시 교육비 500만원 배상" | **위법** |
| 근무기간 미충족 벌금 | "3년 미만 근무 시 위약금 1,000만원" | **위법** |
| 경업금지 위약금 | "퇴사 후 동종업계 취업 시 위약금" | **원칙적으로 위법** (별도 합의 가능성 있음) |
| 실손해 배상 | 근로자의 고의·과실로 인한 실제 손해 배상 | **적법** (위약 예정이 아닌 경우) |

#### 위반 시 벌칙

| 위반 행위 | 벌칙 |
|-----------|------|
| 위약 예정 계약 체결 (제20조 위반) | **500만원 이하의 벌금** (제114조) |

#### 실무적 의미
- 위약금 **예정 계약 자체**가 무효 (이행 청구 불가)
- **실제 손해배상 청구**와는 구별: 근로자의 고의·과실로 실제 손해가 발생한 경우 배상 청구 가능
- 교육훈련비 반환약정: 판례에 따라 합리적 범위 내에서 유효할 수 있음 (전액 반환은 위약 예정에 해당할 수 있음)

### 7. 전차금 상계의 금지

#### 근로기준법 제21조 (전차금 상계의 금지)

사용자는 **전차금(前借金)** 이나 그 밖에 **근로할 것을 조건으로 하는 전대(前貸)채권**과 **임금을 상계**하지 못한다.

#### 금지의 의미

| 항목 | 내용 |
|------|------|
| **전차금** | 근로 제공을 조건으로 미리 빌려준 돈 |
| **전대채권** | 근로를 조건으로 한 일체의 대부금 |
| **상계 금지** | 위 채권을 임금에서 공제(상계)할 수 없음 |

#### 허용되는 경우

| 유형 | 허용 여부 | 조건 |
|------|-----------|------|
| 근로 조건이 아닌 순수 대부금 | 상계 가능 | 근로자의 자유의사에 의한 동의 |
| 급여 선지급 (가불) | 공제 가능 | 근로자의 요청에 의한 경우 |
| 법정 공제 | 공제 가능 | 소득세, 4대 보험료 등 |

#### 위반 시 벌칙

| 위반 행위 | 벌칙 |
|-----------|------|
| 전차금 상계 (제21조 위반) | **500만원 이하의 벌금** (제114조) |

#### 실무적 의미
- 근로자를 **채무로 구속**하여 강제 근로시키는 것을 방지하려는 취지
- 근로자가 임의로 빌린 돈이라도, 근로를 조건으로 한 것이면 상계 금지
- 근로자의 **자유로운 의사**에 의한 상계 동의는 별개 (판례에 따라 유효할 수 있음)

### 8. 기타 근로계약 관련 노동자 보호 조항

#### 제7조 (강제 근로의 금지)
사용자는 폭행, 협박, 감금, 그 밖에 정신상 또는 신체상의 자유를 부당하게 구속하는 수단으로써 근로자의 자유의사에 어긋나는 근로를 강요하지 못한다.
- 벌칙: **5년 이하의 징역 또는 5천만원 이하의 벌금**

#### 제19조 (근로조건의 위반)
근로조건이 사실과 다를 경우 근로자는 근로조건 위반을 이유로 손해의 배상을 청구할 수 있으며 즉시 근로계약을 해제할 수 있다.
- 거주지 변경 근로자의 경우: 귀향여비 지급 의무

#### 제23조 (해고 등의 제한)
사용자는 근로자에게 정당한 이유 없이 해고, 휴직, 정직, 전직, 감봉, 그 밖의 징벌을 하지 못한다.
- 부당해고 시 노동위원회에 구제신청 가능 (해고일로부터 3개월 이내)

#### 제26조 (해고의 예고)
사용자는 근로자를 해고하려면 적어도 **30일 전**에 예고하여야 하며, 30일 전에 예고하지 아니하였을 때에는 **30일분 이상의 통상임금**을 지급하여야 한다.

#### 제27조 (해고사유 등의 서면통지)
사용자는 근로자를 해고하려면 **해고사유와 해고시기를 서면으로 통지**하여야 한다. 서면통지를 하지 않은 해고는 효력이 없다.

## 핵심 실무 체크리스트

### 퇴직금 관련
- [ ] 1년 이상 계속 근무 + 주 15시간 이상 근무 확인
- [ ] 평균임금 산정 (퇴직일 이전 3개월 임금 총액 기준)
- [ ] 퇴직 후 14일 이내 지급 확인
- [ ] IRP 이전 또는 직접 지급 요건 확인
- [ ] 미지급 시 지연이자 (연 20%) 가산 여부

### 근로계약 관련
- [ ] 근로계약서 서면 교부 여부
- [ ] 필수 명시사항 포함 여부 (임금, 근로시간, 휴일, 연차)
- [ ] 위약금 조항 포함 여부 (포함 시 무효)
- [ ] 전차금 관련 조항 포함 여부
