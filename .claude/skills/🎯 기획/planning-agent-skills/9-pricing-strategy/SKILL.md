# Pricing Strategy Skill

서비스의 가격 정책을 체계적으로 설계합니다.
가격 모델 선택부터 플랜 구조, 구체적인 가격까지 결정합니다.

## 가격 책정 접근법

### 3가지 접근법

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│   1. 비용 기반 (Cost-Based)                                      │
│   ─────────────────────────                                      │
│   가격 = 비용 + 마진                                              │
│   → 최소 가격 기준점                                              │
│                                                                  │
│   2. 경쟁 기반 (Competition-Based)                               │
│   ─────────────────────────                                      │
│   경쟁사 가격 대비 포지셔닝                                        │
│   → 시장 기대 가격대                                              │
│                                                                  │
│   3. 가치 기반 (Value-Based) ⭐ 권장                              │
│   ─────────────────────────                                      │
│   고객이 얻는 가치에 기반                                          │
│   → 최적 가격점                                                   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 가격 모델 유형

| 모델 | 설명 | 적합한 경우 |
|------|------|------------|
| **월 구독** | 월별 정액 결제 | SaaS, 지속 사용 |
| **연 구독** | 연간 결제 (할인) | 장기 고객 확보 |
| **사용량 기반** | 쓴 만큼 지불 | API, 클라우드 |
| **시트 기반** | 사용자 수 × 가격 | B2B 협업 툴 |
| **티어 기반** | 기능별 차등 | 프리미엄 |
| **일회성** | 한 번 결제 | 앱, 소프트웨어 |

## 4. 가격 책정

### 가격 결정 프로세스

#### Step 1: 비용 기반 최소 가격

```
변동비/고객: ₩{variable_cost}
목표 마진: {margin}%
최소 가격 = ₩{variable_cost} / (1 - {margin}%)
         = ₩{min_price}
```

#### Step 2: 경쟁 기반 가격대

| 경쟁사 | 유사 플랜 | 가격 |
|--------|----------|------|
| {competitor_1} | {plan} | ₩{price} |
| {competitor_2} | {plan} | ₩{price} |
| {competitor_3} | {plan} | ₩{price} |
| **평균** | - | **₩{avg_price}** |

경쟁 가격대: ₩{low_price} ~ ₩{high_price}

#### Step 3: 가치 기반 가격

```
고객이 얻는 가치:
- 시간 절약: {hours}시간/월 × ₩{hourly_rate} = ₩{time_value}
- 비용 절감: ₩{cost_saving}
- 추가 수익: ₩{revenue_gain}
─────────────────────────────
총 가치: ₩{total_value}/월

가치의 10-20% = ₩{value_based_price}
```

#### 최종 가격 결정

| 플랜 | 비용 기반 | 경쟁 기반 | 가치 기반 | 최종 가격 |
|------|----------|----------|----------|----------|
| Basic | ₩{cost} | ₩{comp} | ₩{value} | **₩{final}** |
| Pro | ₩{cost} | ₩{comp} | ₩{value} | **₩{final}** |
