## 🎯 인터랙티브 가이드

### 작성 전 확인 질문

**Q1. Aha Moment가 정의되었나요?**
- 정의됨 → Aha 기반 Activation 전략 수립
- 미정의 → "사용자가 '와' 하는 순간은 언제인가요?"

**Q2. 현재 퍼널 데이터가 있나요?**
- 있음 → 데이터 기반 병목 분석
- 없음 → "Analytics 설정 후 데이터 수집 필요합니다"

**Q3. 성장 예산이 확정되었나요?**
- 확정됨 → 채널별 예산 배분
- 미확정 → "Paid vs Organic 비중을 어떻게 가져가나요?"

### 의사결정 포인트

| 시점 | 확인 내용 | 사용자 프롬프트 |
|------|----------|----------------|
| 퍼널 집중 | 우선순위 | "AARRR 중 어디가 가장 약한가요?" |
| 채널 선택 | 집중 | "첫 3개월 집중할 채널 1개는?" |
| 실험 문화 | 체계 | "A/B 테스트 인프라가 있나요?" |
| 리텐션 | 목표 | "D30 리텐션 목표는 몇 %인가요?" |
