# Legal Checklist Reference

## 서비스 유형별 법적 요구사항

### 1. 일반 웹/앱 서비스

```
필수 문서
─────────────────────────────────────
□ 이용약관 (Terms of Service)
□ 개인정보처리방침 (Privacy Policy)
□ 쿠키 정책 (Cookie Policy) - EU 대상 시

필수 기능
─────────────────────────────────────
□ 개인정보 수집 동의
□ 마케팅 수신 동의 (선택)
□ 회원 탈퇴 기능
□ 개인정보 열람/정정/삭제 기능
```

### 2. 전자상거래 (E-commerce)

```
추가 필수 문서
─────────────────────────────────────
□ 반품/환불 정책
□ 배송 정책
□ 구매 약관

추가 필수 기능
─────────────────────────────────────
□ 청약철회 (7일)
□ 주문 확인/취소
□ 결제 내역 표시
□ 사업자 정보 표시

표시 의무
─────────────────────────────────────
□ 상호, 대표자명
□ 사업자등록번호
□ 통신판매업 신고번호
□ 주소, 연락처
□ 호스팅 제공자
```

### 3. 결제/핀테크

```
인허가
─────────────────────────────────────
□ 전자금융업 등록/허가 (금융위)
□ PG사 계약
□ 정보보호관리체계(ISMS) 인증

보안 요구사항
─────────────────────────────────────
□ PCI-DSS 준수 (카드 결제)
□ 전송 암호화 (TLS 1.2+)
□ 데이터 암호화
□ 접근 통제
□ 이상거래탐지(FDS)
```

### 4. 의료/헬스케어

```
인허가
─────────────────────────────────────
□ 의료기기 인허가 (식약처)
□ 원격의료 관련 규정 확인

데이터 보호
─────────────────────────────────────
□ 민감정보(건강정보) 처리 동의
□ 의료법 준수
□ 익명화/가명화 처리
```

### 5. 교육 (EdTech)

```
아동 대상 서비스
─────────────────────────────────────
□ 법정대리인 동의 (14세 미만)
□ 유해 콘텐츠 차단
□ 광고 제한

학원 운영
─────────────────────────────────────
□ 학원 등록/신고
□ 교육청 신고
```

### 6. 식품/F&B

```
인허가
─────────────────────────────────────
□ 영업 신고/허가
□ 위생 교육
□ HACCP (해당 시)

표시 의무
─────────────────────────────────────
□ 영양정보
□ 알레르기 정보
□ 원산지 표시
```

### 7. SaaS/B2B

```
계약 관련
─────────────────────────────────────
□ 서비스 수준 협약 (SLA)
□ 데이터 처리 계약 (DPA)
□ 보안 인증 (SOC2, ISO27001)

데이터 처리
─────────────────────────────────────
□ 위탁 계약
□ 수탁자 관리
□ 해외 이전 동의 (해당 시)
```

## 개인정보보호

### 개인정보보호법 요구사항

```
수집 원칙
─────────────────────────────────────
□ 최소 수집 원칙
□ 목적 명시
□ 동의 획득
□ 제3자 제공 동의

처리 원칙
─────────────────────────────────────
□ 목적 내 이용
□ 안전한 관리
□ 보유기간 준수
□ 파기 절차

정보주체 권리 보장
─────────────────────────────────────
□ 열람권
□ 정정권
□ 삭제권
□ 처리정지권
□ 동의 철회권
```

### GDPR 요구사항 (EU 대상)

```
추가 요구사항
─────────────────────────────────────
□ DPO 지정 (해당 시)
□ 개인정보영향평가(DPIA)
□ 72시간 내 침해 통지
□ 처리 활동 기록
□ 국외 이전 적법성
```

### 개인정보처리방침 포함 사항

```
1. 수집하는 개인정보 항목
2. 수집 및 이용 목적
3. 보유 및 이용 기간
4. 제3자 제공 (있는 경우)
5. 위탁 현황
6. 정보주체 권리
7. 개인정보 보호책임자
8. 연락처
```

## 이용약관

### 필수 조항

```
1. 목적
2. 정의
3. 약관의 효력 및 변경
4. 서비스 이용
5. 회원의 의무
6. 서비스 제공자의 의무
7. 저작권 및 지적재산권
8. 면책조항
9. 분쟁 해결
10. 관할법원
```

### 유의 조항

```
불공정 약관 유형 (피해야 함)
─────────────────────────────────────
❌ 일방적 해지권
❌ 불합리한 손해배상 제한
❌ 일방적 약관 변경
❌ 청약철회권 제한
❌ 부당한 면책
```

```

─────────────────────────────────────

```

### 체크리스트

```

□ 제3자 콘텐츠 권리 확인
□ 사용자 콘텐츠 라이선스 명시
```

| 세금 | 대상 | 세율 |
|------|------|------|

| 소득세 | 개인 소득 | 6-45% |

```
부가세

법인세
─────────────────────────────────────

종합소득세
─────────────────────────────────────
- 5월 말
```
