# Lean Canvas Template

## 개요

Lean Canvas는 Ash Maurya가 Business Model Canvas를 스타트업에 맞게 변형한 1-page 비즈니스 모델 도구입니다.

## 각 섹션 설명

### 1. Customer Segments (고객 세그먼트)
- **무엇**: 타겟 고객 정의
- **질문**: 누구를 위한 것인가? 얼리어답터는 누구인가?
- **예시**: "25-35세 직장인, 생산성 앱에 월 $10 이상 지출하는 사람"

### 2. Problem (문제)
- **무엇**: 고객이 겪는 Top 3 문제
- **질문**: 어떤 고통을 해결하는가?
- **기존 대안**: 현재 고객이 사용하는 해결책
- **예시**: "1) 시간 관리 어려움 2) 할일 누락 3) 우선순위 혼란"

### 3. Unique Value Proposition (고유 가치 제안)
- **무엇**: 왜 당신의 제품인지 설명하는 한 문장
- **질문**: 왜 다른 것이 아닌 당신의 제품을 선택해야 하는가?
- **포맷**: "{결과}를 {방법}으로 {타겟}에게 제공"
- **예시**: "AI가 우선순위를 자동 정렬해주는 유일한 할일 관리 앱"

### 4. Solution (솔루션)
- **무엇**: 각 문제에 대한 Top 3 해결책/기능
- **질문**: 어떻게 문제를 해결하는가?
- **예시**: "1) AI 자동 정렬 2) 음성 입력 3) 캘린더 연동"

### 5. Channels (채널)
- **무엇**: 고객에게 도달하는 경로
- **질문**: 어떻게 고객을 찾고 전달하는가?
- **예시**: "앱스토어, SEO, 콘텐츠 마케팅, 추천"

### 6. Revenue Streams (수익원)
- **무엇**: 수익 모델과 예상 수익
- **질문**: 어떻게 돈을 버는가?
- **포함**: LTV, ARPU, Gross Margin
- **예시**: "Freemium, Pro $9.99/월, LTV $120"

### 7. Cost Structure (비용 구조)
- **무엇**: 주요 비용 항목
- **질문**: 비용이 어디에 드는가?
- **포함**: CAC, 운영비, 개발비
- **예시**: "CAC $15, 서버 $500/월, 인건비 $10K/월"

### 8. Key Metrics (핵심 지표)
- **무엇**: 추적해야 할 핵심 활동과 지표
- **질문**: 성공을 어떻게 측정하는가?
- **예시**: "WAU, Activation Rate, D7 Retention, MRR"

### 9. Unfair Advantage (경쟁 우위)
- **무엇**: 쉽게 복제할 수 없는 강점
- **질문**: 왜 경쟁사가 따라올 수 없는가?
- **예시**: "독점 데이터, 네트워크 효과, 전문가 팀, 특허"

## 작성 순서

```
1 → 2 → 3 → 4 → 5 → 6 → 7 → 8 → 9

1. Customer Segments (누구를 위한 것인가)
2. Problem (그들의 문제는)
3. UVP (왜 우리인가)
4. Solution (어떻게 해결하나)
5. Channels (어떻게 도달하나)
6. Revenue (어떻게 돈을 버나)
7. Cost (비용은)
8. Key Metrics (어떻게 측정하나)
9. Unfair Advantage (왜 이길 수 있나)
```

## 검증 질문

| 섹션 | 검증 질문 |
|------|----------|
| Customer | 실제로 존재하는 세그먼트인가? |
| Problem | 실제로 돈을 내고 해결하고 싶은 문제인가? |
| UVP | 차별화가 명확한가? |
| Solution | 문제를 실제로 해결하는가? |
| Channels | 확장 가능한 채널인가? |
| Revenue | 단위 경제가 성립하는가? |
| Cost | 지속 가능한 비용 구조인가? |
| Metrics | 측정 가능한 지표인가? |
| Advantage | 진정한 경쟁 우위인가? |
