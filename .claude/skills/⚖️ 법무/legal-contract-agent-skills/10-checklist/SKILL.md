# Checklist Skill

계약 유형별 체크리스트로 필수 항목을 확인합니다.

## 공통 체크리스트 (모든 계약)

### 기본 요소

```
□ 계약 제목 명확
□ 체결일/효력발생일 명시
□ 당사자 정보 정확 (법인명, 대표자, 주소)
□ 당사자 정의 (갑, 을 또는 명칭)
□ 계약 목적 명시
```

### 핵심 조항

```
□ 계약 기간 명시
□ 대가/지급 조건 명시
□ 해지 조건 명시
□ 준거법 명시
□ 분쟁해결 조항 명시
```

### 형식 요소

```
□ 조항 번호 체계 일관
□ 정의 용어 일관 사용
□ 참조 조항 정확
□ 서명란 완비
□ 별첨 목록 명시
```

## NDA (비밀유지계약) 체크리스트

### 필수 항목 (Required)

```
□ 비밀정보 정의
  - 범위가 명확한가?
  - 표시 요건이 있는가? (서면, "Confidential" 표시 등)

□ 비밀유지 의무
  - 수령자의 의무 명시
  - 사용 범위 제한

□ 허용 공개 범위
  - 임직원, 자문사 등
  - Need-to-know 기준

□ 비밀유지 기간
  - 시작점 (수령시/계약체결시)
  - 종료점 (계약종료 후 X년)

□ 반환/파기 의무
  - 반환 시점
  - 파기 방법
  - 잔존 복사본 처리
```

### 권장 항목 (Recommended)

```
□ 비밀정보 예외
  - 공지 정보
  - 사전 보유 정보
  - 독자 개발 정보
  - 제3자로부터 합법 취득

□ 강제 공개 예외
  - 법원 명령
  - 정부 기관 요청
  - 통지 의무

□ 잔존 지식 (Residuals)
  - 개인 기억에 남은 정보 처리

□ 손해배상/구제수단
  - 금지청구권
  - 손해배상

□ 분쟁해결
  - 준거법
  - 관할/중재
```

### 주의 사항

```
⚠️ 비밀정보 범위가 과도하지 않은가?
⚠️ 비밀유지 기간이 합리적인가? (통상 3-5년)
⚠️ 쌍방 의무인가, 일방 의무인가?
⚠️ 반환 의무가 이행 가능한가?
```

## 서비스 계약 (MSA/SaaS) 체크리스트

### 필수 항목

```
□ 서비스 범위
  - 제공 서비스 명확히 정의
  - 제외 서비스 명시

□ 서비스 수준 (SLA)
  - 가용성 목표 (예: 99.9%)
  - 응답 시간
  - 장애 대응 시간
  - SLA 미달 시 구제수단

□ 대가 및 지급
  - 요금 구조 (정액/종량/혼합)
  - 지급 시기
  - 가격 조정 조항

□ 기간 및 해지
  - 초기 기간
  - 갱신 조건
  - 해지 사유 및 절차

□ 데이터 처리
  - 데이터 소유권
  - 데이터 보안
  - 데이터 반환/삭제

□ 지적재산권
  - 기존 IP 귀속
  - 커스터마이징 결과물
  - 라이선스 범위

□ 비밀유지
  - 범위 및 기간

□ 책임제한
  - 손해배상 범위
  - 책임 상한
  - 면책 사항
```

### 권장 항목

```
□ 변경 관리
  - 서비스 변경 절차
  - 가격 변경 통지

□ 보안 요건
  - 인증 (ISO27001, SOC2 등)
  - 침해 통지 의무

□ 감사권
  - 제공자 시설 감사
  - 보안 감사

□ 하도급
  - 하도급 허용 여부
  - 승인 절차

□ 보험
  - 필요 보험 종류
  - 최소 보상한도

□ 출구 전략
  - 마이그레이션 지원
  - 데이터 포맷
  - 전환 기간
```

## 라이선스 계약 체크리스트

### 필수 항목

```
□ 라이선스 범위
  - 독점/비독점
  - 허용 용도
  - 지역 범위
  - 기간

□ 라이선스 대상
  - 대상 IP 명확히 특정
  - 버전/범위

□ 대가
  - 라이선스 피
  - 로열티 구조
  - 최소 보장금

□ 서브라이선스
  - 허용 여부
  - 조건

□ 보증
  - 권리 보유 보증
  - 제3자 권리 비침해
  - 기술적 보증

□ 책임/배상
  - 침해 대응
  - 손해배상
```

### 권장 항목

```
□ 개선사항 처리
  - 개선 IP 귀속
  - 그랜트백

□ 감사권
  - 사용량 감사
  - 로열티 검증

□ 소스코드 에스크로
  - 조건
  - 관리 기관

□ 기술지원
  - 범위
  - 대응 시간

□ 업데이트/업그레이드
  - 포함 여부
  - 추가 비용
```

## 고용/용역 계약 체크리스트

### 필수 항목

```
□ 업무 범위
  - 수행 업무 명시
  - 결과물 정의

□ 기간
  - 시작/종료일
  - 연장 조건

□ 대가
  - 급여/보수
  - 지급 시기
  - 원천징수

□ 근무 조건
  - 근무 장소
  - 근무 시간
  - 원격근무

□ 비밀유지
  - 기간
  - 범위

□ 지적재산권
  - 업무상 저작물 귀속
  - 직무발명 처리
```

### 권장 항목

```
□ 경쟁금지
  - 기간 (합리적으로)
  - 범위 (명확하게)
  - 지역

□ 고객유인금지
  - 기간
  - 범위

□ 해지/해고
  - 사유
  - 통지 기간
  - 퇴직금

□ 분쟁해결
  - 준거법
  - 관할

□ 보험
  - 산재보험
  - 기타 보험
```

## 투자계약 체크리스트

### 필수 항목

```
□ 투자 조건
  - 투자 금액
  - 주식 종류
  - 가격 (Pre/Post Valuation)

□ 거래 종결 조건
  - 선행 조건
  - 종결 일정

□ 진술 및 보증
  - 회사 진술
  - 투자자 진술

□ 주주간 권리
  - 우선매수권
  - 공동매도권
  - 동반매도청구권

□ 경영 참여
  - 이사회 구성
  - 중요 사항 동의권
  - 정보제공권

□ 희석화 방지
  - 전환가격 조정
  - 안티딜루션
```

### 권장 항목

```
□ 주식매수청구권
  - 풋옵션 조건
  - 행사 가격

□ 청산우선권
  - 배수
  - 참가 여부

□ 잔여재산분배
  - 순서
  - 비율

□ 비경쟁/전념의무
  - 창업자 전념
  - 경쟁 금지

□ Exit 조항
  - IPO
  - M&A
```
