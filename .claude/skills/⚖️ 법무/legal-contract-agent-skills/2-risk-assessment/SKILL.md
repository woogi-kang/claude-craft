# Risk Assessment Skill

계약서의 위험 조항을 식별하고 심각도를 평가합니다.

## Risk Matrix

```
                        영향도 (Impact)
                   낮음      보통      높음
              ┌─────────┬─────────┬─────────┐
         높음 │  🟡     │  🟠     │  🔴     │
   발생       │ Medium  │  High   │ Critical│
   가능성     ├─────────┼─────────┼─────────┤
   (Likeli-  보통 │  🟢     │  🟡     │  🟠     │
   hood)      │  Low    │ Medium  │  High   │
              ├─────────┼─────────┼─────────┤
         낮음 │  🟢     │  🟢     │  🟡     │
              │  Low    │  Low    │ Medium  │
              └─────────┴─────────┴─────────┘
```

### 심각도 정의

| 심각도 | 정의 | 대응 |
|--------|------|------|
| 🔴 **Critical** | 서명 불가, 심각한 법적/재무적 리스크 | 수정 필수, 거부 고려 |
| 🟠 **High** | 불리한 조건, 상당한 리스크 | 수정 강력 권고 |
| 🟡 **Medium** | 개선 권장, 잠재적 이슈 | 협상 고려 |
| 🟢 **Low** | 수용 가능, 참고 사항 | 인지만 |

## 위험 카테고리

### 1. 책임 관련 (Liability)

| 위험 유형 | 위험 조항 예시 | 영향도 |
|----------|---------------|--------|
| 무제한 손해배상 | "모든 손해를 배상한다" | 🔴 Critical |
| 간접손해 포함 | "결과적 손해 포함" | 🔴 Critical |
| 과도한 면책 | "상대방 과실 시에도 면책" | 🔴 Critical |
| 제3자 청구 전액 | "제3자 청구 전액 부담" | 🟠 High |
| 배상 상한 부재 | 상한 규정 없음 | 🟠 High |

**Red Flag 문구:**
```
- "일체의 손해"
- "모든 손해를 배상"
- "직접, 간접 손해 모두"
- "제한 없이 배상"
- "어떠한 책임도 지지 않는다" (면책)
```

### 2. 지적재산권 (IP)

| 위험 유형 | 위험 조항 예시 | 영향도 |
|----------|---------------|--------|
| 전면 IP 양도 | "모든 결과물 IP 양도" | 🔴 Critical |
| 광범위 라이선스 | "전세계, 영구, 독점" | 🟠 High |
| 기존 IP 위험 | "기존 IP도 라이선스" | 🟠 High |
| 2차 저작물 | "2차 저작물도 양도" | 🟡 Medium |

**Red Flag 문구:**
```
- "모든 지적재산권을 양도"
- "영구적, 전세계적, 독점적"
- "기존 IP 포함"
- "포기할 수 없는 권리도 포기"
```

### 3. 기간/해지 (Term/Termination)

| 위험 유형 | 위험 조항 예시 | 영향도 |
|----------|---------------|--------|
| 일방 즉시해지 | "상대방은 즉시 해지 가능" | 🔴 Critical |
| 자동갱신 함정 | "30일 전 통지 없으면 3년 자동갱신" | 🟠 High |
| 불균형 해지권 | "갑만 해지권 보유" | 🟠 High |
| 짧은 통지기간 | "7일 전 통지로 해지" | 🟡 Medium |
| 과도한 lock-in | "최소 3년 의무 기간" | 🟡 Medium |

**Red Flag 문구:**
```
- "즉시 해지"
- "어떠한 이유로든"
- "사전 통지 없이"
- "자동으로 갱신"
- "갱신 거부 30일 전"
```

### 4. 비밀유지 (Confidentiality)

| 위험 유형 | 위험 조항 예시 | 영향도 |
|----------|---------------|--------|
| 무기한 유지 | "영구히 비밀 유지" | 🟠 High |
| 과도한 범위 | "모든 정보 = 비밀정보" | 🟠 High |
| 파기증명 요구 | "파기 사실 증명 필요" | 🟡 Medium |
| 예외 없음 | "어떠한 예외도 없이" | 🟡 Medium |

**Red Flag 문구:**
```
- "영구히"
- "영원히"
- "모든 정보"
- "어떠한 예외도 없이"
- "파기를 증명"
```

### 5. 경쟁금지 (Non-Compete)

| 위험 유형 | 위험 조항 예시 | 영향도 |
|----------|---------------|--------|
| 장기간 | "종료 후 5년간" | 🔴 Critical |
| 광범위 지역 | "전 세계" | 🟠 High |
| 넓은 범위 | "유사 업종 전체" | 🟠 High |
| 모호한 정의 | "경쟁적 활동" | 🟡 Medium |

**Red Flag 문구:**
```
- "3년 이상"
- "전 세계"
- "모든 경쟁 활동"
- "유사 업종"
- "직간접적"
```

### 6. 분쟁해결 (Dispute)

| 위험 유형 | 위험 조항 예시 | 영향도 |
|----------|---------------|--------|
| 불리한 관할 | "상대방 본사 관할" | 🟠 High |
| 외국 중재 | "싱가포르 중재" (국내 기업) | 🟠 High |
| 소비자 불리 | "소송만 허용" (B2C) | 🟡 Medium |
| 일방 선택권 | "갑이 관할 선택" | 🟡 Medium |

**Red Flag 문구:**
```
- "상대방 소재지"
- "외국 중재"
- "갑의 선택에 따라"
- "이의 없이 동의"
```

### 7. 보증/진술 (Warranty)

| 위험 유형 | 위험 조항 예시 | 영향도 |
|----------|---------------|--------|
| 과도한 보증 | "모든 목적 적합성 보증" | 🟠 High |
| 보증 배제 | "어떠한 보증도 없음" (제공자) | 🟡 Medium |
| 제3자권리 침해 | "침해 없음 보증" | 🟡 Medium |

## 위험 평가 워크플로우

```
1. 조항별 스캔
      │
      ├─ Red Flag 키워드 탐지
      ├─ 카테고리별 분류
      │
      ▼
2. 위험 식별
      │
      ├─ 위험 유형 분류
      ├─ 영향도 평가
      ├─ 발생 가능성 평가
      │
      ▼
3. Risk Matrix 적용
      │
      ├─ 심각도 결정
      ├─ 우선순위 산정
      │
      ▼
4. 권고사항 도출
      │
      ├─ 수정 제안
      ├─ 협상 포인트
      │
      ▼
5. 리포트 생성
   → workspace/work-legal/{project}/analysis/{project}-risk-assessment.md
```
