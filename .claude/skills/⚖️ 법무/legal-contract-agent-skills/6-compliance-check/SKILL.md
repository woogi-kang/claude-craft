# Compliance Check Skill

계약서의 법령 준수 여부를 검토하고 규제 리스크를 식별합니다.

## 검토 대상 법규

### 1. 공정거래 관련법

```yaml
fair_trade_laws:
  korea:
    - name: "독점규제 및 공정거래에 관한 법률"
      key_provisions:
        - "불공정거래행위 금지"
        - "부당한 공동행위 금지"
        - "경쟁제한적 기업결합 규제"

    - name: "하도급거래 공정화에 관한 법률"
      applies_to: "원사업자-수급사업자 관계"
      key_provisions:
        - "서면 발급 의무"
        - "대금 지급 기한 (60일 이내)"
        - "부당 단가 인하 금지"
        - "부당 발주취소 금지"

    - name: "대리점거래의 공정화에 관한 법률"
      applies_to: "공급업자-대리점 관계"
      key_provisions:
        - "서면계약 의무"
        - "경영정보 제공"
        - "판매목표 강제 금지"

    - name: "가맹사업거래의 공정화에 관한 법률"

        - "정보공개서 제공"
        - "영업지역 보장"
        - "부당한 의무 부과 금지"
```

## 워크플로우

```
1. 계약 유형 확인
      │
      ├─ 적용 법규 식별
      ├─ 관할권 확인
      │
      ▼
2. 필수 조항 확인
      │
      ├─ 법정 필수 기재사항
      ├─ 금지 조항 여부
      │
      ▼
3. 내용 적법성 검토
      │
      ├─ 법정 기준 충족 여부
      ├─ 금지 행위 해당 여부
      │
      ▼
4. 위반 리스크 평가
      │
      ├─ 위반 심각도
      ├─ 제재 수준
      │
      ▼
5. 컴플라이언스 리포트 생성
   → workspace/work-legal/{project}/review/{project}-compliance.md
```

## ❌ 위반 사항 상세

### V-001: 대금 지급 기한 초과

| 항목 | 내용 |
|------|------|
| **위반 법규** | 하도급법 제13조 |
| **현재 조항** | 제8조 "... 90일 이내 지급한다" |
| **법정 기준** | 60일 이내 |
| **위반 정도** | 30일 초과 |
| **제재 수준** | 지연이자 2배, 시정명령 |

**권고:**
> 제8조의 "90일"을 "60일"로 수정

---

### V-002: 재위탁 제한 조항 누락

| 항목 | 내용 |
|------|------|
| **위반 법규** | 개인정보보호법 제26조 |
| **현재 조항** | 해당 조항 없음 |
| **법정 기준** | 재위탁 제한 규정 필수 |
| **제재 수준** | 과징금, 형사처벌 가능 |

**권고:**
> 다음 조항 추가:
> "을은 갑의 사전 서면 동의 없이 개인정보 처리 업무를 제3자에게 재위탁할 수 없다."
