# Board Meeting Skill

이사회의 소집부터 의사록 작성까지 전 과정을 지원합니다.

## 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **적법한 소집** | 소집권자, 소집절차, 소집기간 준수 |
| **정족수 확인** | 재적이사 과반수 출석, 출석이사 과반수 찬성 |
| **기록 보존** | 의사록 10년 보존 의무 |

---

## 이사회 설치 여부

```yaml
board_requirement:
  mandatory:
    condition: "이사 3인 이상"
    note: "상장회사는 필수"

  optional:
    condition: "자본금 10억 미만 + 이사 2인 이하"
    alternative: "이사 과반수 결의로 대체"

  practical_tip: |
    스타트업 초기: 이사 1-2인으로 이사회 미설치
    투자 유치 시: 이사회 설치 전환 많음
```

---

## 이사회 권한

```yaml
board_authority:
  exclusive_matters:  # 이사회 전결 사항
    - "대표이사 선임/해임"
    - "업무집행 결정"
    - "이사 직무 감독"
    - "주주총회 소집 결정"
    - "신주 발행 (정관 위임 시)"
    - "사채 발행"
    - "지배인 선임/해임"

  delegated_from_shareholders:  # 정관으로 위임 가능
    - "중간배당"
    - "준비금 자본전입"
    - "주식배당"

  cannot_delegate:  # 대표이사 위임 불가
    - "중요 자산 처분"
    - "대규모 차입"
    - "이사 보수 결정"
```

## 소집 절차

### 소집권자

```yaml
convening_authority:
  principle: "대표이사"

  exceptions:
    이사_소집청구:
      who: "각 이사"
      process:
        - "회의 목적 사항 제시하여 소집 청구"
        - "대표이사 5일 내 미소집 시"
        - "청구 이사가 직접 소집 가능"

    감사_소집청구:
      trigger: "회사에 회복할 수 없는 손해 우려"
      process: "이사회 소집 요구"
```

### 소집 통지

```yaml
notice_requirements:
  기간: "1주 전"

  단축_가능:
    condition: "정관으로 단축 가능"
    typical: "3일, 5일 등"

  방법:
    - "서면"
    - "전자문서 (동의 시)"

  contents:
    - "일시"
    - "장소"
    - "목적사항 (안건)"

  생략_가능:
    condition: "이사 전원 동의"
    documentation: "의사록에 '소집절차 생략 동의' 기재"

  template: |
    이사회 소집통지서

    이사 각위

    당사 이사회를 아래와 같이 개최하오니 참석하여 주시기 바랍니다.

    1. 일시: 2026년 1월 30일(목) 오전 10시
    2. 장소: 서울시 강남구 OO로 OO, 본사 회의실
    3. 목적사항
       제1호 의안: 대표이사 선임의 건
       제2호 의안: 본점 이전의 건
       제3호 의안: 신주발행의 건

    2026년 1월 23일
    주식회사 OOO
    대표이사 홍길동
```

## 의사록 작성

### 필수 기재사항

```yaml
mandatory_contents:
  header:
    - "회사명"
    - "개최 일시"
    - "개최 장소"

  attendance:
    - "출석 이사 (성명 기재)"
    - "출석 감사"
    - "재적 이사 총수"
    - "정족수 충족 여부"

  proceedings:
    - "의장 (통상 대표이사)"
    - "개회 선언"
    - "안건별 심의 내용"
    - "의결 결과"
    - "폐회 선언"

  each_agenda:
    - "안건 제목"
    - "안건 내용 설명"
    - "질의응답 (있는 경우)"
    - "찬성/반대/기권 이사 성명"
    - "가결/부결 결과"

  closing:
    - "작성일"
    - "의장 기명날인"
    - "출석이사 전원 기명날인"
```

### 의사록 템플릿

```markdown
# 이사회 의사록

## 기본 정보

| 항목 | 내용 |
|------|------|
| 회사명 | 주식회사 OOO |
| 일시 | 2026년 1월 30일 오전 10시 |
| 장소 | 서울시 강남구 OO로 OO, 본사 회의실 |
| 재적 이사 | 3명 |
| 출석 이사 | 3명 (홍길동, 김철수, 이영희) |
| 출석 감사 | 1명 (박지성) |

## 정족수 확인

재적이사 3명 중 3명 출석하여 과반수 정족수를 충족하였으므로
적법하게 이사회가 성립되었음을 확인함.

---

## 의사 진행

### 1. 개회

의장 홍길동이 상기와 같이 이사회의 정족수가 충족되었음을 확인하고
오전 10시 05분 개회를 선언하다.

### 2. 제1호 의안: 대표이사 선임의 건

의장이 이사 홍길동을 대표이사로 선임할 것을 제안하다.

**심의 내용:**
- 김철수 이사: 홍길동 이사의 경영 역량과 실적을 고려할 때 적임자로 판단됨
- 이영희 이사: 동의함

**의결 결과:**
| 이사명 | 의결 |
|--------|------|
| 홍길동 | 찬성 |
| 김철수 | 찬성 |
| 이영희 | 찬성 |

→ **찬성 3, 반대 0으로 원안대로 가결**

피선임자 홍길동은 즉석에서 취임을 승낙하다.

### 3. 제2호 의안: 본점 이전의 건

의장이 본점을 서울시 강남구 OO로 OO에서
서울시 서초구 XX로 XX로 이전할 것을 제안하다.

이전 예정일: 2026년 2월 15일

**의결 결과:**
| 이사명 | 의결 |
|--------|------|
| 홍길동 | 찬성 |
| 김철수 | 찬성 |
| 이영희 | 찬성 |

→ **찬성 3, 반대 0으로 원안대로 가결**

### 4. 제3호 의안: 신주발행의 건

의장이 다음과 같이 신주를 발행할 것을 제안하다.

| 항목 | 내용 |
|------|------|
| 발행 주식 수 | 보통주 10,000주 |
| 발행가액 | 1주당 10,000원 |
| 납입기일 | 2026년 2월 28일 |
| 배정방법 | 제3자배정 (투자자 A사) |

**의결 결과:**
| 이사명 | 의결 |
|--------|------|
| 홍길동 | 찬성 |
| 김철수 | 찬성 |
| 이영희 | 찬성 |

→ **찬성 3, 반대 0으로 원안대로 가결**

### 5. 폐회

의장이 상기와 같이 모든 안건이 가결되었음을 선언하고,
오전 10시 45분 폐회를 선언하다.
```

## CLI 사용법

```bash
# 이사회 소집통지서 생성
/board-meeting notice --date 2026-01-30 --agenda "대표이사선임,본점이전"

# 의사록 초안 작성
/board-meeting minutes --agenda "대표이사선임,신주발행" --directors "홍길동,김철수,이영희"

# 정족수 확인
/board-meeting quorum --total-directors 3 --attending 2

# 서면결의서 작성
/board-meeting written --agenda "주주총회소집"

# 이사회 없는 회사 결정서
/board-meeting decision --no-board --agenda "본점이전"
```
