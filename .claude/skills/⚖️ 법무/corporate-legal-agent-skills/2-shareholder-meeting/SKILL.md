# Shareholder Meeting Skill

주주총회의 소집부터 의사록 작성까지 전 과정을 지원합니다.

## 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **적법한 소집** | 소집권자, 소집절차, 소집기간 준수 |
| **정족수 확인** | 의결 요건 충족 확인 |
| **기록 보존** | 의사록 10년 보존 의무 |

---

## 주주총회 유형

```yaml
meeting_types:
  정기주주총회:
    timing: "사업연도 종료 후 3개월 내"
    typical: "매년 3월"
    mandatory_agenda:
      - "재무제표 승인"
      - "이익배당 결정"
    optional_agenda:
      - "이사/감사 선임"
      - "정관 변경"
      - "임원 보수 한도 승인"

  임시주주총회:
    timing: "필요 시 수시 개최"
    triggers:
      - "자본금 변경"
      - "정관 변경"
      - "합병/분할"
      - "임원 선임 (정기총회 외)"
      - "주요 자산 처분"
```

## 소집 절차

### 소집권자

```yaml
convening_authority:
  원칙: "이사회 결의"

  exceptions:
    소수주주_소집청구:
      requirement: "발행주식 총수 3% 이상"
      process:
        - "이사회에 서면 청구"
        - "이사회 2주 내 미소집 시"
        - "법원 허가 받아 직접 소집"

    감사_소집:
      trigger: "회사에 회복할 수 없는 손해 우려"
```

```yaml
notice_requirements:

  방법:

  생략_가능:

  template: |
    주주총회 소집통지서

    주주님께

    당사 주주총회를 아래와 같이 개최하오니 참석하여 주시기 바랍니다.

    1. 일시: 2026년 3월 25일(수) 오전 10시
    2. 장소: 서울시 강남구 OO로 OO, 본사 회의실
    3. 목적사항
       제1호 의안: 제O기 재무제표 승인의 건
       제2호 의안: 이익배당의 건
       제3호 의안: 이사 선임의 건
       제4호 의안: 감사 선임의 건
       제5호 의안: 이사 보수 한도 승인의 건

    2026년 3월 15일
    주식회사 OOO
    대표이사 홍길동
```

## 정족수 확인

보통결의 정족수(25%) 및 특별결의 정족수(33.3%)를 충족하여
적법하게 총회가 성립되었음을 확인함.

## 의사 진행

### 1. 개회

의장 홍길동이 상기와 같이 주주총회의 정족수가 충족되었음을 확인하고
오전 10시 05분 개회를 선언하다.

### 2. 제1호 의안: 제5기 재무제표 승인의 건

의장이 제5기(2025년 1월 1일 ~ 2025년 12월 31일) 재무제표의
내용을 설명하고 승인을 요청하다.

**의결 결과:**
- 찬성: 80,000주 (100%)
- 반대: 0주 (0%)
- 기권: 0주 (0%)

→ **원안대로 가결**

### 3. 제2호 의안: 이익배당의 건

의장이 제5기 이익 중 1주당 500원의 현금배당을 제안하다.
총 배당금액: 50,000,000원

**의결 결과:**
- 찬성: 80,000주 (100%)
- 반대: 0주 (0%)
- 기권: 0주 (0%)

→ **원안대로 가결**

### 4. 제3호 의안: 이사 선임의 건

의장이 임기 만료 이사의 중임 및 신규 이사 선임을 제안하다.

- 중임 대상: 홍길동 (대표이사)
- 신규 선임: 김철수 (사외이사)

**의결 결과:**
- 찬성: 80,000주 (100%)
- 반대: 0주 (0%)
- 기권: 0주 (0%)

→ **원안대로 가결**

피선임자 홍길동, 김철수는 즉석에서 취임을 승낙하다.

### 5. 제4호 의안: 이사 보수 한도 승인의 건

의장이 제6기 이사 보수 한도를 연 500,000,000원으로 제안하다.

**의결 결과:**
- 찬성: 80,000주 (100%)
- 반대: 0주 (0%)
- 기권: 0주 (0%)

→ **원안대로 가결**

### 6. 폐회

의장이 상기와 같이 모든 안건이 가결되었음을 선언하고,
오전 10시 45분 폐회를 선언하다.
