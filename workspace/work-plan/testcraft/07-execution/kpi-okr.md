# TestCraft - KPI & OKR

> 핵심 성과 지표(KPI)와 목표-핵심 결과(OKR) 정의

---

## 1. North Star Metric

### 북극성 지표 정의

```
+------------------------------------------------------------------+
|                      North Star Metric                             |
+------------------------------------------------------------------+
|                                                                    |
|   "주간 TC 생성 완료 수 (Weekly Test Cases Generated)"             |
|                                                                    |
|   정의: 1주일 동안 사용자가 생성 완료한 총 테스트케이스 수          |
|                                                                    |
|   선택 이유:                                                       |
|   1. 핵심 가치와 직결 (TC 생성이 제품의 본질)                      |
|   2. 사용자 가치 반영 (실제로 사용한 만큼 측정)                    |
|   3. 수익과 연결 (사용량 → 업그레이드 → 매출)                     |
|   4. 행동 유도 (팀 전체가 이 지표 개선에 집중)                     |
|                                                                    |
|   목표:                                                            |
|   - MVP (Week 8): 200 TC/주                                       |
|   - Month 3: 1,000 TC/주                                          |
|   - Month 6: 5,000 TC/주                                          |
|   - Month 12: 15,000 TC/주                                        |
|                                                                    |
+------------------------------------------------------------------+
```

### 북극성 지표 분해

```
Weekly TC Generated
       │
       ├── Active Users × TC per User
       │
       ├── (Free Active × Free TC) + (Paid Active × Paid TC)
       │
       └── 개선 레버:
           ├── 신규 가입 증가
           ├── 활성화율 개선
           ├── 사용 빈도 증가
           └── 사용자당 TC 수 증가
```

### 2.4 Retention (유지)

```
+------------------------------------------------------------------+
|                    Retention KPIs                                  |
+------------------------------------------------------------------+

1. 주간 리텐션 (Week 1, 2, 4)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   정의: 가입 후 N주 후 재방문 비율

   목표:
   +----------+----------+----------+
   |  Week 1  |  Week 2  |  Week 4  |
   +----------+----------+----------+
   |   50%    |   35%    |   25%    |
   +----------+----------+----------+

2. 월간 리텐션 (Month 1, 3, 6)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   정의: 가입 후 N개월 후 활성 비율

   목표:
   +----------+----------+----------+
   | Month 1  | Month 3  | Month 6  |
   +----------+----------+----------+
   |   40%    |   25%    |   18%    |
   +----------+----------+----------+

3. Net Revenue Retention (NRR)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   정의: 기존 고객의 수익 유지율 (업그레이드 포함)

   목표: 100% → 110%

   계산: (시작 MRR + 확장 - 축소 - 이탈) / 시작 MRR

4. Churn Rate (이탈율)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   정의: 월간 유료 고객 이탈 비율

   목표: 5% → 3%

   계산: 이탈 고객 / 시작 고객 × 100

+------------------------------------------------------------------+
```

### 2.5 Revenue (수익)

```
+------------------------------------------------------------------+
|                    Revenue KPIs                                    |
+------------------------------------------------------------------+

1. MRR (Monthly Recurring Revenue)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   정의: 월간 반복 수익

   목표:
   +----------+----------+----------+----------+
   |  MVP     | Month 3  | Month 6  | Month 12 |
   +----------+----------+----------+----------+
   |   $75    |  $750    | $3,000   | $8,250   |
   +----------+----------+----------+----------+

2. ARR (Annual Recurring Revenue)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   정의: 연간 반복 수익 (MRR × 12)

   Year 1 목표: $99,000

3. 유료 전환율 (Free → Paid)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   정의: Free 사용자 중 유료 전환 비율

   목표: 5% → 8%

4. ARPU (Average Revenue Per User)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   정의: 유료 사용자당 평균 수익

   목표: $15/월

5. LTV (Customer Lifetime Value)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   정의: 고객 생애 가치

   목표: $225 → $375

   계산: ARPU × Gross Margin × (1 / Churn)

6. CAC (Customer Acquisition Cost)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   정의: 고객 획득 비용

   목표: $80 → $40

7. LTV:CAC Ratio
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   정의: 고객 가치 대비 획득 비용

   목표: 3:1 → 9:1

+------------------------------------------------------------------+
```
