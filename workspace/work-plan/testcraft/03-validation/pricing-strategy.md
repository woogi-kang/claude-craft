## 1. Pricing Philosophy

### 핵심 원칙

```
┌─────────────────────────────────────────────────────────────────┐
│                     Pricing Philosophy                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │

│                                                                  │
│   3. Seat-based: 팀 성장과 함께 수익 성장                         │
│      └─ 사용자 수 증가 = 자연스러운 확장 수익                     │
│                                                                  │
│   4. Simplicity: 이해하기 쉬운 3-Tier 구조                        │
│      └─ 복잡한 옵션 없이 명확한 차이                              │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 3. Pricing Rationale

### Free Tier 설계

**목적**: 제품 가치 경험, 전환 퍼널 진입점

| 제한 | 이유 | 전환 트리거 |
|-----|------|------------|
| 5 TC/월 | 충분히 경험, 지속 사용엔 부족 | 실무 사용 시 |
| 1 플랫폼 | 핵심 가치 경험 | 멀티플랫폼 필요 시 |
| 7일 히스토리 | 최근 작업 확인 | 이력 관리 필요 시 |

**Free 사용자 Journey**:
```
Day 1: 가입 → 첫 TC 생성 (Wow 경험)
Day 7: 3-4개 TC 생성, 가치 인식
Day 14: 5개 한도 도달, 업그레이드 고려
Day 30: Pro 전환 or 이탈
```

### Pro Tier 설계

**목적**: 주력 수익원, 개인/소규모 팀

| 기능 | 가치 | 경쟁 대비 |
|-----|------|----------|
| 무제한 TC | 제한 없는 생산성 | 대부분 제한 있음 |
| 3 플랫폼 | 대부분의 프로젝트 커버 | 추가 과금 일반적 |
| 전체 엣지케이스 | TC 품질 향상 | 유일한 차별점 |

**$12 가격 근거**:
```
QA 시간 절감: 20시간/월 × $40/시간 = $800
가치의 1.5% = $12

경쟁사 대비:
├── TestRail: $45/user (우리의 3.75배)
├── PractiTest: $35/user (우리의 2.9배)
└── Zephyr: $20/user (우리의 1.7배)
```

### Team Tier 설계

**목적**: 팀 협업, 높은 ARPU

| 추가 기능 | 팀에 필요한 이유 |
|----------|----------------|
| 무제한 멤버 | 팀 확장에 제한 없음 |
| TestRail/Jira 연동 | 기존 워크플로우 통합 |
| 커스텀 템플릿 | 팀 표준화 |
| API 접근 | CI/CD 연동 |
| SSO/SAML | 엔터프라이즈 보안 요구 |

**$22 가격 근거**:
```
Pro 대비 추가 가치:
├── 팀 협업 기능: +$5 가치
├── 연동 기능: +$3 가치
└── 엔터프라이즈 기능: +$2 가치

$12 + $10 = $22 (83% 프리미엄)
```

## 5. Pricing Psychology

### Anchoring (닻 내리기)

```
가격 페이지 배치:

┌────────────────────────────────────────────────────────┐
│                                                         │
│    Free          Pro            Team                    │
│    $0            $12/user       $22/user                │
│                  ▲ POPULAR                              │
│                                                         │
└────────────────────────────────────────────────────────┘

Team을 먼저 보여주어 Pro가 합리적으로 느껴지게
```

### Decoy Effect (미끼 효과)

```
Pro와 Team 사이 기능 차이를 명확하게:

Pro ($12):   무제한 TC, 3플랫폼
Team ($22):  + 무제한 플랫폼, API, SSO

83% 프리미엄에 확실한 추가 가치
→ 필요한 팀은 Team 선택, 아니면 Pro
```

### Loss Aversion (손실 회피)

```
Free → Pro 업그레이드 시:

"현재 Free 플랜으로 이번 달 5개 TC를 모두 사용했습니다.
 Pro로 업그레이드하면:
 - 지금까지 만든 TC를 90일간 보관
 - 무제한으로 새 TC 생성 가능"
```

### Van Westendorp Survey

```
질문:
1. 어느 가격이 너무 비싸서 고려하지 않겠습니까?
2. 어느 가격이 비싸지만 고려할 만합니까?
3. 어느 가격이 적당합니까?
4. 어느 가격이 너무 싸서 품질이 의심됩니까?

예상 결과:
├── Too Cheap: $5 미만
├── Optimal: $10-15
```
