## 2. Scenario A: Solo Founder (1인 창업자)

### 역할 정의

```
+------------------------------------------------------------------+
|                    Solo Founder                                   |
+------------------------------------------------------------------+
|                                                                   |
|  [You]                                                            |
|   |                                                               |
|   +-- Product: 기획, 요구사항 정의, 우선순위                       |
|   +-- Design: UI/UX, Figma/디자인 시스템                          |
|   +-- Frontend: Next.js, React, Tailwind                         |
|   +-- Backend: API, Database, Auth                               |
|   +-- AI/ML: 프롬프트 엔지니어링, LLM 연동                        |
|   +-- DevOps: 배포, CI/CD, 모니터링                               |
|   +-- Marketing: 랜딩 페이지, 콘텐츠, 커뮤니티                    |
|   +-- CS: 초기 고객 지원                                          |
|                                                                   |
+------------------------------------------------------------------+
```

### 장단점

| 장점 | 단점 |
|-----|------|
| 의사결정 빠름 | 번아웃 위험 |
| 비용 최소화 | 개발 기간 길어짐 (5개월) |
| 모든 것을 파악 | 전문성 분산 |
| 지분 100% | 피드백 부족 |

### 생존 전략

```
1. 철저한 우선순위:
   - Week 1-8: 핵심 기능만 집중
   - Week 9-16: 피드백 기반 개선
   - Week 17-20: 런칭 준비

2. 아웃소싱 활용:
   - 디자인: Figma 템플릿 또는 디자이너 1회성 계약
   - 랜딩 페이지: 템플릿 구매
   - 로고/브랜딩: 간단히 시작, 후에 개선

3. AI 도구 최대 활용:
   - GitHub Copilot: 코딩 속도 30% 향상
   - Claude/ChatGPT: 코드 리뷰, 문서 작성
   - v0.dev: UI 컴포넌트 빠른 생성

4. 검증된 스택만:
   - 새로운 기술 학습 시간 최소화
   - Next.js + Supabase (검증된 조합)
```

### 일정 (20주)

| Phase | 기간 | 주요 업무 |
|-------|------|----------|
| Foundation | 1-4주 | 셋업, 인증, 기본 UI |
| Core | 5-10주 | TC 생성 엔진 |
| Polish | 11-16주 | 엣지케이스, Export, 결제 |
| Launch | 17-20주 | 랜딩, QA, 런칭 |

### 비용 (1인, 5개월)

| 항목 | 비용 |
|-----|------|
| 인건비 (기회비용) | 3,000만원 |
| 인프라 | 50만원 |
| 도구/서비스 | 50만원 |
| **총계** | **3,100만원** |

## 3. Scenario B: Co-founders (2인)

### 역할 분담

```
+------------------------------------------------------------------+
|                    2-Person Team                                  |
+------------------------------------------------------------------+

[Technical Co-founder]              [Product Co-founder]
========================            ========================
- Backend API                       - Frontend UI
- Database 설계                     - UX Design
- AI/LLM 연동                       - Product 기획
- DevOps/Infra                      - 사용자 리서치
- 시스템 아키텍처                    - Marketing
                                    - CS

공유 업무:
- 코드 리뷰
- 기술 의사결정
- 우선순위 결정

+------------------------------------------------------------------+
```

### 협업 방식

```
Daily:
+----------------------------------------------------------+
|  09:00  Daily Standup (15분)                             |
|         - 어제 한 일                                      |
|         - 오늘 할 일                                      |
|         - 블로커                                          |
+----------------------------------------------------------+

Weekly:
+----------------------------------------------------------+
|  월요일  Sprint Planning (1시간)                         |
|         - 주간 목표 설정                                  |
|         - 태스크 분배                                     |
|                                                          |
|  금요일  Sprint Review (30분)                            |
|         - 주간 성과 리뷰                                  |
|         - 다음 주 준비                                    |
+----------------------------------------------------------+
```

### 장단점

| 장점 | 단점 |
|-----|------|
| 역할 분담으로 집중 | 의견 충돌 가능 |
| 서로 동기 부여 | 지분 분배 이슈 |
| 기술+제품 균형 | 커뮤니케이션 비용 |
| 더 빠른 개발 (12주) | 초기 비용 증가 |

### 일정 (12주)

| Phase | 기간 | Tech Co-founder | Product Co-founder |
|-------|------|-----------------|-------------------|
| Foundation | 1-3주 | 인프라, Auth | UI 시스템, UX |
| Core | 4-7주 | AI 엔진, API | TC 목록/편집 UI |
| Polish | 8-10주 | 성능, 보안 | 랜딩, 결제 UI |
| Launch | 11-12주 | 배포, 모니터링 | 마케팅, CS 준비 |

### 비용 (2인, 3개월)

| 항목 | 비용 |
|-----|------|
| Tech Co-founder | 1,800만원 |
| Product Co-founder | 1,650만원 |
| 인프라 | 50만원 |
| 도구/서비스 | 100만원 |
| **총계** | **3,600만원** |

## 6. Hiring Recommendations

### 채용 우선순위

```
+------------------------------------------------------------------+
|                    Hiring Priority                                |
+------------------------------------------------------------------+

1순위: Tech Lead / Full-stack (필수)
   - MVP의 핵심
   - 없으면 프로젝트 불가
   - 채용 난이도: 높음

2순위: Frontend Developer
   - UI/UX 품질 결정
   - Tech Lead가 커버 가능하나 부담
   - 채용 난이도: 중간

3순위: AI Engineer (Part-time)
   - 프롬프트 품질이 핵심 차별점
   - 외주/프리랜서 가능
   - 채용 난이도: 중간

4순위: Product Manager
   - 초기엔 창업자가 대체 가능
   - 성장 단계에 필요
   - 채용 난이도: 중간

+------------------------------------------------------------------+
```
