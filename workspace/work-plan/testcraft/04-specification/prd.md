## 1. 제품 개요

### 1.1 제품 비전

> **"기획서만 넣으면, 플랫폼별 엣지케이스까지 포함된 테스트케이스가 5분 만에 완성됩니다"**

TestCraft는 PRD/기획서를 분석하여 플랫폼별(Android, iOS, Web, PC) 상세 테스트케이스를 자동 생성하는 AI 기반 SaaS입니다.

### 1.2 제품 목표

| 목표 | 측정 지표 | 목표값 |
|-----|----------|--------|
| 테스트케이스 작성 시간 단축 | 평균 작성 시간 | 1-2주 → 5분 (95% 단축) |
| 엣지케이스 커버리지 향상 | 자동 생성 엣지케이스 수 | 기능당 평균 15+ 케이스 |
| QA 생산성 향상 | 월간 TC 생성량 | 10배 증가 |
| 사용자 만족도 | NPS | 50+ |

### 1.3 성공 기준 (MVP)

```yaml
launch_criteria:
  - 100명 Active User 확보
  - 70% 이상 사용자 만족도
  - 10% 유료 전환율
  - PRD→TC 변환 정확도 80%+

business_metrics:
  # MVP 시점 (100 MAU × 10% × $20 = $200 MRR)
  - MRR: $200 (MVP 출시 시)
  # 6개월 목표 (2,500 MAU × 10% × $20 = $5,000 MRR)
  - MRR: $5,000 (6개월 내, MAU 2,500 필요)
  - CAC: < $100
  - LTV: > $300
  - ARPU: $20 (Basic $10, Pro $25 가중평균)
```

> ⚠️ **메트릭 정합성 검증**: MRR $5,000 달성을 위해서는 MAU 2,500명 이상 필요
> (계산: 2,500 MAU × 10% 전환율 × $20 ARPU = $5,000 MRR)

##### 인수 조건 (Acceptance Criteria)

**F-001 회원가입/로그인**
```gherkin
Given 미가입 사용자가 회원가입 페이지에 접속했을 때
When 유효한 이메일과 비밀번호를 입력하고 가입 버튼을 클릭하면
Then 계정이 생성되고 이메일 인증 메일이 발송된다

Given 가입된 사용자가 로그인 페이지에 접속했을 때
When Google OAuth 버튼을 클릭하면
Then Google 인증 후 대시보드로 리다이렉트된다
```

**F-003 PRD 업로드**
```gherkin
Given 로그인한 사용자가 프로젝트 상세 페이지에 있을 때
When 10MB 이하의 PDF 파일을 업로드하면
Then 파일이 저장되고 업로드 완료 메시지가 표시된다

Given 사용자가 PDF 파일을 업로드할 때
When 파일 크기가 10MB를 초과하면
Then "파일 크기는 10MB 이하여야 합니다" 오류 메시지가 표시된다
```

**F-005 AI TC 생성**
```gherkin
Given 사용자가 PDF를 업로드하고 플랫폼을 선택했을 때
When TC 생성 버튼을 클릭하면
Then 180초 이내에 TC 목록이 생성되고 진행률이 표시된다

Given TC 생성이 진행 중일 때
When 사용자가 취소 버튼을 클릭하면
Then 생성이 중단되고 부분 결과가 저장된다
```

**F-009 Excel Export**
```gherkin
Given TC 목록이 존재하는 프로젝트에서
When Excel Export 버튼을 클릭하면
Then .xlsx 파일이 다운로드되고 모든 TC 정보가 포함된다
```

## 5. 비기능 요구사항

### 5.1 성능 요구사항

| 항목 | 요구사항 | 측정 기준 | 비고 |
|-----|---------|----------|------|
| TC 생성 시간 | < 180초 (10페이지 PRD) | P95 기준 | AI 분석 + 생성 단계 |
| 페이지 로드 시간 | < 2초 | LCP 기준 | |
| API 응답 시간 | < 500ms | P95 기준 | TC 생성 제외 |
| 동시 사용자 | 100명 (MVP) | 부하 테스트 | 100 MAU 기준 |
| 업타임 | 99.5%+ | 월간 기준 | |

> 📝 **시간 지표 정합성**: 제품 비전 "PRD→TC 5분" = 업로드(30초) + AI 생성(180초) + 후처리(90초) + 검토 여유(60초)
