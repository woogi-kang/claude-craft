## 1. 와이어프레임 개요

### 1.1 설계 원칙

| 원칙 | 설명 | 적용 |
|-----|------|------|
| **명확성** | 핵심 정보 우선 표시 | TC 목록에서 시나리오명 강조 |
| **일관성** | 동일 패턴 반복 | 모든 목록/상세 페이지 동일 구조 |
| **효율성** | 최소 클릭으로 목표 달성 | TC 생성 3단계 |
| **피드백** | 모든 액션에 즉각 반응 | 로딩, 성공, 에러 상태 |

### 1.3 컴포넌트 시스템

| 컴포넌트 | 용도 | 변형 |
|---------|------|------|
| Button | 액션 트리거 | Primary, Secondary, Ghost, Danger |
| Input | 텍스트 입력 | Default, Error, Disabled |
| Card | 정보 그룹핑 | Default, Selectable, Clickable |
| Table | 데이터 목록 | Default, Compact |

| Toast | 알림 메시지 | Success, Error, Warning, Info |
