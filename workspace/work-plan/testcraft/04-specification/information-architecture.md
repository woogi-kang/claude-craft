## 1. IA 개요

### 1.1 설계 원칙

| 원칙 | 설명 | 적용 |
|-----|------|------|
| **단순성** | 3클릭 이내 핵심 기능 접근 | TC 생성까지 3단계 |
| **일관성** | 동일한 패턴 반복 | 모든 목록/상세 동일 구조 |
| **가시성** | 현재 위치 명확히 | Breadcrumb, 활성 메뉴 |
| **효율성** | 반복 작업 최소화 | 단축키, 일괄 처리 |

### 1.2 사용자 멘탈 모델

```
┌─────────────────────────────────────────────────────────────────┐
│                    사용자 멘탈 모델                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   [나의 프로젝트들]                                               │
│         │                                                        │
│         ├── 프로젝트 A                                           │
│         │     ├── 기획서 (PRD)                                   │
│         │     ├── 테스트케이스                                    │
│         │     │     ├── 기능별 그룹                              │
│         │     │     └── 플랫폼별 필터                            │
│         │     └── 팀원                                           │
│         │                                                        │
│         └── 프로젝트 B                                           │
│               └── ...                                            │
│                                                                  │
│   [핵심 플로우]                                                   │
│   기획서 업로드 → 플랫폼 선택 → TC 생성 → 검토 → Export          │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 3. 네비게이션 구조

### 3.1 Global Navigation (GNB)

```
┌─────────────────────────────────────────────────────────────────┐
│  [Logo]        Dashboard    Projects    Pricing    [User Menu]  │
└─────────────────────────────────────────────────────────────────┘
                                                          │
                                                   ┌──────┴──────┐
                                                   │ 프로필      │
                                                   │ 설정        │
                                                   │ ─────────── │
                                                   │ 로그아웃    │
                                                   └─────────────┘
```

| 요소 | 설명 | 비고 |
|-----|------|------|
| Logo | 홈(Dashboard)으로 이동 | |
| Dashboard | 대시보드 | AUTH 시 표시 |
| Projects | 프로젝트 드롭다운 (스위처) | AUTH 시 표시, 최근 5개 + 전체 보기 |
| Pricing | 가격 정책 | 항상 표시 |
| User Menu | 드롭다운 | AUTH 시 표시 |

### 3.1.1 프로젝트 스위처 (Projects 드롭다운)

```
┌─────────────────────────┐
│  최근 프로젝트           │
├─────────────────────────┤
│  ▸ TestCraft MVP        │ ← 현재 프로젝트
│  ▸ E-commerce App       │
│  ▸ Admin Dashboard      │
├─────────────────────────┤
│  [모든 프로젝트 보기]    │
│  [+ 새 프로젝트]         │
└─────────────────────────┘
```

프로젝트 간 빠른 전환을 위한 GNB 드롭다운 제공.

### 3.2 프로젝트 내 네비게이션 (LNB)

```
┌─────────────────┐
│  [프로젝트명]   │
│  └─ 설정 아이콘 │
├─────────────────┤
│                 │
│  ▸ Overview     │ ← 기본 선택
│  ▸ Upload       │
│  ▸ Test Cases   │
│  ▸ Traceability │ [P1]
│  ▸ Analytics    │ [P2]
│                 │
├─────────────────┤
│  Settings       │
└─────────────────┘
```

| 메뉴 | 설명 | 우선순위 |
|-----|------|---------|
| Overview | 프로젝트 요약 | P0 |
| Upload | PRD 업로드 & TC 생성 | P0 |
| Test Cases | TC 목록/관리 | P0 |
| Traceability | 요구사항 추적 | P1 |
| Analytics | 분석 대시보드 | P2 |
| Settings | 프로젝트 설정 | P0 |

### 3.3 Breadcrumb

```
Dashboard > 프로젝트명 > Test Cases > TC-001
```

| 레벨 | 예시 |
|-----|------|
| 1 | Dashboard |
| 2 | 프로젝트명 (링크) |
| 3 | Test Cases / Upload / Settings |
| 4 | TC-001 (상세 시) |
