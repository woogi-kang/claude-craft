### 3.3 AS-IS 요약

| 지표 | 수치 |
|-----|------|
| 총 소요 시간 | 30시간 (약 4일) |
| 작성된 TC | 98개 |
| 리뷰 후 추가 TC | 13개 (13% 누락률) |
| 야근 | 1일 |
| 감정 곡선 | 보통 → 부담 → 지침 → 압박 → 좌절 |

**핵심 문제점**:
1. ⏱️ **시간**: TC 작성에 4일 소요 (업무의 50%)
2. 🎯 **품질**: 13%의 케이스가 리뷰 후 추가 (놓침)
3. 😫 **감정**: 지속적인 부담감과 불완전감
4. 🔄 **변경 대응**: 기획 변경 시 연쇄 수정 필요
