**핵심 차별화:**

1. **진단 퍼널(무료)의 부재**: 현재 시장에 "병원 홈페이지 URL 입력 -> AI 자동 진단 -> 상세 리포트" 형태의 무료 도구가 없음
2. **SEO + GEO + AEO 통합**: 기존 병원 SEO 업체는 전통 SEO 중심. GEO/AEO는 2025~2026년 신규 영역으로 선점 기회

4. **1인 개발자의 비용 효율**: AI 에이전트를 활용한 빠른 진단/최적화로 기존 에이전시 대비 합리적 가격 가능

**평가: POSITIVE** - GEO/AEO + 의료관광 특화의 교차점에 명확한 틈새가 존재. 무료 진단 도구라는 리드 제너레이션 전략이 경쟁자에게 없는 독자적 요소.

### 3.3 가치 제안 검증

**"불공평한 시장에서 공평한 경쟁 기회를 준다"**

이 가치 제안의 강점:

1. **감정적 공감**: 병원장 입장에서 "내 병원 실력은 좋은데 온라인에서 안 보인다"는 명확한 좌절감
2. **데이터 기반 설득**: 진단 리포트가 객관적 수치로 문제를 보여줌 (경쟁사 대비 노출 순위, 기술 점수 등)
3. **즉시 행동 유도**: "이 점수로는 구글 검색에서 절대 안 나옵니다" → 긴급성 생성

**약점 및 보완 필요:**

- "불공평"이라는 표현이 다소 추상적 → "현재 귀원은 구글 검색에서 경쟁 병원 대비 XX점 뒤처져 있습니다"로 구체화 필요
- 진단에서 영업으로의 전환율이 관건 → 진단 리포트의 질이 핵심

**평가: STRONG POSITIVE** - 문제가 실재하며, 시장 데이터로 뒷받침됨. 특히 피부과/성형외과의 폭발적 성장(+194.9%)은 경쟁 심화를 의미하며, SEO/GEO 최적화 수요가 급증할 것을 시사.

### 4.2 유닛 이코노믹스 (Unit Economics)

**고객 획득 비용 (CAC) 추정:**

| 채널 | 예상 CAC | 설명 |
|------|---------|------|
| 무료 진단 도구 → 영업 | 30~70만원 | 서버 비용 + 영업 시간(상담 2h + 제안서 3h + 협상 2h, 시간당 5만원) |
| 의료관광 학회/컨퍼런스 | 50~80만원 | 부스/참가비 + 교통 + 영업 시간 |
| 레퍼럴 (기존 고객 소개) | 10~30만원 | 인센티브 + 영업 시간 |
| 콘텐츠 마케팅 (블로그/SNS) | 20~50만원 | 콘텐츠 제작 + 영업 시간 |

> **주의**: 1인 개발자의 영업 시간 비용(시간당 ~5만원)을 반드시 포함해야 함.
> 상담 2h + 제안서 3h + 협상 2h = 35만원의 숨겨진 CAC 존재.

**고객 생애 가치 (LTV) 추정:**

```
[낙관적 시나리오 - 24개월 리텐션]
최초 프로젝트:        800만원 (SEO/GEO 최적화)
월 유지보수:          50만원 x 24개월 = 1,200만원
추가 프로젝트:        500만원 (LINE/WeChat 대시보드)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
LTV (24개월):        2,500만원

[보수적 시나리오 - 12개월 리텐션, 80% 구독 전환]  ← 기본 가정
최초 프로젝트:        500만원 (진입가 기준)
월 유지보수:          50만원 x 12개월 x 80% = 480만원
추가 프로젝트:        200만원 (일부만 추가 구매)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
LTV (12개월):        1,180만원

LTV / CAC 비율 (보수적): 1,180만원 / 70만원 = 16.9:1 (우수)
LTV / CAC 비율 (낙관적): 2,500만원 / 30만원 = 83:1
```

### 7.4 검증해야 할 핵심 가설

| 가설 | 검증 방법 | 성공 기준 | 시기 |
|------|----------|----------|------|
| 병원장이 무료 진단 도구를 사용할 것이다 | 랜딩페이지 트래픽, 진단 실행 수 | 월 100회 이상 진단 실행 | Month 2 |
| 진단 결과를 보면 상담을 요청할 것이다 | 진단 → 상담 전환율 | 5% 이상 전환 | Month 3 |
| 500만원 이상 지불할 의사가 있다 | 실제 계약 체결 | 3개 이상 계약 | Month 3-4 |
| 최적화 후 실제 검색 노출이 개선된다 | 구글 순위, AI 검색 언급 빈도 | 3개월 내 측정 가능한 개선 | Month 6 |
| 월 구독으로 전환할 것이다 | 프로젝트 고객 → 구독 전환율 | 50% 이상 | Month 6 |

### 7.5 피봇 시나리오

만약 핵심 가설이 실패할 경우:

| 실패 상황 | 피봇 방향 |
|-----------|----------|
| 병원이 SEO에 돈을 쓰지 않음 | → 의료관광 에이전시/플랫폼에 SaaS 진단 도구 B2B 판매 |
| 영업이 안 됨 | → 진단 도구를 SEO 교육 콘텐츠와 결합하여 인바운드 리드 강화 |
| 가격 저항이 큼 | → 월 구독 모델로 전환 (초기 비용 0, 월 50~100만원) |
| 한국 시장이 작음 | → 동남아 의료관광 시장으로 확장 (태국, 인도 병원 타겟) |
