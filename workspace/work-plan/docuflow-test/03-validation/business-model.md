# DocuFlow - Business Model

> 작성일: 2026-01-16
> 버전: 1.0
> 상태: Draft

---

## Executive Summary

DocuFlow는 **프리미엄 SaaS 모델**을 채택하며, 개인 개발자부터 엔터프라이즈까지 확장 가능한 수익 구조를 구축합니다. 핵심은 **낮은 CAC + 높은 LTV**를 통한 지속 가능한 성장입니다.

## 1. 비즈니스 모델 유형

### 선택 모델: Freemium SaaS + Usage-Based Hybrid

```

│                      DocuFlow 수익 모델                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │

│   │    Free      │ ──▶ │     Pro      │ ──▶ │    Team      │   │
│   │   (Hook)     │     │  (Convert)   │     │   (Expand)   │   │
│   └──────────────┘     └──────────────┘     └──────────────┘   │
│          │                    │                    │            │

│          ▼                    ▼                    ▼            │
│   ┌──────────────────────────────────────────────────────────┐ │

### 모델 선택 근거
| 요소 | 선택 | 이유 |
|-----|------|-----|
| **진입 장벽** | Freemium | 개발자 신뢰 획득 필요 |
| **확장성** | Seat-based | 팀 성장에 따른 자연스러운 확장 |

## 2. 가치 사슬 (Value Chain)

### 가치 흐름
```
[개발자 코드]
      │
      ▼
[DocuFlow 파싱 엔진] ← 코드 분석 가치
      │
      ▼
[AI 변환 엔진] ← 자연어 변환 가치
      │
      ▼
[문서 생성] ← 시간 절감 가치
      │
      ▼
[호스팅 + CDN] ← 배포 편의성 가치
      │
      ▼
[API 문서 공개] ← 비즈니스 가치 (신뢰, 개발자 경험)
```

### 각 단계별 가치
| 단계 | 제공 가치 | 과금 가능성 |
|-----|---------|-----------|
| 파싱 | 자동화 | 기본 제공 |
| AI 변환 | 품질 향상 | 사용량 과금 |
| 문서 생성 | 시간 절감 | 기본 제공 |
| 호스팅 | 편의성 | 프리미엄 |
| 커스터마이징 | 브랜딩 | 프리미엄 |

### 4.2 CAC 상세 분석

| 채널 | 예상 CAC | 비중 | 품질 |
|-----|---------|-----|------|
| **Organic (SEO)** | $10 | 30% | High |
| **Product Hunt** | $5 | 10% | Medium |
| **GitHub Marketplace** | $15 | 25% | High |
| **Content Marketing** | $25 | 20% | High |
| **Paid Ads** | $80 | 15% | Medium |

**Blended CAC**: $30-40

### 4.3 LTV 계산

```
LTV = ARPU x Gross Margin x Lifetime

ARPU (Average Revenue Per User)
- Free: $0
- Pro: $15/월
- Team: $99/월 (÷5 = $19.8/시트)
- Blended ARPU: ~$25/월

Gross Margin
- 매출 - (인프라 + AI API) = ~80%
- 낮은 Pro 가격으로 AI 비용 비중 최적화

Lifetime (1 / Churn Rate)
- 목표 월간 Churn: 5%
- Lifetime: 20개월

LTV = $25 x 0.80 x 20 = $400
```

### 4.4 LTV:CAC Ratio

```
LTV:CAC = $400 : $40 = 10:1

┌────────────────────────────────────────────────────────────┐
│  비교 벤치마크                                              │
├────────────────────────────────────────────────────────────┤
│  나쁨 (< 3:1)    │░░░░░░░░░░│                              │
│  보통 (3:1)      │██████████│                              │
│  좋음 (5:1)      │██████████████████│                      │
│  우수 (10:1+)    │██████████████████████████████│ DocuFlow │
└────────────────────────────────────────────────────────────┘

낮은 Pro 가격($15)으로 전환율 향상 기대 → LTV:CAC 유지/개선
```

### 4.5 Payback Period

```
Payback = CAC / (ARPU x Gross Margin)
        = $40 / ($25 x 0.80)
        = 2개월

목표: 12개월 이내 → 달성 (여유 있음)
```

## 5. 비용 구조 상세

### 5.1 운영 비용 (OpEx)

#### 인프라 비용
| 항목 | 월 비용 | 사용자당 | 비고 |
|-----|--------|---------|------|
| 클라우드 (AWS/GCP) | $3,000 | $0.30 | 10K 사용자 기준 |
| AI API (OpenAI) | $2,000 | $0.20 | 문서당 ~$0.02 |
| CDN (CloudFlare) | $500 | $0.05 | Pro 플랜 |
| DB (Supabase) | $500 | $0.05 | Pro 플랜 |
| 모니터링 | $300 | $0.03 | DataDog 등 |
| **합계** | **$6,300** | **$0.63** | |

#### 인건비
| 역할 | 인원 | 월 비용 | 비고 |
|-----|------|--------|------|
| 풀스택 개발자 | 2명 | $12,000 | 시니어 $6K |
| ML 엔지니어 | 1명 | $7,000 | AI 파이프라인 |
| 디자이너 | 0.5명 | $2,500 | 파트타임 |
| DevOps | 0.5명 | $3,000 | 파트타임 |
| **합계** | **4명** | **$24,500** | |

#### 기타 운영비
| 항목 | 월 비용 |
|-----|--------|
| SaaS 도구 | $500 |
| 법무/회계 | $500 |
| 마케팅 | $2,000 |
| 기타 | $500 |
| **합계** | **$3,500** |

### 5.2 총 월간 운영 비용

```
인프라:     $6,300   (18%)
인건비:    $24,500   (72%)
기타:       $3,500   (10%)
────────────────────────────
합계:      $34,300
```

### 5.3 손익분기점 (Break-Even)

```
월 운영비용: $34,300
Gross Margin: 80%
평균 ARPU: $25

필요 유료 사용자 = $34,300 / ($25 x 0.80)
                = 1,715명

Free 포함 전체 사용자 (전환율 7% 가정 - 낮은 가격으로 전환율 상승)
= 1,715 / 0.07
= 24,500명
```

### 6.2 시나리오별 분석

```
가정: 전환율 3%, Churn 7%, 느린 성장
Year 3 ARR: $2.5M
```

#### Base Case (기본)
```
가정: 전환율 5%, Churn 5%, 안정 성장
Year 3 ARR: $4.2M
```

```
