### 2.2 상세 산정 근거

#### TAM (Total Addressable Market)

| 항목 | 수치 | 근거 |
|------|------|------|
| 전 세계 API 관리 시장 | $34B (2026) | Gartner, MarketsandMarkets 리포트 기반 |
| 연평균 성장률 (CAGR) | 21.5% | 2022-2028 예측 |
| 주요 성장 동인 | - | 클라우드 전환, 마이크로서비스, API 경제 확대 |

#### SAM (Serviceable Addressable Market)

| 항목 | 수치 | 산정 방식 |
|------|------|-----------|
| API 문서화 도구 시장 | $2.7B | TAM의 약 8% (문서화 세그먼트) |
| 대상 사용자 수 | 480만 명 | 전 세계 API 개발자 2,400만 x 문서화 도구 사용률 20% |
| ARPU (연간) | $562 | $2.7B / 480만 명 |

#### SOM (Serviceable Obtainable Market)

| 항목 | 수치 | 산정 방식 |
|------|------|-----------|
| 스타트업 셀프서브 시장 | $270M | SAM의 10% (스타트업 + 셀프서브) |
| 초기 타겟 사용자 | 48만 명 | SAM의 10% (스타트업 개발자) |
| 1차년도 목표 | $500K | SOM의 0.2% (MAU 1,000 x ARPU $500) |

### 2.3 시장 규모 검증

| 검증 방법 | 결과 | 신뢰도 |
|-----------|------|--------|
| Top-down (시장 리포트) | $270M | 중간 |
| Bottom-up (타겟 사용자 x ARPU) | $240M | 높음 |
