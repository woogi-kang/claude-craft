```

### 2.2 상세 설문 문항

### 3.2 질문 가이드

#### Part 1: 현재 상황 탐색 (10분)

**오프닝 질문**
> "최근 업무에서 API 문서를 작성하거나 수정한 경험을 말씀해 주시겠어요?
> 어떤 상황이었고, 어떤 과정을 거쳤나요?"

**후속 질문**
- "그 작업에 얼마나 시간이 걸렸나요?"
- "평소에도 비슷한 빈도로 문서 작업을 하시나요?"
- "팀에서 문서화는 누가 주로 담당하나요?"

**도구 탐색**
> "현재 어떤 도구로 API 문서를 관리하고 계신가요?"

**후속 질문**
- "그 도구를 선택한 이유가 있나요?"
- "처음 설정할 때 어땠나요?"
- "팀원들은 그 도구에 만족하나요?"

#### Part 2: 문제 심층 탐색 (10분)

**핵심 질문**
> "API 문서화와 관련해서 가장 힘들었거나 짜증났던 경험이 있으신가요?
> 구체적으로 말씀해 주시면 좋겠습니다."

**5 Whys 적용**
- "그게 왜 문제였나요?"
- "그로 인해 어떤 일이 생겼나요?"
- "그 상황이 자주 발생하나요?"
- "해결하려고 시도해 본 적 있으신가요?"
- "왜 해결이 안 됐을까요?"

**Swagger 사용자 추가 질문**
> "Swagger/OpenAPI를 사용하실 때 가장 불편한 점은 뭔가요?"

**후속 질문**
- "YAML 작성은 어떠신가요?"
- "문서와 코드가 맞지 않았던 적 있나요?"
- "다른 도구로 바꾸고 싶은 적 있으신가요?"

#### Part 3: 솔루션 반응 (10분)

**컨셉 소개**
> "제가 지금 개발 중인 도구에 대해 간단히 소개해 드릴게요.
> [DocuFlow 컨셉 설명]
> 첫인상이 어떠신가요?"

**반응 탐색**
- "가장 끌리는 부분이 있다면요?"
- "반대로 걱정되는 부분은요?"
- "현재 쓰시는 도구에서 이 도구로 바꿀 의향이 있으신가요?"
- "어떤 조건이 충족되면 바꾸실 것 같아요?"

**가격 탐색**
> "이런 도구에 월 얼마 정도면 적정하다고 생각하세요?"

**후속 질문**
- "무료 플랜에서는 어떤 기능을 기대하세요?"
- "회사에서 비용 승인은 어떤 과정을 거치나요?"

#### Part 4: 마무리 (5분)

**이상향 질문**
> "마법 지팡이가 있다고 가정해 볼게요.
> 완벽한 API 문서화 도구가 있다면 어떤 모습일까요?"

**추가 수집**
- "제가 미처 물어보지 못한 것 중에 하고 싶은 말씀 있으신가요?"
- "비슷한 고민을 하는 동료 분을 소개해 주실 수 있나요?"

│   - 포스트잇 형태로 정리 (1인용구 = 1포스트잇)                  │
│                                                                  │
│   Step 2: 그룹핑                                                │
│   ─────────────────────────────────────────────────────────────  │
│   - 유사한 인용구끼리 그룹화                                     │
│   - 각 그룹에 테마 라벨 부여                                     │
│                                                                  │
│   Step 3: 패턴 발견                                             │
│   ─────────────────────────────────────────────────────────────  │
│   - 반복되는 테마 식별                                           │
│   - 예상 외 인사이트 표시                                        │
│                                                                  │
│   Step 4: 인사이트 도출                                         │
│   ─────────────────────────────────────────────────────────────  │
│   - "우리가 배운 것" 정리                                        │
│   - 액션 아이템 도출                                             │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 4.3 분석 템플릿
