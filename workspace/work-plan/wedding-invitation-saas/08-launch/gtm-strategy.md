# 모바일 청첩장 SaaS - GTM (Go-to-Market) 전략

## 1. GTM 핵심 원칙

### 제품 자체가 마케팅이다

모바일 청첩장의 특수성:
- 청첩장을 받는 하객 200-300명이 자연스럽게 서비스에 노출
- 인상적인 청첩장은 "이거 어디서 만들었어?"라는 질문을 유발
- 아케이드 모드는 SNS 공유 욕구를 자극

```
1명 구매 → 200-300명 노출 → 3-5명 문의 → 1-2명 전환
바이럴 계수: 1건 구매 → 1-2건 추가 구매 (R > 1 가능)
```

## 2. 채널 전략 (ORB Framework)

### Owned Channels (소유 채널)

| 채널 | 전략 | 우선순위 |
|------|------|---------|
| **자체 웹사이트** | SEO 최적화 랜딩페이지, 블로그 | 1순위 |
| **블로그 (SEO)** | "모바일 청첩장 만들기" 가이드, 웨딩 준비 콘텐츠 | 1순위 |
| **이메일** | 결제 후 만족도 조사 + 후기 요청 | 2순위 |

**SEO 키워드 전략**:
```
주요 키워드 (검색량 높음):
- "모바일 청첩장" / "모바일 청첩장 만들기"
- "모바일 청첩장 무료" / "모바일 청첩장 추천"
- "예쁜 청첩장" / "특별한 청첩장"

롱테일 키워드 (전환율 높음):
- "인터랙티브 모바일 청첩장"
- "게임 청첩장" / "RPG 청첩장"
- "프리미엄 모바일 청첩장"
- "모바일 청첩장 직접 만들기 에디터"
```

### Rented Channels (임대 채널)

| 채널 | 전략 | 전환 구조 | 우선순위 |
|------|------|-----------|---------|
| **인스타그램** | 완성된 청첩장 쇼케이스, Before/After, 제작 과정 릴스 | 릴스/피드 → 프로필 링크 → 자체 사이트 | 1순위 |
| **네이버 스마트스토어** | 상품 3개 등록 (Basic/Premium/Arcade) + 검색 트래픽 | "모바일 청첩장" 검색 → 상품 → 구매 후 에디터 URL 발송 | 1순위 |
| **카카오톡 스토어** | 카카오 생태계 내 구매 완결 | 카카오톡 공유 → 스토어 → 결제 → 에디터 | 2순위 |
| **네이버 블로그** | 체험단 후기, 웨딩 준비 가이드, 비교글 | 검색 유입 → 블로그 → 자체 사이트 | 2순위 |
| **유튜브 쇼츠** | 아케이드 모드 플레이 영상, "이런 청첩장 본 적 있어?" | 쇼츠 → 설명란 링크 → 자체 사이트 | 2순위 |
| **아이디어스** | "수제/커스텀" 포지셔닝, MZ 여성 유저 타겟 | "청첩장" 검색 → 상품 → 주문 | 3순위 |
| **크몽** | "청첩장 제작" 카테고리 입점 | 견적 요청 → 상담 → 제작 | 3순위 |

**인스타그램 콘텐츠 전략**:
```
주 3-4회 포스팅:
- 월: 완성된 청첩장 쇼케이스 (캐러셀)
- 수: 제작 과정 릴스 (15-30초)
- 금: 아케이드 모드 플레이 영상
- (선택) 토: 고객 후기 / Q&A
```

### Borrowed Channels (차용 채널)

| 채널 | 전략 | 우선순위 |
|------|------|---------|
| **웨딩 커뮤니티** | 더웨딩, 웨딩의여신, 결혼준비 카페 후기 | 1순위 |
| **청첩장 자체 바이럴** | 하단 CTA + 아케이드 엔딩 CTA (CAC 0원) | 1순위 |
| **카카오톡 채널** | CS 상담 + 완성 후 리뷰 유도 + 프로모션 발송 | 2순위 |
| **IT 커뮤니티** | 사이드 프로젝트 소개 (개발자 커뮤니티) | 2순위 |
| **웨딩 플래너 협업** | 플래너가 고객에게 추천 (수수료 제공) | 3순위 |
| **체험단/인플루언서** | 웨딩 인스타그래머 무료 체험 + 후기 | 2순위 |

## 3. 5단계 런칭 전략

### Phase 1: Internal Launch (Week 1-2)

- 지인 5쌍에게 무료 제작 + 피드백 수집
- 아케이드 모드 데모를 개발자 커뮤니티에 공유
- 사용성 이슈 파악 + 수정

### Phase 2: Alpha Launch (Week 3-4)

- 랜딩페이지 오픈 + 사전예약 폼
- "곧 출시됩니다" + 데모 체험
- 인스타그램 계정 오픈 + 티저 콘텐츠
- 목표: 사전예약 100명

### Phase 3: Beta Launch (Week 5-8)

- 사전예약자 50명에게 50% 할인 베타 오픈
- 피드백 수집 + 빠른 반복
- 베타 사용자 후기 수집
- 네이버 블로그 SEO 콘텐츠 발행 시작
- 목표: 유료 결제 20건

### Phase 4: Early Access (Week 9-12)

- 전체 기능 오픈 (일반 가격)
- 네이버 스마트스토어 상품 등록
- 인스타그램 유료 광고 테스트 (월 30만원)
- 웨딩 커뮤니티 후기 마케팅 시작
- 목표: 월 50건

### Phase 5: Full Launch (Week 13+)

- 모든 채널 본격 가동
- 인스타그램 광고 확대
- 체험단/인플루언서 마케팅
- 크몽/아이디어스/카카오톡 스토어 추가 채널
- 목표: 월 100건+

## 6. 프로모션 전략

### 런칭 프로모션

| 프로모션 | 기간 | 내용 | 목표 |
|---------|------|------|------|
| 얼리버드 50% 할인 | 첫 2주 | 전 티어 50% 할인 | 초기 고객 확보 |
| 후기 작성 캐시백 | 상시 | 후기 작성 시 5,000원 페이백 | 사회적 증거 |
| 친구 추천 할인 | 상시 | 추천인/피추천인 각 3,000원 할인 | 바이럴 |
| 웨딩 시즌 프로모션 | 3-5월, 9-11월 | 프리미엄 업그레이드 30% 할인 | 시즌 매출 극대화 |

### 시즌별 마케팅 캘린더

```
1-2월: 봄 웨딩 사전예약 프로모션
3-5월: 봄 웨딩 시즌 (최대 수요기)
6-8월: 비수기 → 가을 사전예약 + 콘텐츠 생산
9-11월: 가을 웨딩 시즌 (최대 수요기)
12월: 연말 회고 + 다음해 준비
```
