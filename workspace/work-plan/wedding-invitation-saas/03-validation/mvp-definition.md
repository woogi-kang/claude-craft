# 모바일 청첩장 SaaS - MVP 정의서

## 1. MVP 목표

### 검증할 핵심 가설

> **"우리가 섹션 기반 에디터와 프리미엄 인터랙션을 제공하면, MZ세대 예비부부가 2만원 이상을 지불하고 모바일 청첩장을 구매할 것이다"**

### 가설 상세

| # | 가설 | 유형 | 리스크 | 검증 방법 |
|---|------|------|--------|----------|
| H1 | 2만원+ 유료 모바일 청첩장 시장이 존재한다 | 가치 | 중간 | 실제 판매 데이터 |
| H2 | 비개발자가 에디터로 15분 내 청첩장 완성 가능 | 사용성 | 높음 | 사용성 테스트 |
| H3 | 아케이드/인터랙티브가 바이럴을 만든다 | 성장 | 중간 | 공유 수 추적 |

### MVP 성공 기준

| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| 월 판매 건수 | 20건+ (M3 기준) | 결제 데이터 |
| 에디터 완성률 | 60%+ (시작 -> 결제) | 퍼널 분석 |
| 카카오 공유 횟수 | 평균 30회/청첩장 | 공유 API 로그 |
| 평균 에디터 완성 시간 | 15분 이하 | 세션 분석 |

**성공 정의**: "출시 3개월 내 월 20건 판매 + 에디터 완성률 60% 이상 달성 시 다음 단계 진행"

## 2. 타겟 사용자 (MVP)

### 얼리 어답터

| 항목 | 내용 |
|------|------|
| 누구 | 20대 후반~30대 초반 IT/크리에이티브 직종 커플 |
| 왜 먼저 | 기술에 관심 많고, "남들과 다른" 청첩장을 원하며, 유료 결제에 거부감 적음 |
| 어디서 | IT 커뮤니티(디스코드, 개발자 카페), 인스타그램 웨딩 해시태그 |
| 목표 수 | 첫 3개월 60명 (월 20명) |

## 3. 핵심 사용자 플로우 (MVP)

```
랜딩페이지 방문
    │
    ▼
데모 청첩장 체험 (아케이드 모드 포함)
    │
    ▼
에디터 시작 (회원가입/로그인)
    │
    ▼
템플릿 선택 → 섹션 on/off → 내용 입력 → 미리보기  ← 핵심 가치 경험
    │
    ▼
결제 (티어 선택) → 청첩장 URL 발급
    │
    ▼
카카오톡 공유
```

## 4. MoSCoW 기능 분류

### Must Have (필수) - MVP에 반드시 포함

| # | 기능 | 설명 | 예상 공수 |
|---|------|------|----------|
| M1 | 에디터 - 섹션 on/off | 각 섹션 활성화/비활성화 토글 | 1주 |
| M2 | 에디터 - 내용 입력 폼 | 텍스트, 사진, 날짜 등 입력 UI | 2주 |
| M3 | 에디터 - 실시간 미리보기 | 편집 중 결과물 미리보기 | 1주 |
| M4 | 템플릿 렌더링 엔진 | 사용자 데이터로 청첩장 동적 생성 | 2주 |
| M5 | 결제 시스템 | 토스페이먼츠 연동, 3개 티어 | 1주 |
| M6 | 고유 URL 발급 | weddingcraft.kr/couple-name | 0.5주 |
| M7 | 카카오톡 공유 | 카카오 SDK 연동 | 0.5주 |
| M8 | 기본 섹션 10개 | Hero, Greeting, CoupleIntro, Gallery, WeddingInfo, Location, Account, Share, Video, Footer | 2주 (기존 코드 활용) |
| M9 | 동적 OG 이미지 | 커플명+날짜 포함 카카오 공유 미리보기 (바이럴 핵심) | 0.5주 |
| M10 | GuestSnap 기본 | 업로드 무제한 + 200GB 무료 다운로드 (전 티어 포함) | 1주 (기존 코드 활용) |
| M11 | 관리자 대시보드 | 주문/고객 관리 기본 (Supabase 콘솔 활용으로 최소화) | 0.5주 |
| M12 | 랜딩페이지 | 서비스 소개 + 데모 + CTA | 1주 |
| M13 | 법적 준비 | 통신판매업 신고, 이용약관, 개인정보 처리방침, 환불 정책 | 0.5주 |

**MVP 예상 총 공수: 13주 (약 3.5개월, 1인 개발 기준, 버퍼 2주 포함 시 15주)**

> **가격 체계**: Basic 19,000원 / Premium 39,000원 / Arcade 49,000원

### Should Have (중요) - MVP 직후 추가

| # | 기능 | 설명 | 포함 안 하는 이유 |
|---|------|------|------------------|
| S1 | 섹션 순서 변경 (drag-and-drop) | 드래그로 섹션 순서 재배치 | 기술적 복잡도, 초기에는 고정 순서로 충분 |
| S2 | 프리미엄 섹션 (Interview, Timeline) | 커플 스토리 섹션 | Basic에 없어도 가치 검증 가능 |
| S3 | GuestSnap 에디터 커스터마이징 | GuestSnap 세부 설정 에디터 | 업로드 무제한은 전 티어 포함, 에디터 커스텀은 이후 |
| S4 | 아케이드 모드 에디터 | 아케이드 모드 커스터마이징 | 초기에는 고정 템플릿으로 제공 |
| S5 | 네이버 스마트스토어 연동 | 주문 자동 처리 | 수동 처리로 시작 가능 |
| S6 | AfterParty 섹션 | 식 종료 후 하객에게 뒤풀이/맛집/주변 행사 안내 | 핵심 가치 검증 후 추가 |

## 6. MVP 기술 절충

| 영역 | MVP | 이후 |
|------|-----|------|
| 에디터 | 폼 기반 (섹션별 입력 폼) | 비주얼 WYSIWYG 에디터 |
| 섹션 순서 | 고정 순서 + on/off | 드래그앤드롭 재배치 |
| 이미지 업로드 | Vercel Blob (단순) | CDN + 최적화 파이프라인 |
| 결제 | 토스페이먼츠 (단순) | 복수 결제사 + 쿠폰 시스템 |
| 아케이드 모드 | 고정 템플릿 (커스텀 불가) | 에디터에서 캐릭터/대사 편집 |
| 관리자 | 기본 CRUD | 대시보드 + 분석 |
| SEO | 기본 메타태그 + **동적 OG** (카카오 공유 필수) | 구조화된 데이터 + 고급 SEO |

## 7. MVP 검증 계획

### 실험 1: 랜딩페이지 전환율

| 항목 | 내용 |
|------|------|
| 검증 가설 | 예비부부가 프리미엄 모바일 청첩장에 관심을 가진다 |
| 방법 | 랜딩페이지 + 데모 + 사전예약 폼 |
| 대상 | 100명+ 방문 |
| 기간 | 2주 |
| 성공 기준 | 사전예약 전환율 5%+ |

### 실험 2: 에디터 사용성

| 항목 | 내용 |
|------|------|
| 검증 가설 | 비개발자가 에디터로 청첩장을 완성할 수 있다 |
| 방법 | 5명 사용성 테스트 (화면 녹화) |
| 대상 | 결혼 예정 비개발자 커플 5쌍 |
| 기간 | 1주 |
| 성공 기준 | 5명 중 4명이 15분 내 완성 |

### 실험 3: 유료 전환

| 항목 | 내용 |
|------|------|
| 검증 가설 | 2만원 이상 유료 결제가 발생한다 |
| 방법 | MVP 소프트 런칭 |
| 대상 | 얼리 어답터 60명 |
| 기간 | 3개월 |
| 성공 기준 | 월 20건 유료 결제 |

## 8. MVP 일정

| 주차 | 활동 | 산출물 |
|------|------|--------|
| W1-2 | Supabase 설정 + DB 스키마 + Auth + 데이터 분리 | 인프라 + JSON 스키마 |
| W3-4 | 에디터 폼 UI + 섹션 on/off + 내용 입력 | 에디터 프로토타입 |
| W5-6 | 템플릿 렌더링 엔진 (기존 코드 리팩토링) + 10개 섹션 | 동적 청첩장 생성 |
| W7-8 | 결제 연동 + URL 발급 + GuestSnap 연동 | 결제→발급 파이프라인 |
| W9-10 | 랜딩페이지 + 동적 OG + 카카오 공유 + 관리자 | 마케팅 + 운영 도구 |
| W11 | 법적 준비 + QA + 사용성 테스트 | 약관/정책 + 안정화 |
| W12-13 | 버그 수정 + 소프트 런칭 (얼리 어답터) | 첫 유료 고객 |
| W14-15 | (버퍼) 피드백 수집 + 개선 | v0.2 |

## 9. 킬 스위치 (Abort Criteria)

> 다음 조건 발생 시 MVP 중단 및 재검토

- 3개월간 누적 30건 미만 판매 (단, 성장 추세가 명확하면 1개월 연장 검토)
- 에디터 완성률 20% 미만 (사용성 실패)
- 환불률 20% 이상 (디지털 상품 기준 심각한 불만족)
- CAC가 12,000원을 초과 (ARPU의 30% 기준)
