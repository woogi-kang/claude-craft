# 모바일 청첩장 SaaS - 경쟁사 분석

## 1. 경쟁 구도 개요

한국 모바일 청첩장 시장은 크게 3가지 유형의 플레이어로 구성된다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    경쟁 구도 맵                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  [전통 청첩장 + 모바일 번들]     [디지털 네이티브]                 │
│  - 보자기카드                    - 데어무드                       │
│  - 바른손카드                    - 필카드                         │
│  - 카드1번가                     - 잇츠카드                       │
│  - 카드마켓                                                      │
│                                                                  │
│  [범용 도구]                     [개인 개발자/프리랜서]            │
│  - Wix                           - GitHub 오픈소스 템플릿         │
│  - Adobe Express                 - 프리랜서 외주 (크몽 등)        │
│  - Canva                                                         │
│                                                                  │
│  ────────────── 포지셔닝 갭 ──────────────                       │
│                                                                  │
│  [프리미엄 인터랙티브 에디터] ← WeddingCraft가 노리는 영역        │
│  - 경쟁자 없음                                                    │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 주요 경쟁사 분석

### 경쟁사 1: 보자기카드

| 항목 | 내용 |
|------|------|
| 포지셔닝 | 청첩장 업계 1위, 브랜드 선호도 5년 연속 1위 |
| 주요 제품 | 종이 청첩장 + 모바일 청첩장 번들 |
| 가격 | 모바일 청첩장 단독 18,000원 (2년), 종이 구매 시 무료 |
| 모바일 기능 | 다양한 스킨, SNS 공유, 위치 안내, 참석 확인, 방명록 |
| 강점 | 높은 브랜드 인지도, 종이+모바일 번들, 무료 샘플 |
| 약점 | 템플릿 고정형, 인터랙티브 요소 부재, 디자인 획일화 |

### 경쟁사 2: 데어무드 (TheIrMood)

| 항목 | 내용 |
|------|------|
| 포지셔닝 | 무료/저가 모바일 청첩장의 대표 |
| 주요 제품 | 모바일 청첩장 전문 |
| 가격 | 기본 무료, 유료 옵션 없음 (광고 기반 추정) |
| 모바일 기능 | 사진 최대 40장, BGM, 세례명, 약도/지도앱 연동, 카톡송금 |
| 강점 | 완전 무료, 카톡송금 수수료 무료, 다양한 디자인 |
| 약점 | 수익 모델 불명확, 프리미엄 기능 부재, 사용 기간 제한 (예식일+30일) |

### 경쟁사 3: 필카드 (FEEL Card)

| 항목 | 내용 |
|------|------|
| 포지셔닝 | 무료 셀프 모바일 청첩장 |
| 주요 제품 | 필카드 FREE (무료), 필메이커 (셀프 제작) |
| 가격 | 기본 무료 (워터마크 포함), 워터마크 제거 유료 |
| 모바일 기능 | 네이버 로그인, 셀프 제작 도구, 평생소장 옵션 |
| 강점 | 네이버 연동, 셀프 제작 가능, 진입장벽 낮음 |
| 약점 | 디자인 제한적, 프리미엄 인터랙션 없음 |

### 경쟁사 4: 잇츠카드

| 항목 | 내용 |
|------|------|
| 포지셔닝 | 종합 웨딩카드 서비스 |
| 주요 제품 | 종이 청첩장 + 모바일 청첩장 + 웨딩 굿즈 |
| 가격 | 합리적 가격대 (종이 청첩장 구매 시 모바일 번들) |
| 모바일 기능 | 표준 기능 세트 |
| 강점 | 웨딩 굿즈까지 원스톱, 합리적 가격 |
| 약점 | 모바일 청첩장 자체의 혁신성 부족 |

### 경쟁사 5: 바른손카드

| 항목 | 내용 |
|------|------|
| 포지셔닝 | 1970년 설립, 50년+ 전통의 청첩장 1위 브랜드 |
| 주요 제품 | 프리미엄 종이 청첩장 + 모바일 청첩장 + 식전영상 |
| 가격 | 종이 청첩장 중심 (모바일은 부가 서비스) |
| 모바일 기능 | 100% 무료 모바일 청첩장, 식전영상 |
| 강점 | 최고 브랜드 인지도, 프리미엄 종이 청첩장 |
| 약점 | 디지털 혁신 느림, 모바일이 주력이 아님 |

## 5. 차별화 기회

### 경쟁사가 제공하지 않는 것

1. **인터랙티브 게이미피케이션**: 아케이드 모드는 시장에 유일한 기능
2. **GuestSnap**: 결혼식 당일 하객 사진/영상 수집 기능
3. **비주얼 에디터**: 섹션 기반 drag-and-drop 편집기
4. **고급 애니메이션**: Framer Motion, Three.js 기반 프리미엄 효과
5. **커플 스토리텔링**: 인터뷰 Q&A, 연애 타임라인 등 이야기 중심 섹션

### 포지셔닝 권장

```
가격
  ^
  |  [바른손]          [WeddingCraft 프리미엄]
  |  [보자기]     [WeddingCraft 기본]
  |
  |  [필카드]  [잇츠카드]
  |
  |  [데어무드]
  +───────────────────────────────> 커스터마이징/인터랙션
      낮음                            높음
```

**권장 포지셔닝**: "프리미엄 인터랙티브 모바일 청첩장"
- 가격: 중상위 (2.9만~8.9만원)
- 차별화: 게이미피케이션 + 인터랙티브 + 에디터
- 타겟: 개성과 기술에 민감한 MZ세대 커플

## 6. 경쟁 위협 모니터링

| 위협 | 확률 | 대응 |
|------|------|------|
| 보자기/바른손이 에디터 기능 출시 | 중간 | 인터랙티브/게임 차별화 강화 |
| 새로운 디지털 네이티브 경쟁자 진입 | 높음 | 빠른 MVP + 커뮤니티 구축 |
| AI 기반 청첩장 자동 생성 서비스 | 중간 | AI 기능 조기 통합 |
| 글로벌 서비스의 한국 진출 (Zola 등) | 낮음 | 한국 특화 기능 (카카오, 축의금) |
