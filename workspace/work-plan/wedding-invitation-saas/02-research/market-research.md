# 모바일 청첩장 SaaS - 시장 분석

## 1. 시장 정의

| 항목 | 내용 |
|------|------|
| 문제 공간 | 결혼을 앞둔 커플의 디지털 청첩장 제작 니즈 |
| 지역 | 대한민국 |
| 고객 세그먼트 | 예비 신혼부부 (20대 후반~30대 중반) |
| 핵심 니즈 | 개성 있고 편리한 모바일 청첩장 제작 및 배포 |

## 2. TAM (Total Addressable Market)

### Top-Down 추정

**기반 데이터**:
- 2025년 한국 혼인 건수: 약 24만 300건 (통계청)
- 2024년: 22만 2천건 (전년비 14.8% 증가)
- 3년 연속 증가 추세 (2023: +1.0%, 2024: +14.8%, 2025: +8.1%)
- 모바일 청첩장 이용률: 추정 85-90% (거의 모든 커플이 모바일 청첩장 사용)

**TAM 계산**:
```
연간 혼인 건수: ~240,000건
모바일 청첩장 이용률: ~87%
모바일 청첩장 잠재 고객: ~209,000쌍/년

평균 지불 가능 금액 (무료~프리미엄):
- 무료/기본: 50% -> 0원
- 유료 (1.5~3만원): 35% -> 평균 2만원
- 프리미엄 (3~10만원): 15% -> 평균 5만원

가중 평균 ARPU: 0.50 x 0 + 0.35 x 20,000 + 0.15 x 50,000 = 14,500원

TAM = 209,000 x 14,500 = 약 30.3억원/년
```

### Bottom-Up 추정

```
유료 전환 고객: 209,000 x 50% = 104,500쌍
평균 객단가: 29,000원 (유료 평균)
TAM (유료 시장만) = 104,500 x 29,000 = 약 30.3억원/년
```

**교차 검증**: 두 방법 모두 약 300억원 규모. 일치.

### 인접 시장 확장 가능성

| 시장 | 규모 추정 | 비고 |
|------|----------|------|
| 돌잔치 초대장 | ~5억원/년 | 출생 건수 기반 |
| 기업 이벤트 초대장 | ~10억원/년 | B2B 확장 |
| 해외 한인 시장 | ~5억원/년 | 미국/일본/동남아 |
| 기타 (환갑, 졸업 등) | ~3억원/년 | 범용 초대장 |

**확장 TAM: 약 50억원/년**

## 3. SAM (Serviceable Addressable Market)

### 서비스 가능 시장 범위

| 제약 조건 | SAM 축소 요인 |
|-----------|-------------|
| 지역 | 한국 내 (글로벌 미진출) |
| 채널 | 온라인 직접 판매 + 스마트스토어 |
| 가격대 | 유료 세그먼트만 (무료 제외) |
| 타겟 | 20대 후반~30대, 디지털 네이티브 |

```
유료 모바일 청첩장 시장 (기본 + 프리미엄): 약 30억원
우리 서비스 타겟 세그먼트 (프리미엄 + 차별화 선호):
  - 전체 유료 시장의 40%가 프리미엄/차별화 관심

SAM = 30억원 x 40% = 약 12억원/년
```

## 4. SOM (Serviceable Obtainable Market)

### 1-3년 달성 가능 시장

```
Year 1 (MVP + 초기 성장):
  - 월 50-100건 판매
  - 연간 ~800건 x 평균 35,000원 = 약 2,800만원

Year 2 (성장기):
  - 월 200-400건 판매
  - 연간 ~3,600건 x 평균 38,000원 = 약 1.4억원

Year 3 (안정기):
  - 월 500-800건 판매
  - 연간 ~7,800건 x 평균 42,000원 = 약 3.3억원
  - SAM 대비 약 27% 점유
```

## 6. 성장 드라이버 및 트렌드

### 긍정적 요인

1. **혼인 건수 반등**: 3년 연속 증가세 (2023-2025), 18년 만의 연속 증가
2. **디지털 전환 가속**: 종이 청첩장에서 모바일로의 전환 지속
3. **개성화 트렌드**: MZ세대의 "나만의 결혼식" 문화 확산
4. **SNS 공유 문화**: 인스타그래머블한 청첩장 수요 증가
5. **기술 발전**: 3D/AR/인터랙티브 웹 기술의 대중화

### 부정적/리스크 요인

1. **장기 혼인율 하락**: 구조적 저출생/비혼 트렌드
2. **무료 서비스 경쟁**: 종이 청첩장 구매 시 무료 모바일 청첩장 제공 관행
3. **짧은 사용 주기**: 1회성 상품 (재구매율 극히 낮음)
4. **시장 천장**: 연간 혼인 건수에 의해 시장 규모 제한
5. **진입장벽 낮음**: 기술적 진입장벽이 상대적으로 낮음
