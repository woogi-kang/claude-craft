# WeddingCraft - 랜딩페이지 상세 UI 설계서

## 1. 페이지 개요

| 항목 | 내용 |
|------|------|
| URL | `weddingcraft.kr` (루트) |
| 렌더링 | SSG (정적 생성, ISR 5분) |
| 목표 | 방문자 → 에디터 시작 전환율 15%+ |
| 타겟 로딩 | LCP < 2.5s, FID < 100ms |
| 모바일 비중 | 70%+ (모바일 퍼스트 설계) |

### [5] 작동 방식 (How It Works)

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│                   "3단계면 끝나요"                                │

│   │          │         │          │         │          │       │
│   └──────────┘         └──────────┘         └──────────┘       │
│                                                                  │

│   자유롭게 조합하고,    결제하면 바로         200-300명에게        │
```

|------|------|

| 핵심 메시지 | "결제 전까지 무료" — 진입 장벽 제거 강조 |
