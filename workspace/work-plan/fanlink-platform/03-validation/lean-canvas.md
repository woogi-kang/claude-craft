## 2. 블록별 상세

### 2.1 Problem (문제)

**인플루언서 측 문제**

| 문제 | 현재 상황 | 고통 지수 |
|------|----------|----------|
| 플랫폼 종속 | 유튜브 알고리즘 변경 시 수익 급감 | 9/10 |
| 팬 데이터 부재 | 구독자 100만이어도 연락처 없음 | 8/10 |
| 수익 불안정 | 월별 광고 수익 30-50% 변동 | 8/10 |
| 진성팬 식별 불가 | 누가 진짜 팬인지 데이터 없음 | 7/10 |
| 커뮤니티 분산 | 유튜브, 디스코드, 팬카페 각각 관리 | 6/10 |

**팬 측 문제**

| 문제 | 현재 상황 | 고통 지수 |
|------|----------|----------|
| 인정받지 못함 | 5년 응원해도 증거/차별화 없음 | 8/10 |
| 소통 불가 | 댓글은 묻히고, DM 불가 | 8/10 |
| 팬활동 분산 | 멤버십, 팬카페, 디스코드 따로 | 7/10 |
| 히스토리 없음 | 응원 기록이 남지 않음 | 6/10 |

**인플루언서 세그먼트**

| 세그먼트 | 규모 | 특성 | 우선순위 |
|----------|------|------|---------|
| 중형 유튜버 (10-100만) | 1.2만명 | 전업 가능, 팬덤 있음 | **P0** |
| 소형 유튜버 (1-10만) | 6만명 | 성장 중, 수익화 니즈 | P1 |
| 게임 스트리머 | 2만명 | 실시간 소통, 충성 팬 | P1 |
| 버튜버 | 3천명 | 높은 충성도, 굿즈 수요 | P2 |

**팬 세그먼트**

| 세그먼트 | 월 지출 | 특성 | 우선순위 |
|----------|--------|------|---------|

| Super Fan | 10만원+ | 올인, 모든 활동 참여 | P1 |

### 2.4 Solution (솔루션)

| 문제 | 솔루션 | 핵심 기능 |
|------|--------|----------|
| 팬 데이터 부재 | **팬 분석 대시보드** | 팬 목록, 활동 분석, 이탈 예측 |
| 진성팬 식별 불가 | **팬 레벨 시스템** | 자동 포인트, 레벨업, 티어 |
| 인정받지 못함 | **배지/히스토리** | 업적 배지, 응원 기록, 프로필 |
| 커뮤니티 분산 | **통합 플랫폼** | 멤버십+커뮤니티+굿즈 원스톱 |
| 수익 불안정 | **다중 수익원** | 구독, 굿즈, 이벤트, 후원 |

| # | 가설 | 검증 방법 | 성공 기준 | 우선순위 |
|---|------|----------|----------|---------|
