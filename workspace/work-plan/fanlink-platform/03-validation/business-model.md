# FanLink - 비즈니스 모델

## 1. 비즈니스 모델 개요

### 1.1 모델 유형

**양면 플랫폼 (Two-Sided Platform) + 수수료 기반 (Commission-Based)**

```
┌─────────────────────────────────────────────────────────────────┐
│                    FanLink 비즈니스 모델                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│      [인플루언서]                         [팬]                  │
│          │                                  │                   │
│          │  콘텐츠, 혜택                    │  구독료, 참여     │
│          │  ─────────→                      │  ─────────→       │
│          ▼            │                     ▼            │      │
│    ┌─────────────────────────────────────────────────────┐     │
│    │                                                      │     │
│    │                   FanLink 플랫폼                     │     │
│    │                                                      │     │
│    │   ┌─────────────────────────────────────────────┐   │     │
│    │   │                 가치 흐름                    │   │     │
│    │   │                                              │   │     │
│    │   │   인플루언서 → 독점 콘텐츠, 혜택 → 팬       │   │     │
│    │   │   팬 → 구독료, 참여, 데이터 → 인플루언서    │   │     │
│    │   │                                              │   │     │
│    │   └─────────────────────────────────────────────┘   │     │
│    │                                                      │     │
│    │   ┌─────────────────────────────────────────────┐   │     │
│    │   │                 수익 흐름                    │   │     │
│    │   │                                              │   │     │
│    │   │   구독 수수료 10% ← 멤버십 결제             │   │     │
│    │   │   거래 수수료 5-10% ← 굿즈/이벤트           │   │     │
│    │   │   프리미엄 요금 ← 고급 기능                  │   │     │
│    │   │                                              │   │     │
│    │   └─────────────────────────────────────────────┘   │     │
│    │                                                      │     │
│    └─────────────────────────────────────────────────────┘     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 3. 단위 경제학 (Unit Economics)

### 3.1 팬 LTV 계산

```
┌─────────────────────────────────────────────────────────────────┐
│                     팬 LTV 계산                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   가정:                                                         │
│   • 평균 구독료: ₩7,000/월                                     │
│   • 플랫폼 수수료: 10%                                         │
│   • 평균 구독 기간: 8개월                                      │
│   • 굿즈 구매: ₩20,000/년 (거래 수수료 10%)                    │
│                                                                 │
│   계산:                                                         │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                                                          │  │
│   │   구독 수익 = ₩7,000 × 10% × 8개월 = ₩5,600             │  │
│   │   굿즈 수익 = ₩20,000 × 10% × (8/12) = ₩1,333           │  │
│   │                                                          │  │
│   │   팬 LTV = ₩5,600 + ₩1,333 = ₩6,933                     │  │
│   │                                                          │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   세그먼트별 LTV:                                               │
│   • Casual (무료): ₩0                                          │
│   • Light: ₩3,500 (4개월 × ₩3,900 × 10% + 굿즈)               │
│   • Core: ₩8,500 (10개월 × ₩7,900 × 10% + 굿즈)               │
│   • Super: ₩25,000 (18개월 × ₩14,900 × 10% + 굿즈 다수)       │
│                                                                 │
│   가중 평균 LTV: 약 ₩7,000                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.3 LTV/CAC 분석

```
┌─────────────────────────────────────────────────────────────────┐
│                    LTV/CAC 분석                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   팬 LTV/CAC                                                    │
│   ─────────────────────────────────────────────────────         │
│   LTV: ₩7,000                                                  │
│   CAC: ₩650                                                    │
│   LTV/CAC: 10.8x ✓ (목표: 3x 이상)                             │
│                                                                 │
│   크리에이터 LTV/CAC                                            │
│   ─────────────────────────────────────────────────────         │
│   가정: 크리에이터당 평균 100명 유료 팬, 12개월 활동            │
│                                                                 │
│   크리에이터 LTV = 100팬 × ₩7,000(팬LTV) = ₩700,000            │
│   크리에이터 CAC = ₩50,000                                     │
│   LTV/CAC: 14x ✓                                               │
│                                                                 │
│   ─────────────────────────────────────────────────────         │
│                                                                 │
│   Payback Period (팬)                                          │
│   CAC ₩650 ÷ 월 수익 ₩700 = 0.9개월 ✓ (목표: 12개월 이내)     │
│                                                                 │
│   Payback Period (크리에이터)                                   │
│   CAC ₩50,000 ÷ 월 수익 ₩58,000 = 0.9개월 ✓                   │
│   (100팬 × ₩7,000 × 10% × 1/12 = ₩58,000/월)                  │
│                                                                 │

```

## 6. 락인(Lock-in) 메커니즘

### 6.1 팬 락인

| 메커니즘 | 설명 | 이탈 비용 |
|----------|------|----------|
| **팬 레벨** | Bronze → Legend 레벨 축적 | 다시 Bronze부터 시작 |
| **배지 컬렉션** | 획득한 배지들 | 이전 불가, 소멸 |
| **포인트 잔액** | 적립한 포인트 | 환불/이전 불가 |
| **팬 히스토리** | 응원 기록, 통계 | 데이터 소멸 |
| **커뮤니티 관계** | 다른 팬들과 형성한 관계 | 관계 단절 |
| **팬 순위** | 크리에이터별 팬 랭킹 | 순위 소멸 |

### 6.2 크리에이터 락인

| 메커니즘 | 설명 | 이탈 비용 |
|----------|------|----------|
| **팬 데이터** | 축적된 팬 분석 데이터 | 데이터 이전 어려움 |
| **구독자 기반** | FanLink에서만 유효한 구독 | 구독자 재획득 필요 |
| **콘텐츠 아카이브** | 업로드한 독점 콘텐츠 | 이전 작업 필요 |
| **팬 레벨 시스템** | 팬들이 축적한 레벨/배지 | 팬 불만 |
| **브랜딩** | FanLink URL, 위젯 | 재설정 필요 |

### 6.3 네트워크 효과

```
┌─────────────────────────────────────────────────────────────────┐
│                     네트워크 효과 구조                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Same-Side Network Effect (동일측 네트워크 효과)               │
│   ─────────────────────────────────────────────────────         │
│                                                                 │
│   팬 ↔ 팬:                                                     │
│   • 커뮤니티에서 다른 팬과 교류                                │
│   • 더 많은 팬 → 더 활발한 커뮤니티 → 더 많은 팬              │
│                                                                 │
│   Cross-Side Network Effect (교차측 네트워크 효과)              │
│   ─────────────────────────────────────────────────────         │
│                                                                 │
│   크리에이터 ↔ 팬:                                              │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                                                          │  │
│   │   더 많은 크리에이터                                     │  │
│   │         │                                                │  │
│   │         ▼                                                │  │
│   │   더 다양한 콘텐츠 ──→ 더 많은 팬 유입                   │  │
│   │                              │                           │  │
│   │                              ▼                           │  │
│   │   더 매력적인 플랫폼 ←── 더 큰 잠재 팬덤                 │  │
│   │         │                                                │  │
│   │         ▼                                                │  │
│   │   더 많은 크리에이터 유입 (반복)                         │  │
│   │                                                          │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   크로스 프로모션:                                              │
│   • 한 크리에이터의 팬 → 다른 크리에이터 발견                  │
│   • 플랫폼 내 탐색 → 다중 구독 증가                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
