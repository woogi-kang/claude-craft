# FanLink - MVP 정의서

## 1. MVP 개요

### 1.1 MVP 목적

> **"최소 기능으로 핵심 가설을 검증하고, 제품-시장 적합성(PMF)을 확인한다"**

### 1.2 MVP 범위 요약

```
┌─────────────────────────────────────────────────────────────────┐
│                      MVP 범위 정의                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   포함 (In Scope)                    제외 (Out of Scope)        │
│   ┌─────────────────────┐            ┌─────────────────────┐   │
│   │ ✓ 멤버십 구독       │            │ ✗ 실물 굿즈샵       │   │
│   │ ✓ 팬 레벨 시스템    │            │ ✗ 라이브 스트리밍   │   │
│   │ ✓ 배지 시스템       │            │ ✗ 복잡한 커뮤니티   │   │
│   │ ✓ 독점 콘텐츠       │            │ ✗ AI 추천          │   │
│   │ ✓ 기본 대시보드     │            │ ✗ API 연동         │   │
│   │ ✓ 결제 시스템       │            │ ✗ 네이티브 앱      │   │
│   │ ✓ 간단한 커뮤니티   │            │ ✗ 다국어 지원      │   │
│   └─────────────────────┘            └─────────────────────┘   │
│                                                                 │
│   MVP 기간: 12주 (3개월)                                        │
│   MVP 예산: ₩150M                                              │
│   목표 크리에이터: 50명                                         │
│   목표 팬: 5,000명                                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 검증할 핵심 가설

### 2.1 가설 목록

| # | 가설 | 유형 | 우선순위 | 검증 방법 |
|---|------|------|---------|----------|
| **H1** | 팬은 레벨/배지 시스템에 동기부여된다 | 가치 | P0 | 참여율 측정 |
| **H2** | 크리에이터는 팬 데이터에 가치를 느낀다 | 가치 | P0 | 인터뷰, NPS |
| **H3** | 10% 수수료는 수용 가능하다 | 가격 | P0 | 전환율 측정 |
| **H4** | 팬은 유료 구독할 의향이 있다 | 수익 | P0 | 유료 전환율 |
| **H5** | 크리에이터는 자발적으로 홍보한다 | 성장 | P1 | 행동 관찰 |

## 3. MVP 기능 명세 (MoSCoW)

### 3.1 Must Have (필수)

**크리에이터 기능**

| 기능 | 설명 | 검증 가설 |
|------|------|----------|
| 회원가입/로그인 | 소셜 로그인 (구글, 카카오) | - |
| 프로필 설정 | 기본 정보, 소개, 프로필 이미지 | - |
| 멤버십 티어 설정 | 최대 3개 티어, 가격/혜택 설정 | H3 |
| 독점 콘텐츠 업로드 | 텍스트, 이미지, 영상 링크 | H2 |
| 기본 대시보드 | 팬 수, 수익, 최근 가입자 | H2 |
| 수익 확인 | 월별 수익, 출금 신청 | H3 |

**팬 기능**

| 기능 | 설명 | 검증 가설 |
|------|------|----------|
| 회원가입/로그인 | 소셜 로그인 (구글, 카카오) | - |
| 크리에이터 탐색 | 검색, 카테고리별 목록 | - |
| 멤버십 구독 | 티어 선택, 결제 | H4 |
| 독점 콘텐츠 열람 | 구독 티어에 따른 접근 | H4 |
| 팬 레벨 표시 | 현재 레벨, 포인트, 다음 레벨까지 | H1 |
| 기본 배지 | 구독 기간 배지 (1개월, 3개월, 6개월) | H1 |

**공통 기능**

| 기능 | 설명 |
|------|------|
| 결제 시스템 | 카드, 카카오페이 |
| 알림 | 이메일, 푸시 알림 기본 |
| 반응형 웹 | 모바일/데스크톱 대응 |

### 3.2 Should Have (권장)

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 팬 메시지 | 크리에이터 → 전체 팬 메시지 | S1 |
| 활동 포인트 | 콘텐츠 시청, 댓글로 포인트 적립 | S2 |
| 추가 배지 | 활동 기반 배지 (댓글왕 등) | S3 |
| 팬 프로필 | 배지, 레벨, 응원 히스토리 | S4 |
| 간단한 댓글 | 콘텐츠에 댓글 기능 | S5 |

### 3.3 Could Have (선택)

| 기능 | 설명 |
|------|------|
| 친구 초대 | 초대 링크, 보너스 포인트 |
| 팬 순위 | 크리에이터별 팬 랭킹 |
| 소셜 공유 | 레벨업/배지 SNS 공유 |
| 기본 통계 | 크리에이터용 간단한 분석 |

### 3.4 Won't Have (MVP 제외)

| 기능 | 제외 이유 | 향후 계획 |
|------|----------|----------|
| 실물 굿즈샵 | 물류 복잡도 | v1.5 |
| 라이브 스트리밍 | 기술 복잡도 | v2.0 |
| 네이티브 앱 | 개발 리소스 | v1.5 |
| AI 추천 | 데이터 부족 | v2.0 |
| API 연동 | 우선순위 낮음 | v2.0 |
| 다국어 | 한국 집중 | v2.0 |
