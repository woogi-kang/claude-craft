# FanLink - 개발 공수 산정

## 1. 산정 개요

### 1.1 산정 방법

- **방법**: T-Shirt Sizing + Story Points
- **기준**: 시니어 개발자 1인 기준
- **버퍼**: 20% 추가 (리스크 대비)

### 1.2 T-Shirt Size 정의

| Size | Story Points | 예상 기간 | 설명 |
|------|-------------|----------|------|
| XS | 1-2 | 0.5일 | 간단한 UI 변경, 버그 수정 |
| S | 3-5 | 1-2일 | 단일 기능, 컴포넌트 |
| M | 8-13 | 3-5일 | 복합 기능, API 포함 |
| L | 21-34 | 1-2주 | 주요 기능, 여러 화면 |
| XL | 55+ | 2주+ | 복잡한 시스템, 통합 |
