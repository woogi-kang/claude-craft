## 1. 개요

### 1.1 로드맵 범위

| 구분 | 내용 |
|------|------|
| **총 기간** | 18개월 (MVP 12주 + 성장 15개월) |
| **주요 단계** | MVP → Product-Market Fit → Scale |
| **목표** | MAU 10만, 크리에이터 1,000명, MRR $100K |

```

└── MRR: $100K
```

## 2. Phase 1: MVP (Month 1-3)

### 2.1 타임라인

```
Month 1: Foundation
├── Week 1-2: 프로젝트 셋업 & 인프라
├── Week 3-4: 인증 시스템 & 사용자 관리
└── 마일스톤: 개발 환경 완성, 로그인 작동

Month 2: Core Features
├── Week 5-6: 크리에이터 페이지 & 멤버십
├── Week 7-8: 결제 시스템 통합
└── 마일스톤: 유료 멤버십 결제 가능

Month 3: Polish & Launch
├── Week 9-10: 팬 레벨 시스템 & 대시보드
├── Week 11: QA & 버그 수정
├── Week 12: 소프트 런칭
└── 마일스톤: MVP 출시 완료
```

### 2.2 MVP 마일스톤 상세

#### M1: 개발 환경 완성 (Week 2)

| 항목 | 완료 기준 | 담당 |
|------|----------|------|
| Next.js 프로젝트 셋업 | Vercel 배포 성공 | FE Lead |
| Supabase 연동 | DB, Auth 작동 확인 | BE Lead |
| CI/CD 파이프라인 | GitHub Actions 작동 | BE |
| 디자인 시스템 | shadcn/ui 컴포넌트 구성 | FE |

**Exit Criteria**: 개발 환경에서 사용자 가입/로그인 가능

#### M2: 유료 결제 가능 (Week 8)

| 항목 | 완료 기준 | 담당 |
|------|----------|------|
| 크리에이터 페이지 | 프로필, 소개, 멤버십 노출 | FE |
| 멤버십 티어 설정 | 최대 3개 티어 생성 | BE |
| TossPayments 연동 | 정기결제 테스트 성공 | BE |

#### M3: MVP 출시 (Week 12)

| 항목 | 완료 기준 | 담당 |
|------|----------|------|
| 팬 레벨 시스템 | 포인트 적립, 레벨업 작동 | BE |
| 크리에이터 대시보드 | 수익, 팬 통계 조회 | FE |
| 팬 마이페이지 | 구독 현황, 레벨 확인 | FE |

**Exit Criteria**: 50명 크리에이터, 2,000명 팬 온보딩

## 3. Phase 2: Product-Market Fit (Month 4-9)

### 3.1 타임라인

```
Month 4-5: Engagement
├── 콘텐츠 피드 기능
├── 팬 전용 게시물
├── 댓글 & 반응 시스템
└── 마일스톤: 콘텐츠 소비 플로우 완성

Month 6-7: Community
├── 팬 커뮤니티 (토론방)
├── 1:1 메시지 기능
├── 알림 시스템 고도화
└── 마일스톤: 커뮤니티 기능 출시

Month 8-9: Optimization
├── 크리에이터 분석 고도화
├── 추천 시스템 (팬→크리에이터)
├── A/B 테스트 인프라
└── 마일스톤: PMF 지표 달성
```

### 3.2 주요 마일스톤

#### M4: 콘텐츠 플로우 완성 (Month 5)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| **콘텐츠 피드** | 구독 크리에이터 게시물 타임라인 | Must |
| **멤버십 전용 콘텐츠** | 티어별 잠금 콘텐츠 | Must |
| **댓글 시스템** | 게시물 댓글, 대댓글 | Must |
| **반응 시스템** | 좋아요, 이모지 반응 | Should |
| **콘텐츠 알림** | 새 게시물 푸시 알림 | Should |

**Exit Criteria**: 팬 1인당 주간 콘텐츠 조회 10회 이상

#### M5: 커뮤니티 기능 (Month 7)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| **팬 커뮤니티** | 크리에이터별 토론 공간 | Must |
| **DM 기능** | 팬↔크리에이터 1:1 메시지 | Must |
| **배지 시스템** | 활동 기반 배지 부여 | Should |
| **팬 랭킹** | 크리에이터별 팬 리더보드 | Should |

**Exit Criteria**: 커뮤니티 DAU 3,000명

#### M6: PMF 달성 (Month 9)

| 지표 | 목표 | 현재(예상) |
|------|------|-----------|
| **40-40 Rule** | 40% "매우 실망" | 측정 필요 |
| **주간 리텐션** | 60% | - |
| **Organic 비율** | 30% | - |
| **NPS** | 50+ | - |

**PMF 체크리스트**:
- [ ] 크리에이터가 자발적으로 팬에게 홍보
- [ ] 팬이 다른 팬에게 추천
- [ ] 이탈 크리에이터 재가입 사례 발생
- [ ] 경쟁사에서 이전 사례 발생
