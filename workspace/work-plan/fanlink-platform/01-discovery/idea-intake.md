# FanLink - 아이디어 정의서

## 1. 핵심 아이디어

**한 줄 정의**: 인플루언서와 팬이 함께 성장하는 멤버십 & 리워드 플랫폼

**서비스명**: FanLink (팬링크)

**슬로건**: "팬이 곧 파트너다"

---

## 2. 문제 정의 (Problem Statement)

### 2.1 인플루언서가 겪는 문제

| 문제 | 현재 상황 | 고통 지수 |
|------|----------|----------|
| **플랫폼 종속성** | 유튜브/틱톡 알고리즘에 수입이 좌우됨 | 9/10 |
| **팬과의 직접 관계 부재** | 구독자 100만이어도 DM 불가, 이메일 없음 | 8/10 |
| **수익 다각화 어려움** | 광고 외 수익원 발굴이 어려움 | 8/10 |
| **진성 팬 식별 불가** | 누가 진짜 팬인지 데이터로 알 수 없음 | 7/10 |
| **커뮤니티 관리 부담** | 디스코드/카페 운영에 시간 소모 | 6/10 |

### 2.2 팬이 겪는 문제

| 문제 | 현재 상황 | 고통 지수 |
|------|----------|----------|
| **인정받지 못함** | 오래 응원해도 신규 팬과 차별 없음 | 8/10 |
| **크리에이터와 소통 불가** | 댓글은 묻히고, 소통 기회 제한적 | 8/10 |
| **흩어진 멤버십** | 유튜브 멤버십, 트위치 구독, 팬카페 등 분산 | 7/10 |
| **팬활동 기록 없음** | 5년 응원해도 증거/보상 없음 | 6/10 |
| **굿즈/이벤트 접근성** | 정보가 흩어져 있어 놓치기 쉬움 | 5/10 |

## 3. 솔루션 가설 (Solution Hypothesis)

### 3.1 핵심 가설

> **"팬의 응원 활동을 데이터화하고 보상하면, 팬은 더 적극적으로 참여하고 인플루언서는 진성 팬과의 관계를 강화할 수 있다"**

### 3.2 가설 분해

| # | 가설 | 검증 방법 | 성공 기준 |
|---|------|----------|----------|
| H1 | 팬은 자신의 응원 활동이 기록되고 인정받기를 원한다 | 설문조사, MVP 테스트 | 70% 이상 긍정 응답 |
| H2 | 인플루언서는 진성 팬 데이터가 있으면 기꺼이 특별 혜택을 제공한다 | 인터뷰 | 80% 긍정 응답 |
| H3 | 팬 레벨 시스템은 더 높은 참여도와 지출을 유도한다 | A/B 테스트 | 참여율 30% 증가 |
| H4 | 인플루언서는 플랫폼 독립적인 팬 커뮤니케이션 채널을 원한다 | 인터뷰 | 90% 긍정 응답 |

## 4. 핵심 기능 개요

### 4.1 인플루언서용 기능

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| **대시보드** | 팬 현황, 수익, 참여도 분석 | P0 |
| **멤버십 설정** | 티어별 혜택, 가격 설정 | P0 |
| **콘텐츠 관리** | 독점 콘텐츠 업로드, 스케줄링 | P0 |
| **팬 메시지** | 전체/그룹/개인 메시지 발송 | P1 |
| **이벤트 관리** | 팬미팅, 라이브, 굿즈 드랍 | P1 |
| **수익 정산** | 수익 확인, 출금 관리 | P0 |

### 4.2 팬용 기능

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| **팬 프로필** | 레벨, 배지, 응원 히스토리 | P0 |
| **멤버십 구독** | 좋아하는 크리에이터 구독 | P0 |
| **독점 콘텐츠** | 멤버십 전용 콘텐츠 열람 | P0 |

## 6. Why Now? (타이밍 분석)

### 6.1 시장 트렌드

| 트렌드 | 설명 | 영향 |
|--------|------|------|
| **크리에이터 이코노미 성장** | 글로벌 $250B → $480B (2027) | 시장 확대 |
| **팬덤 문화 확산** | K-POP, V-tuber, 게임 스트리머 | 수요 증가 |
| **D2C(Direct-to-Consumer) 트렌드** | 플랫폼 탈중개화 움직임 | 니즈 증가 |
| **멤버십 경제 성장** | 구독 서비스 일반화 | 결제 저항 감소 |
| **Web3/NFT 경험** | 디지털 소유권 개념 확산 | 인식 변화 |

### 6.2 기술 환경

- **결제 인프라**: 다양한 결제 수단 연동 용이
- **라이브 스트리밍**: 저비용 라이브 인프라 확보
- **모바일 우선**: 팬들의 모바일 사용 증가
- **AI 도구**: 콘텐츠 관리 자동화 가능

## 7. 성공 정의

### 7.1 MVP 성공 기준 (3개월)

| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| 인플루언서 온보딩 | 50명 | 가입 수 |
| 총 팬 가입자 | 5,000명 | MAU |
| 유료 구독 전환율 | 5% | 유료/전체 |
| 월 반복 수익(MRR) | ₩10M | 결제 데이터 |
| 팬 리텐션 (D30) | 40% | 코호트 분석 |

### 7.2 제품-시장 적합성 신호

- 인플루언서가 자발적으로 SNS에서 FanLink 홍보
- 팬이 다른 팬을 초대하는 행동 발생
- 월 재방문율 60% 이상
- 유료 구독자 이탈률 5% 미만/월

## 8. 리스크 및 가정

### 8.1 주요 리스크

| 리스크 | 영향 | 확률 | 대응 |
|--------|------|------|------|
| 인플루언서 확보 어려움 | 상 | 중 | 초기 인센티브, 타겟 마케팅 |
| 기존 플랫폼 경쟁 | 상 | 상 | 차별화 기능, 니치 집중 |
| 팬 결제 저항 | 중 | 중 | 무료 티어, 점진적 가치 증명 |
| 규제 이슈 | 중 | 하 | 법률 검토, 컴플라이언스 |

### 8.2 핵심 가정

1. 인플루언서는 팬 데이터를 원한다
2. 팬은 인정받고 싶어한다
3. 레벨/배지 시스템이 참여를 유도한다
4. 플랫폼 수수료 10-20%는 합리적이다
