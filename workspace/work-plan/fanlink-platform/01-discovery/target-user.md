# FanLink - 타겟 사용자 분석

## 1. 양면 시장 타겟 개요

FanLink는 **인플루언서**와 **팬** 양측을 동시에 타겟팅하는 양면 플랫폼입니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                      타겟 사용자 구조                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   공급측 (인플루언서)              수요측 (팬)                   │
│   ┌─────────────────┐            ┌─────────────────────┐       │
│   │                 │            │                     │       │
│   │  중소형 크리에이터│   ←→     │  충성도 높은 팬      │       │
│   │  (1만-100만)    │            │  (결제 의향 있음)   │       │
│   │                 │            │                     │       │
│   └─────────────────┘            └─────────────────────┘       │
│           │                              │                      │
│           │                              │                      │
│           ▼                              ▼                      │
│   ┌─────────────────┐            ┌─────────────────────┐       │
│   │ 1차 타겟:       │            │ 1차 타겟:           │       │
│   │ • 유튜버        │            │ • 20-35세           │       │
│   │ • 스트리머      │            │ • 월 3만원 이상     │       │
│   │ • 틱톡커        │            │   덕질 지출         │       │
│   │                 │            │ • 팬활동 적극적     │       │
│   │ 2차 타겟:       │            │                     │       │
│   │ • 버튜버        │            │ 2차 타겟:           │       │
│   │ • 인스타그래머  │            │ • 학생              │       │
│   │ • 팟캐스터      │            │ • 라이트 팬         │       │
│   └─────────────────┘            └─────────────────────┘       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 5. 사용자 세그먼트

### 5.1 인플루언서 세그먼트

| 세그먼트 | 구독자/팔로워 | 특성 | 타겟 우선순위 |
|----------|-------------|------|-------------|
| **Micro** | 1천-1만 | 열정적, 시간 많음, 수익 니즈 높음 | P2 |
| **Small** | 1만-10만 | 성장 중, 팬덤 형성기, 도구 필요 | P1 |
| **Medium** | 10만-100만 | 전업 가능, 데이터 니즈 높음 | **P0** |
| **Large** | 100만+ | 이미 팀 운영, 협상력 필요 | P3 |

**1차 타겟: Medium (10만-100만)**
- 이유: 팬덤 존재, 수익화 니즈 높음, 의사결정 빠름

### 5.2 팬 세그먼트

| **Super** | 10만원+ | 올인, 모든 활동 참여 | P1 |

**1차 타겟: Core (3-10만원/월)**
- 이유: 결제 의향 높음, 참여도 높음, 바이럴 가능성
