---

## 2. 제품 비전

### 2.1 미션

> **"인플루언서와 팬이 함께 성장하는 멤버십 플랫폼"**

### 2.2 비전

3년 내 한국 최대의 크리에이터-팬 직접 연결 플랫폼이 되어, 크리에이터 5,000명과 팬 100만 명이 사용하는 서비스로 성장한다.

### 2.3 핵심 가치

| 가치 | 인플루언서 | 팬 |
|------|-----------|-----|
| **연결** | 팬과 직접 관계 | 크리에이터와 가까워짐 |
| **인정** | 진성 팬 식별 | 응원이 기록되고 인정됨 |
| **성장** | 안정 수익, 팬덤 확장 | 레벨업, 특별 혜택 |
| **재미** | 팬덤 운영의 즐거움 | 게임처럼 즐기는 팬활동 |

---

## 3. 목표 및 성공 지표

### 3.1 MVP 목표 (12주)

| 목표 | 지표 | 목표치 |
|------|------|--------|
| 크리에이터 확보 | 온보딩 수 | 50명 |
| 팬 확보 | MAU | 5,000명 |
| 유료 전환 | 유료 전환율 | 5% |
| 수익 | MRR | ₩10M |
| 리텐션 | D30 | 30% |

### 3.2 OKR

**Objective**: MVP를 통해 제품-시장 적합성(PMF)을 검증한다

| Key Result | 목표 | 측정 방법 |
|-----------|------|----------|
| KR1 | 유료 전환율 5% 달성 | 유료 팬 / 전체 팬 |
| KR2 | D30 리텐션 30% 달성 | 코호트 분석 |
| KR3 | 크리에이터 NPS 30+ 달성 | NPS 설문 |
| KR4 | 크리에이터 50% 자발적 홍보 | 홍보 행동 추적 |

## 10. 출시 기준

### 10.1 MVP 출시 체크리스트

**기능 완성도**
- [ ] 소셜 로그인 동작
- [ ] 크리에이터 온보딩 완료 가능
- [ ] 멤버십 설정 및 콘텐츠 업로드 가능
- [ ] 팬 구독 결제 동작
- [ ] 팬 레벨/배지 표시
- [ ] 크리에이터 대시보드 동작

**품질**
- [ ] 주요 플로우 버그 없음
- [ ] 모바일/데스크톱 반응형 동작
- [ ] 페이지 로딩 3초 이내

**운영**
- [ ] 결제 테스트 완료
- [ ] 에러 모니터링 설정
- [ ] CS 채널 준비

### 10.2 출시 후 모니터링

| 지표 | 목표 | 알림 기준 |
|------|------|----------|
| 에러율 | <1% | >5% |
| API 응답시간 | <500ms | >2s |
| 결제 성공률 | >95% | <90% |
| DAU | 증가 추세 | 3일 연속 감소 |
