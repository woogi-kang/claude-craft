# Claude Code 팀별 활용법 & 에이전트 설계 가이드

## 개요
- **발표 제목:** Claude Code 실전 활용: 팀별 AI 에이전트 설계 가이드
- **부제:** 개발팀, 영업팀, 회사 전체를 위한 맞춤형 AI 자동화 전략
- **청중:** 기술/비기술 임직원 (팀 리더, 매니저, 실무자)
- **프레임워크:** SCQA + 실전 사례 중심
- **총 슬라이드:** 45장
- **예상 발표 시간:** 45-50분

## 청중 분석
- **유형:** Claude Code 기본 개념을 아는 임직원
- **목표:** 팀별 구체적 활용 방안 및 에이전트 설계 방법 습득
- **관심사:** 실무에 바로 적용 가능한 사례와 설계 템플릿
- **선호 구조:** 팀별 사례 → 에이전트 설계 → 실습 가이드

## 핵심 메시지 (5개)
1. **팀 맞춤형 AI** - 개발팀, 영업팀, 경영지원팀 각각 다른 활용법
2. **에이전트 설계** - Skills + MCP로 반복 업무 자동화
3. **ROI 중심** - 연간 $850,000 절감 사례 (Anthropic 데이터)
4. **비개발자도 가능** - 자연어 프롬프트로 복잡한 작업 수행
5. **점진적 도입** - 작은 성공부터 시작하여 확장

**헤드라인:** 개발팀의 반복 업무들
**콘텐츠:**

- 코드베이스 문서화 미비
- **결과:** 개발 속도 저하, 기술 부채 누적

### Slide 9: 활용 사례 1 - PR 리뷰 자동화
**헤드라인:** 30% 빠른 PR 처리
**콘텐츠:**
```bash
# PR 리뷰 요청
> "이 PR 리뷰해줘. 보안 이슈와 성능 문제 중심으로"

Claude Code:
1. 변경된 파일 분석
2. 코드 스타일 검사
3. 보안 취약점 탐지
4. 성능 이슈 식별
5. 우선순위별 피드백 제공
```

### Slide 10: 활용 사례 2 - 신입 온보딩
**헤드라인:** 몇 주 → 1-2일로 단축
**콘텐츠:**
```bash

> "이 프로젝트 아키텍처 설명해줘"
> "인증 로직은 어디서 처리해?"
> "테스트 실행 방법 알려줘"

Claude Code:
- 전체 코드베이스 분석
- 아키텍처 다이어그램 생성
- 주요 패턴 설명
- 시니어 개발자 시간 절약
```

### Slide 11: 활용 사례 3 - CI/CD 연동

**실제 사례:** Anthropic 공식 레포에서 사용 중

### Slide 39: 반복 업무 식별 매트릭스
**헤드라인:** 어떤 업무를 자동화할까?
**콘텐츠:**
```
          ↑ 반복 빈도 높음
          │
    ┌─────┼─────┐
    │  B  │  A  │  ← 자동화 우선순위
    │     │ ⭐  │
    ├─────┼─────┤
    │  D  │  C  │
    │     │     │
    └─────┴─────┘
          │
          → 자동화 가능성 높음

A: 즉시 자동화 (PR 리뷰, 경쟁사 뉴스)
B: 반자동화 (제안서 초안, 보고서)
C: 도구 지원 (코드 리팩토링)
D: 유지 (창의적 업무)
```
