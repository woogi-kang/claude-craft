# Couple App Customer Personas

## Persona 1: 대학생 커플 "수진 & 민호"

> "우리 사진 진짜 많은데... 정리할 엄두가 안 나ㅠㅠ"

### 기본 정보

| 항목 | 수진 | 민호 |
|------|------|------|
| 이름 | 김수진 | 이민호 |
| 나이 | 23세 | 24세 |
| 직업 | 대학생 (마케팅 전공) | 대학생 (컴공) |
| 위치 | 서울 신촌 |  |
| 연애 기간 | 1년 6개월 |  |
| 소득 | 아르바이트 + 용돈 |  |

### 하루 일과

```
08:00 - 기상, 인스타 스토리 체크
09:00 - 수업 (틈틈이 카톡)
12:00 - 점심 (학식에서 데이트)
15:00 - 수업/과제
18:00 - 카페 데이트 (사진 10장)
21:00 - 집 가서 TikTok 스크롤
23:00 - 커플통화하면서 취침
```

---

### 프로필

#### 성격 & 가치관

- **성격**: 활발함, SNS 좋아함, 트렌드에 민감
- **가치관**: 추억 중시, "인생은 기록이다", YOLO
- **관심사**: 카페 투어, 사진 찍기, 커플템, 여행

#### 디지털 행동

- **주 사용 앱**: Instagram, TikTok, 카카오톡, 비트윈
- **콘텐츠 소비**: 숏폼 하루 2시간+, 커플 브이로그
- **앱 다운로드**: 친구 추천, TikTok에서 발견

---

### Goals (목표)

#### 핵심 목표

**"우리 추억을 예쁘게 정리하고, 친구들에게 자랑하고 싶다"**

#### 부가 목표

1. 100일, 1주년 같은 기념일 절대 까먹지 않기
2. 데이트 코스 기록해두기 (다음에 또 가려고)
3. 커플 앨범 예쁘게 만들어서 나중에 보기

---

### Pain Points (문제점)

#### 핵심 문제

> "사진이 폰에 8,000장 있는데... 커플 사진만 모으려니까 포기"

갤러리에서 커플 사진 찾아서 비트윈에 올리는 게 너무 귀찮음. 데이트할 때마다 사진 20장씩 찍는데, 정리는 안 하고 쌓이기만 함.

#### 부가 문제

1. 비트윈 UI가 너무 옛날 느낌 ("2015년 느낌")
2. 기념일 캘린더는 있는데 직접 다 입력해야 함
3. "우리 언제 그 카페 갔었지?" 매번 까먹음

#### 좌절감

- 친구 커플은 앨범 잘 정리해서 부러움
- 정리하려고 마음먹어도 3분 만에 포기
- 남자친구한테 "너가 정리해" 하면 싸움

---

### 행동 패턴

#### 정보 습득 채널

| 채널 | 사용 빈도 | 목적 |
|------|----------|------|
| TikTok | 매일 2시간+ | 트렌드, 재미 |
| Instagram | 매일 1시간+ | 친구 일상, 핫플 |
| 에브리타임 | 매일 30분 | 학교 정보 |
| YouTube | 주 3-4회 | 커플 브이로그, 튜토리얼 |

#### 소셜 미디어

- **주 사용**: Instagram (스토리 데일리), TikTok (릴스 저장)
- **부 사용**: 유튜브 (브이로그), Threads (가끔)
- **사용 패턴**: 자기 전 1시간 스크롤, 점심시간 틈틈이

#### 구매 행동

- **구매 성향**: 무료 앱 선호, 예쁘면 깔아봄
- **결정 요인**: 친구 추천 > TikTok 바이럴 > 앱스토어 순위
- **가격 민감도**: 매우 높음 (학생), 무료 필수

---

### 공감 지도

#### Thinks & Feels (생각 & 감정)

- "다른 커플들은 어떻게 정리하는 거지?"
- "비트윈 너무 오래 써서 질려"
- "새로운 앱 써보고 싶은데 뭐가 좋을까"
- "AI가 알아서 해주면 진짜 편하겠다"

#### Sees (보는 것)

- 친구들 인스타에 예쁜 커플 사진
- TikTok에 커플 챌린지, 밈
- 비트윈 광고 (근데 안 끌림)

#### Hears (듣는 것)

- "야 너네 아직 비트윈 써? 올드해~"
- "요즘 이런 앱 나왔대"
- "너네 100일 때 뭐 해?"

#### Says & Does (말 & 행동)

- 데이트 때마다 사진 왕창 찍음
- 비트윈에 가끔 올리다가 귀찮아서 포기
- "나중에 정리해야지" (안 함)

#### Pain (고통)

- 사진 정리 스트레스
- 기념일 까먹어서 싸움
- 데이트 코스 까먹음

#### Gain (이득)

- 예쁜 커플 앨범 갖고 싶음
- 추억 자동 정리
- 친구들에게 자랑

---

### 대표 인용문

> "사진 정리? 그게 되면 내가 벌써 했지ㅋㅋ"

> "비트윈 5년 썼는데... 이제 좀 새로운 거 써보고 싶음"

> "AI가 알아서 해준다고? 그거 진짜면 바로 깔아볼 듯"

---

### 제품에 대한 반론 (Objections)

1. **"개인정보 괜찮아? 사진 다 올리는 거잖아"**
   → 대응: 기기 내 AI 처리, 클라우드 선택적, 보안 강조

2. **"비트윈에 추억 다 있는데 옮기기 귀찮아"**
   → 대응: 비트윈 데이터 가져오기 기능, "새 출발" 포지셔닝

3. **"무료 맞아? 나중에 돈 내라 그러는 거 아냐?"**
   → 대응: 핵심 기능 무료 약속, 프리미엄은 고급 기능만

---

### 마케팅 시사점

#### 이 페르소나에게 효과적인 것

- **메시지**: "사진 정리? AI한테 시키세요", "비트윈 졸업할 때"
- **채널**: TikTok, Instagram Reels, 에브리타임
- **톤**: 친근, 밈 감성, 공감 ("나만 이래?" 느낌)
- **CTA**: "무료로 시작하기", "사전예약하고 혜택 받기"

#### 피해야 할 것

- 기업체 말투 ("혁신적인 솔루션")
- 긴 설명 (3초 안에 끝내야 함)
- 유료 강조 (학생이라 민감)

---

### 고객 여정 스냅샷

```
인지 → 고려 → 결정 → 구매 → 충성
  |      |      |      |      |
  |      |      |      |      +- 친구에게 추천, 챌린지 참여
  |      |      |      +- 사전예약 완료, 앱 설치
  |      |      +- 랜딩페이지 방문, 기능 확인
  |      +- "이거 괜찮네" 영상 저장
  +- TikTok에서 발견
```

## Persona 2: 사회초년생 커플 "지현 & 준혁"

> "둘 다 바빠서 기념일도 까먹고... 자동으로 해줬으면"

### 기본 정보

| 항목 | 지현 | 준혁 |
|------|------|------|
| 이름 | 박지현 | 최준혁 |
| 나이 | 27세 | 29세 |
| 직업 | 대기업 마케터 (2년차) | 스타트업 개발자 (3년차) |
| 위치 | 서울 강남/판교 |  |
| 연애 기간 | 3년 (결혼 고려 중) |  |
| 소득 | 각 4,000-5,000만원 |  |

### 하루 일과

```
07:30 - 기상, 출근 준비
08:30 - 출근 (지하철에서 뉴스/SNS)
09:00 - 업무 시작
19:00 - 퇴근 (야근 많음)
20:00 - 퇴근 후 데이트 or 각자 집
22:00 - 영통하면서 하루 마무리
24:00 - 취침
주말 - 데이트 몰아서 (맛집, 카페, 전시)
```

---

### 프로필

#### 성격 & 가치관

- **성격**: 효율 중시, 깔끔함, 실용적
- **가치관**: 시간 = 돈, 자동화 최고, 워라밸
- **관심사**: 맛집, 여행, 재테크, 결혼 준비

#### 디지털 행동

- **주 사용 앱**: 카카오톡, 인스타그램, 토스, 캘린더
- **콘텐츠 소비**: 인스타 릴스, 유튜브 브이로그
- **앱 다운로드**: 필요하면 결제도 함, 효율적인 거 찾음

---

### Goals (목표)

#### 핵심 목표

**"바쁜 일상 속에서도 우리 추억을 잘 관리하고 싶다"**

#### 부가 목표

1. 기념일 자동 알림 (안 까먹게)
2. 데이트 기록 자동화 (따로 정리할 시간 없음)
3. 결혼 전 연애 추억 예쁘게 정리

---

### Pain Points (문제점)

#### 핵심 문제

> "둘 다 야근이 많아서 기념일도 까먹고, 사진 정리할 시간이 없어"

주중엔 각자 바쁘고, 주말에 데이트해도 사진 정리까지 할 에너지가 없음. 3년 연애했는데 제대로 정리된 앨범이 없음.

#### 부가 문제

1. 비트윈 계속 쓰는데 솔직히 귀찮음
2. 캘린더 따로, 앨범 따로, 채팅 따로... 통합이 안 됨
3. 100일에 100일인 줄 몰랐다가 싸움

#### 좌절감

- 시간 없어서 정리 못 함 → 죄책감
- 결혼 앨범 만들려는데 사진 찾기 어려움
- 남친/여친한테 "너가 해" 하면 둘 다 못 함

---

### 행동 패턴

#### 정보 습득 채널

| 채널 | 사용 빈도 | 목적 |
|------|----------|------|
| Instagram | 매일 30분 | 트렌드, 맛집 |
| YouTube | 주 3-4회 | 브이로그, 정보 |
| 블라인드 | 매일 10분 | 회사 정보, 꿀팁 |
| 네이버 | 필요 시 | 검색, 리뷰 |

#### 구매 행동

- **구매 성향**: 효율적이면 결제 OK, 시간 > 돈
- **결정 요인**: 기능 > 디자인 > 가격
- **가격 민감도**: 낮음 (월 5,000원 정도는 OK)

---

### 공감 지도

#### Thinks & Feels (생각 & 감정)

- "시간 없는데 알아서 해주는 앱 없나?"
- "결혼 전에 추억 정리해놓고 싶다"
- "비트윈 계속 쓰긴 하는데... 뭔가 아쉬움"

#### Sees (보는 것)

- 인스타에 예쁜 커플 피드 (부러움)
- AI 앱들 광고 (궁금함)

#### Hears (듣는 것)

- "요즘 AI가 다 해줘"
- "너네 결혼 언제 해?" (압박)

#### Pain (고통)

- 시간 부족 스트레스
- 기념일 관리 실패
- 정리 안 된 사진 죄책감

#### Gain (이득)

- 자동화된 추억 관리
- 시간 절약
- 결혼 전 예쁜 앨범

### 대표 인용문

> "자동으로 해주면 좋겠어. 진짜 시간이 없거든"

> "결혼 전에 연애 앨범 만들어두고 싶은데..."

- **톤**: 실용적, 효율 강조, 시간 절약

- 복잡한 기능 설명

## Persona 3: 장기 연애 커플 "은영 & 태민" (Negative Persona 요소 포함)

> "비트윈 6년 썼는데... 새 앱 깔기 귀찮아"

### 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 한은영, 김태민 |
| 나이 | 30세, 32세 |
| 직업 | 직장인 (안정적) |
| 연애 기간 | 6년 |
| 특성 | 변화 싫어함, 익숙한 것 선호 |

### 왜 중요한가

이 페르소나는 **전환이 어려운 세그먼트**를 대표. 하지만:

1. 전환에 성공하면 → 강력한 팬 (추천 많이 함)
2. "비트윈 졸업" 메시지의 핵심 타겟
3. 입소문 확산의 핵심 (친구 많음)

### 핵심 인사이트

**전환 장벽**
- 6년 치 데이터가 비트윈에 있음
- 새로운 앱 배우기 귀찮음
- 둘 다 동의해야 해서 설득 어려움

**전환 트리거**
- 친구 커플이 쓰고 강추하면
- "비트윈보다 확실히 나은 점"이 보이면
- 마이그레이션이 쉬우면

### 마케팅 시사점

- **초기 타겟 X** (전환 비용 너무 높음)
- **후기 타겟**: 바이럴 충분히 된 후 공략
- **메시지**: "6년 추억도 안전하게 옮겨드려요"

*Persona Type: Negative (전환 어려운 세그먼트)*
