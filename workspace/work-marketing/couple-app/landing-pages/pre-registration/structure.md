# Couple App Pre-Registration Landing Page Structure

## Page Info

| 항목 | 내용 |
|------|------|
| 유형 | Lead Gen (Squeeze) |
| 목표 | 사전예약 1,000명 수집 |
| 타겟 | 20-30대 커플 (대학생 중심) |
| 주요 CTA | 평생 무료로 시작하기 |
| 도구 | Carrd (무료, 빠른 구축) |

---

## AI Era (2025) 적용 원칙

```
1. 모바일 퍼스트 (숏폼 트래픽 90%가 모바일)
2. 3초 내 가치 전달 (스크롤 유도)
3. 폼 필드 1개 (이메일만)
4. 숫자 카운터로 긴급성 (선착순 000명)
5. 데모 영상/GIF 삽입 (AI 기능 보여주기)
6. 공유 기능 내장 (바이럴 루프)
```

## Section Structure

### 1. Hero (Above the Fold)

| 요소 | 내용 |
|------|------|
| 헤드라인 | AI가 알아서 정리해주는 커플앱 |
| 서브헤드라인 | 사진 분류, 데이트 기록, 기념일까지. 수동 정리 끝. |
| CTA 버튼 | 평생 무료로 시작하기 |
| 폼 필드 | 이메일 1개 |
| 비주얼 | 앱 목업 + AI 자동 분류 데모 GIF |
| 긴급성 | 선착순 1,000명 중 [000]명 남음 |

**레이아웃**:
- 모바일: 수직 스택 (헤드라인 → 데모 → 폼 → CTA)
- 데스크탑: 좌측 텍스트 + 우측 목업

**Above the Fold 체크리스트**:
```
[x] 3초 내 가치 전달
[x] 단일 CTA
[x] 긴급성 요소
[x] 비주얼 증거 (데모)
```

### 5. Benefits Section (Feature → Benefit)

| Feature | Benefit | 아이콘 |
|---------|---------|--------|
| AI 사진 자동 분류 | 5시간 정리 → 5초로 끝 | Clock/Magic |
| 데이트 자동 기록 | 기억력 필요 없음, 앱이 기억해줌 | Location/Brain |
| 기념일 스마트 알림 | 까먹어서 싸울 일 없음 | Heart/Bell |
| 커플 공유 캘린더 | 서로 일정 한눈에 | Calendar |
| 미니멀 UI | 예쁘고 깔끔, 자랑하고 싶음 | Sparkle |
| 무료 핵심 기능 | 돈 안 들고 다 됨 | Gift |

**레이아웃**: 3x2 그리드 또는 수직 리스트 (모바일)

### 6. How It Works

| 단계 | 제목 | 설명 |
|------|------|------|
| 1 | 이메일 입력 | 30초면 끝. 사전예약 완료 |
| 2 | 런칭 알림 받기 | 앱 나오면 바로 알려드려요 |
| 3 | 평생 무료로 시작 | 선착순 1,000명만 프리미엄 무료 |

**스타일**:
- 숫자 원형 아이콘 (1 → 2 → 3)
- 화살표 연결
- 간단함 강조 ("30초면 끝")

```
    ┌───┐        ┌───┐        ┌───┐
    │ 1 │ ──→    │ 2 │ ──→    │ 3 │
    └───┘        └───┘        └───┘
   이메일       알림 받기     평생 무료
```

## Technical Implementation

### 도구 선택: Carrd

```yaml
why_carrd:
  - 무료 (Pro는 $19/년, 0원 예산이면 무료 버전)
  - 빠른 구축 (1-2시간)
  - 모바일 반응형 기본 제공
  - 폼 제출 → 이메일 수집 가능
  - 커스텀 도메인 연결 가능
  - 분석 코드 삽입 가능

alternatives:
  - Notion (무료, 제한적)
  - Typedream (무료 티어)
  - Softr (무료 티어)
```

### 필수 연동

```yaml
analytics:
  - Google Analytics 4 (무료)
  - Meta Pixel (무료)

forms:
  - Carrd 내장 폼 → Mailchimp/ConvertKit 연동
  - 또는 Tally (무료 폼 빌더)

counters:
  - 실시간 숫자 카운터 (JavaScript)
  - 또는 수동 업데이트
```
