## Part 3: Value Propositions (가치 제안)

### Version 1: AI 강조

> **AI가 알아서 해주는 커플앱**
>
> 사진 분류, 데이트 기록, 기념일 알림까지.
> 수동으로 할 필요 없어요.
>
> - AI가 커플 사진 자동 분류
> - 데이트 장소/날짜 자동 기록
> - 기념일 잊지 않게 알림
>
> 선착순 1,000명 평생 무료

### Version 2: 문제 해결 강조

> **사진 정리 지옥에서 탈출하세요**
>
> 1만 장 사진, 언제 정리하려고요?
> AI가 5초 만에 정리해드립니다.
>
> - 수동 정리 필요 없음
> - 데이트 기록도 자동
> - 기억력 필요 없음
>
> 사전예약하고 평생 무료로 시작하세요

### Version 3: 비트윈 대체 강조

> **비트윈 졸업, 2025 커플앱으로**
>
> 5년 전 앱 아직도 쓰세요?
> 2025년엔 AI가 해줘야죠.
>
> - 비트윈에 없는 AI 자동 분류
> - 더 예쁘고 현대적인 UI
> - 귀찮음 끝
>
> 새로운 시작, 사전예약하세요

## Part 7: Feature → Benefit

| Feature | Benefit |
|---------|---------|
| AI 사진 자동 분류 | 5시간 정리 → 5초로 끝 |
| 데이트 자동 기록 | 기억력 필요 없음, 앱이 기억해줌 |
| 기념일 알림 | 까먹어서 싸울 일 없음 |
| 커플 공유 캘린더 | 서로 일정 한눈에 |
| 미니멀 UI | 예쁘고 깔끔, 자랑하고 싶음 |
| 무료 핵심 기능 | 돈 안 들고 다 됨 |
