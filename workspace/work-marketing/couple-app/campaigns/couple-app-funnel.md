# Couple App Growth Funnel (Pre-Launch)

## Funnel Overview

```
           Acquisition (인지)
          +---------------+
          |    100,000    |  숏폼 조회자
          +-------+-------+
                  | 2%
                  v
           Activation (참여)
          +---------------+
          |     2,000     |  프로필 방문/팔로우
          +-------+-------+
                  | 20%
                  v
            Retention (관심)
          +---------------+
          |      400      |  2회 이상 방문
          +-------+-------+
                  | 50%
                  v
            Revenue (전환)
          +---------------+
          |      200      |  사전예약 완료
          +-------+-------+
                  | 50%
                  v
            Referral (추천)
          +---------------+
          |      100      |  친구 추천
          +-------+-------+
                  |
                  v
        바이럴 계수 0.5 (추가 50명)
```

**Note**: 이 퍼널은 "사전예약" 단계의 퍼널임. 런칭 후에는 앱 설치 → 활성화 → 유지 → 수익화 퍼널로 전환.

## 1. Acquisition (획득)

### 현재 상태

| 지표 | 현재 | 목표 | Gap |
|------|------|------|-----|
| 월간 노출 | 0 | 100만 (누적) | 100만 |
| 채널별 비중 | - | TikTok 60% | - |
| 획득 비용 | 0원 | 0원 | OK |

### 채널 분석

| 채널 | 예상 트래픽 | 비용 | CPA | 전환율 |
|------|-----------|------|-----|--------|
| TikTok | 60만 | 0원 | 0원 | 2% |
| Instagram Reels | 25만 | 0원 | 0원 | 1.5% |
| YouTube Shorts | 10만 | 0원 | 0원 | 1% |
| 에브리타임 | 5만 | 0원 | 0원 | 3% |

### 최적화 전략

1. **훅 다양화**
   - 현재: 아직 시작 전
   - 목표: 10가지 훅 유형 테스트
   - 액션: ChatGPT로 훅 30개 생성, A/B 테스트

2. **플랫폼 최적화**
   - TikTok: 15-30초, 트렌드 사운드
   - Reels: 감성적, 예쁜 비주얼
   - Shorts: 정보성, 검색 최적화

3. **업로드 일관성**
   - 주 5회 TikTok, 주 3회 Reels
   - 알고리즘 우대 받기 위한 꾸준함

## 2. Activation (활성화)

### "Aha Moment" 정의

> **"이거 진짜 자동으로 정리해준다고? 대박"**
>
> = 데모 영상 보고 "이건 써봐야겠다" 느낌 받는 순간

### 현재 상태

| 지표 | 현재 | 목표 | Gap |
|------|------|------|-----|
| 프로필 방문율 | - | 2% (조회→방문) | - |
| 팔로우 전환율 | - | 50% (방문→팔로우) | - |
| LP 클릭율 | - | 20% (팔로우→LP) | - |

### Activation Funnel (상세)

```
숏폼 조회      프로필 방문      팔로우        LP 방문       사전예약
   100%    →     2%      →     50%     →    20%     →    25%
    |             |             |            |            |
    |   -98%      |    -50%     |   -80%     |   -75%     |
    v             v             v            v            v
 스크롤 패스    관심 없음      "나중에"    이탈       완료!
```

### 병목 지점

| 단계 | 이탈률 | 원인 추정 | 우선순위 |
|------|--------|----------|---------|
| 조회→방문 | 98% | 훅이 약함, 타겟 안 맞음 | High |
| 팔로우→LP | 80% | CTA 불명확, 바이오 링크 | High |
| LP→전환 | 75% | LP 설득력 부족 | Medium |

### 최적화 전략

1. **훅 강화** (조회→방문)
   - 문제: 스크롤 멈추게 하는 훅 부족
   - 가설: 공감형 훅이 질문형보다 효과적
   - 액션: 훅 A/B 테스트 (10개씩 2주간)
   - 예상 개선: 방문율 2% → 3%

2. **CTA 명확화** (팔로우→LP)
   - 문제: 바이오 링크 클릭 유도 부족
   - 가설: 영상 내 CTA + 핀 댓글 조합
   - 액션: 모든 영상에 "링크 바이오" CTA
   - 예상 개선: LP 클릭 20% → 30%

3. **LP 개선** (LP→전환)
   - 문제: 설득력 부족
   - 가설: "평생 무료" 강조하면 전환 상승
   - 액션: CRO 체크리스트 적용
   - 예상 개선: 전환율 25% → 35%

## 3. Retention (유지/관심)

### 현재 상태 (사전예약 기준)

| 코호트 | 현재 | 벤치마크 | 상태 |
|--------|------|---------|------|
| 2회 방문 | - | 40-50% | - |
| 이메일 오픈 | - | 20-30% | - |
| 커뮤니티 참여 | - | 10-20% | - |

### Retention 전략

사전예약자가 "런칭까지 관심 유지"하도록:

```
사전예약 완료
     |
     v
+---------------+
|   이메일      | --> 주 1회 개발 소식
+---------------+
     |
     v
+---------------+
|   커뮤니티    | --> 오픈채팅/Discord
+---------------+
     |
     v
+---------------+
| Building in   | --> SNS 스토리
|   Public      |
+---------------+
     |
     v
런칭 시 앱 설치
```

### 최적화 전략

1. **이메일 드립**
   - 트리거: 사전예약 완료
   - 시퀀스: 5통 (아래 Email Sequence 참조)
   - 목표: 오픈율 30%, 런칭 시 알림

2. **커뮤니티 참여**
   - 채널: 카카오 오픈채팅 (사전예약자 전용)
   - 콘텐츠: 개발 과정 공유, 기능 투표
   - 목표: 활성 멤버 100명+

3. **Building in Public**
   - 채널: 인스타 스토리, TikTok
   - 콘텐츠: 개발 비하인드, 실패담, 질문
   - 목표: 진정성으로 팬덤 구축

## 4. Revenue (전환/수익)

### 현재 상태 (사전예약 기준)

| 지표 | 현재 | 목표 | Gap |
|------|------|------|-----|
| 사전예약 전환율 | - | 25% (LP 방문→예약) | - |
| 총 사전예약 | 0 | 1,000명 | 1,000 |
| CAC | 0원 | 0원 | OK |

### 전환 퍼널

```
LP 방문자      폼 시작       폼 완료      확인 이메일
   400     →    200     →    100     →    100
    |            |            |           |
    |  -50%      |   -50%     |   -0%     |
```

### 최적화 전략

1. **LP 히어로 강화**
   - 현재: (아직 없음)
   - 액션: 3초 내 가치 전달, 단일 CTA
   - 예상 효과: 폼 시작율 50% → 60%

2. **폼 간소화**
   - 필드: 이메일만 (1개)
   - 보너스: 이름/추천코드 (선택)
   - 예상 효과: 완료율 50% → 70%

3. **긴급성 추가**
   - "선착순 1,000명 평생 무료"
   - 카운터 표시 (남은 자리)
   - 예상 효과: 전환율 +10%

## 5. Referral (추천)

### 현재 상태

| 지표 | 현재 | 목표 | Gap |
|------|------|------|-----|
| 추천율 | - | 50% (예약자 중) | - |
| Viral Coefficient | - | 1.0+ | - |
| 앰배서더 | 0 | 50명 | 50 |

### 추천 채널

| 채널 | 예상 추천 수 | 전환율 |
|------|-------------|--------|
| 앰배서더 | 150명 (50명x3) | 80% |
| 일반 추천 | 100명 | 50% |
| 챌린지 UGC | 250명 | 30% |

### 바이럴 루프

```
사전예약 완료
     |
     v
+-----------------+
| 추천 코드 발급   |
+-----------------+
     |
     +--> 친구에게 공유 (카톡, DM)
     |
     +--> 챌린지 참여 (#태그)
     |
     +--> 앰배서더 활동
     |
     v
친구 사전예약
     |
     v
바이럴 계수 측정
```

### 최적화 전략

1. **앰배서더 프로그램**
   - 트리거: 사전예약 + 친구 3명 초대
   - 인센티브: 평생 프리미엄 무료
   - 메커니즘: 추천 코드 추적

2. **챌린지 바이럴**
   - 트리거: UGC 참여
   - 인센티브: 추첨 프리미엄 제공
   - 메커니즘: 해시태그 + 태그

3. **카카오톡 공유**
   - 간편 공유 버튼 (LP에 추가)
   - 공유 시 양측 혜택

## 6. Priority Matrix

### 영향도 vs 난이도

```
      High Impact
           |
     +-----+-----+
     | 1   |  2  |

     | 3   |  4  |
     +-----+-----+
           |
```
