# Couple App Marketing Context

## 1. 제품/서비스 개요

| 항목 | 내용 |
|------|------|
| 이름 | [미정] 2030 커플앱 |
| 한 줄 설명 | AI가 사진을 자동 분류하고, 데이트를 기록하는 2030 커플 전용 앱 |
| 카테고리 | Consumer App (Lifestyle/Dating) |
| 가격 | Freemium (기본 무료, 프리미엄 구독 예정) |
| 상태 | 개발 중 (사전예약 단계) |

### 핵심 기능/특징

1. **AI 앨범 자동 분류**: 함께 찍은 사진을 AI가 자동으로 장소/날짜/이벤트별로 정리
2. **데이트 자동 기록**: 함께 간 장소, 먹은 음식 등을 캘린더에 자동 저장
3. **캘린더 통합**: 커플 공유 캘린더, 기념일 알림, 데이트 플래닝

### 차별화 포인트 (vs 경쟁사)

| 우리 | 비트윈/러블/커플 |
|------|-----------------|
| AI가 알아서 정리 | 수동으로 앨범 정리 |
| 데이트 자동 기록 | 기록은 직접 해야 함 |
| 2025 최신 기술 스택 | 레거시 앱 느낌 |
| 미니멀/깔끔한 UX | 기능 많지만 복잡 |

## 2. 마케팅 목표

| 항목 | 내용 |
|------|------|
| 주요 목표 | 브랜드 인지도 확보 (사전예약) |
| 목표 수치 | 사전예약 1,000명 |
| 기간 | 런칭 전 3개월 |
| 예산 | 0원 (오가닉/바이럴 전략만) |

### 성공 정의

```yaml
primary_success:
  - 사전예약 1,000명 달성
  - 바이럴 계수 1.0 이상 (추천으로 가입)

secondary_success:
  - 브랜드 인지도 형성 (검색량 증가)
  - 소셜 미디어 팔로워 1,000명+
  - 커뮤니티 핵심 팬 100명 확보
```

## 3. 타겟 고객

### 고객 프로필

- **인구통계**: 20-30대 연인, 대학생~직장인 초반
- **설명**: 스마트폰 네이티브, SNS 활발, 데이트 기록/공유를 좋아함

### Pain Points

1. **사진 정리 지옥**: 수천 장 사진 중 커플 사진 찾기 귀찮음
2. **데이트 기억 증발**: "우리 언제 거기 갔었지?" 매번 까먹음
3. **기념일 잊음**: 100일, 1주년... 깜빡해서 싸움
4. **기존 앱 식상함**: 비트윈 오래 씀, 새로운 거 써보고 싶음

### 현재 대안

| 대안 | 한계 |
|------|------|
| 비트윈 | 오래됨, UI 올드함, 수동 정리 |
| 러블 | 기능 비슷, 차별화 부족 |
| 카카오톡 | 커플 전용 아님 |
| 갤러리 앱 | 커플용 아님, 공유 불편 |

## 4. 경쟁 환경

### 직접 경쟁사

| 경쟁사 | 포지셔닝 | 강점 | 약점 |
|--------|----------|------|------|
| 비트윈 | 커플 소통 앱 1위 | 시장 점유율, 브랜드 인지도 | 레거시, UI 노후화 |
| 러블 | 커플 일상 기록 | 귀여운 UI | 차별화 부족, 인지도 낮음 |
| 커플 | 커플 캘린더/D-day | 심플함 | 기능 제한적 |
| 데이트팝 | 데이트 코스 추천 | 장소 데이터 | 기록/앨범 기능 없음 |

### 우리의 차별점

> **"수동 기록 앱 시대는 끝났다. AI가 알아서 해주는 커플앱"**

1. AI 기술 적용 (2025년 답게)
2. 자동화로 귀찮음 제거
3. 미니멀하고 현대적인 UX

## 5. 브랜드 가이드

### 톤앤매너

- **친근함**: 20대가 쓰는 말투, 이모지 OK
- **위트 있음**: 재미있고 밈 감성
- **공감**: 커플의 소소한 일상에 공감
- **MZ 감성**: 트렌디, 힙함

### 사용할 단어

```
- "AI가 알아서"
- "자동으로"
- "귀찮음 끝"
- "우리 추억"
- "데이트 기록"
- "100일, 1주년"
```

### 피할 단어

```
- "혁신적인" (식상)
- "최고의" (과장)
- "커플 필수" (강요 느낌)
- 과도한 기업체 말투
```

### 기존 카피 예시 (톤 참고)

> "사진 정리? AI한테 시키세요"
> "데이트 까먹지 마세요. 앱이 기억해줄게요"
> "비트윈 5년 썼는데... 이제 졸업할 때"

## 6. 제약 조건

### 예산

- **마케팅 예산**: 0원 (완전 무예산)
- **가용 리소스**: 창업자 1-2명의 시간

### 가능한 것

```
- 소셜 미디어 오가닉 콘텐츠
- 숏폼 비디오 (TikTok, Reels, Shorts)
- 커뮤니티 활동 (에브리타임, 블라인드)
- 바이럴 챌린지
- 랜딩페이지 (Carrd, Tally 무료)
- 이메일 수집 (Mailchimp 무료 티어)
```

### 불가능한 것

```
- 유료 광고 (Google, Meta, 네이버)
- 인플루언서 유료 협찬
- 대규모 이벤트
```

## 7. AI 시대 강화 전략 (2025) 적용

### 콘텐츠 믹스

```
숏폼 비디오: 80%
이미지/텍스트: 20%
```

### 플랫폼 우선순위

```
1. TikTok (바이럴 잠재력 최고)
2. Instagram Reels
3. YouTube Shorts
4. 에브리타임/대학 커뮤니티
```

### 핵심 전략 방향

1. **숏폼 집중**: 80% 숏폼, 밈 감성
2. **참여형 챌린지**: 사용자가 콘텐츠 공동 창작
3. **커뮤니티 팬덤**: Building in Public, 앰배서더
4. **AI 도구 활용**: CapCut, Canva, ChatGPT로 대량 생산
