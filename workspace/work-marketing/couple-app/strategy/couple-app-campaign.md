# Couple App Pre-Launch Campaign Plan

## Campaign Overview

| 항목 | 내용 |
|------|------|
| 캠페인명 | "비트윈 졸업, 2025 커플앱 시대" 사전예약 캠페인 |
| 유형 | Awareness + Acquisition (인지도 + 사전예약) |
| 기간 | 12주 (런칭 전 3개월) |
| 총 예산 | 0원 |

### Campaign Summary

예산 0원 제약 하에서 숏폼 바이럴 + 참여형 챌린지 + 커뮤니티 팬덤 빌딩으로 사전예약 1,000명을 확보하는 캠페인. AI 시대(2025) 강화 전략을 전면 적용하여 콘텐츠 80%를 숏폼으로 제작하고, 사용자를 콘텐츠 공동 창작자로 만든다.

---

## 1. Objectives (목표)

### Primary Goal (SMART)

| SMART | 내용 |
|-------|------|
| Specific | 사전예약 페이지를 통해 잠재 고객 이메일 1,000개 수집 |
| Measurable | Tally/Carrd 전환 추적, UTM 파라미터 |
| Achievable | 바이럴 1개 터지면 300+, 꾸준한 업로드로 700 추가 가능 |
| Relevant | 런칭 시 초기 사용자 확보, 바이럴 계수 검증 |
| Time-bound | 12주 (런칭 전 3개월) |

**목표 문장**: "12주 내 숏폼 바이럴과 참여형 챌린지를 통해 사전예약 1,000명을 예산 0원으로 확보한다"

### Secondary Goals

1. 브랜드 인지도 형성 (TikTok 팔로워 1,000명+)
2. 앰배서더 50명 확보 (바이럴 확산 핵심)
3. 바이럴 계수 1.0+ 달성 (자가 성장 가능 증명)

### Success Metrics

| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| 사전예약 수 | 1,000명 | Tally 전환 |
| 숏폼 누적 조회수 | 100만 | TikTok Analytics |
| 챌린지 UGC 수 | 100개 | 해시태그 수집 |
| 앰배서더 수 | 50명 | 가입 추적 |
| 바이럴 계수 | 1.0+ | 추천 코드 추적 |

## 2. Target Audience (타겟)

### Primary Audience

**대학생 커플 (20-24세)**

- 사진 많고 정리 귀찮음
- TikTok 헤비 유저
- 새로운 앱 호기심
- 비트윈 쓰지만 질림

→ Persona: 수진 & 민호

### Secondary Audience

**사회초년생 커플 (25-30세)**

- 시간 없음, 자동화 원함
- Instagram 주로 사용
- 효율적인 도구 선호

### Exclusions (제외 대상)

- 커플앱 사용 의향 없는 사람
- 40대 이상 (타겟 벗어남)
- 비트윈 충성 유저 (전환 비용 높음)

## 4. Channels & Tactics (채널)

### Channel Details

#### TikTok (50%)

| 항목 | 내용 |
|------|------|
| 목적 | 바이럴, 인지도, 사전예약 유도 |
| 타겟팅 | 20-25세, 커플 콘텐츠 관심 |
| 포맷 | 15-30초 숏폼 |
| 예산 | 0원 (오가닉) |
| 예상 결과 | 조회수 60만, 팔로워 800명 |

**Tactics**:
1. 주 5회 업로드 (바이럴 확률 높이기)
2. 훅 A/B 테스트 (10개 중 1개 터짐)
3. 트렌드 사운드 적극 활용
4. 챌린지 시드 콘텐츠 직접 제작

#### Instagram Reels (25%)

| 항목 | 내용 |
|------|------|
| 목적 | 확산, 브랜드 이미지, 25-30세 타겟 |
| 타겟팅 | 25-30세 커플, 라이프스타일 관심 |
| 포맷 | 15-30초 (TikTok 리포스트 + 최적화) |
| 예산 | 0원 |
| 예상 결과 | 조회수 25만, 팔로워 500명 |

**Tactics**:
1. TikTok 콘텐츠 리포스트 (리사이즈)
2. 스토리로 일상/개발 과정 공유
3. 프로필 링크로 사전예약 유도

#### 에브리타임/커뮤니티 (15%)

| 항목 | 내용 |
|------|------|
| 목적 | 대학생 직접 타겟, 신뢰도 |
| 타겟팅 | 대학생 커플 |
| 포맷 | 텍스트 + 이미지, 후기 형식 |
| 예산 | 0원 |
| 예상 결과 | 노출 5만, 사전예약 150명 |

**Tactics**:
1. 학교별 게시판 포스팅
2. "솔직 후기" 형식 (광고 티 안 나게)
3. 댓글 소통으로 관계 구축

#### YouTube Shorts (10%)

| 항목 | 내용 |
|------|------|
| 목적 | SEO, 장기 노출 |
| 포맷 | 30-60초 |
| 예산 | 0원 |
| 예상 결과 | 조회수 10만 |

**Tactics**:
1. TikTok 콘텐츠 리포스트
2. 검색 키워드 최적화 (커플앱, 사진 정리)

## 5. 참여형 챌린지 캠페인 (AI 시대 핵심)

### 챌린지 1: #사진정리포기챌린지

**컨셉**: 커플 사진 몇 장인지 갤러리 스크린샷 인증

**바이럴 포인트**: 숫자가 클수록 재미, 공감, "나만 그런 거 아니었어" 느낌

**참여 방법**:
1. 갤러리에서 커플 사진 개수 확인
2. 스크린샷 찍어서 업로드
3. #사진정리포기챌린지 #[앱이름] 태그
4. 친구 커플 3명 태그

**참여 혜택**:
- 사전예약 우선권
- 추첨 10명 프리미엄 1년 무료

**예상 결과**: UGC 50개+, 노출 30만+

### 챌린지 2: #비트윈졸업챌린지

**컨셉**: 비트윈 탈출 선언, "새로운 시작"

**바이럴 포인트**: 졸업 = 성장 느낌, 비트윈 유저 타겟

**참여 방법**:
1. 비트윈 앱 스크린샷 (사용 기간)
2. "졸업합니다" 선언 영상
3. #비트윈졸업챌린지 #[앱이름] 태그

**참여 혜택**:
- 앰배서더 자격 부여
- 비트윈 데이터 마이그레이션 우선 지원

**예상 결과**: UGC 30개+, 노출 20만+

### 챌린지 3: #AI가알아서챌린지

**컨셉**: Before/After - 수동 vs AI 자동 정리

**바이럴 포인트**: 시각적 임팩트, "이게 돼?" 놀라움

**참여 방법**:
1. 지저분한 갤러리 Before
2. AI 정리된 앨범 After (데모 화면)
3. #AI가알아서챌린지 태그

**타이밍**: 앱 베타 버전 공개 후 (Phase 3)

## 8. Measurement & Optimization (측정)

### KPIs

| 단계 | KPI | 목표 | 측정 도구 |
|------|-----|------|----------|
| 인지 | 조회수 | 100만 | TikTok Analytics |
| 참여 | 좋아요/댓글 | 참여율 5%+ | TikTok Analytics |
| 전환 | 사전예약 | 1,000명 | Tally |
| 확산 | 바이럴 계수 | 1.0+ | 추천 코드 |

### Optimization Plan

**Weekly Review**:
- 영상별 성과 분석 (조회수, 완시율)
- 훅/콘텐츠 유형별 성과 비교
- 다음 주 콘텐츠 방향 조정

**Optimization Triggers**:
- If 조회수 < 1,000 연속 5개 → 훅 전면 교체
- If 전환율 < 1% → LP CTA 변경
- If 챌린지 참여 < 10개 → 혜택 강화

### A/B Tests Planned

| 테스트 | 변수 | 가설 |
|--------|------|------|
| 훅 유형 | 질문형 vs 공감형 | 공감형이 완시율 높을 것 |
| CTA 위치 | 영상 내 vs 바이오 | 영상 내 CTA가 전환 높을 것 |
| LP 헤드라인 | AI 강조 vs 귀찮음 해결 | 귀찮음 해결이 전환 높을 것 |
