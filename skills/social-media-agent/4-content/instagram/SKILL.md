# Instagram 콘텐츠 작성 가이드

## 플랫폼 특성

```yaml
instagram_profile:
  audience: "시각 중심, 라이프스타일, 감성 소비"
  content_lifespan: "24-48시간 (피드), 24시간 (스토리)"
  algorithm_priority:
    - 참여율 (좋아요, 댓글, 저장, 공유)
    - 체류 시간
    - 릴스 시청 완료율
    - 관계성 (DM, 멘션, 태그)
  best_times:
    weekday: "12:00-13:00, 19:00-21:00"
    weekend: "10:00-11:00, 20:00-22:00"
```

## 콘텐츠 유형별 가이드

### 1. 피드 포스트

#### 단일 이미지

```yaml
single_image:
  specs:
    ratio: "1:1 (1080x1080) 또는 4:5 (1080x1350)"
    format: "JPG, PNG"
    file_size: "30MB 이하"

  caption:
    max_length: 2200자
    optimal_length: 150-300자 (미리보기에 표시)
    structure:
      - hook: "첫 문장에 핵심 메시지 또는 질문"
      - body: "가치 제공 (팁, 스토리, 정보)"
      - cta: "행동 유도 (댓글, 저장, 링크)"
      - hashtags: "첫 댓글 또는 캡션 하단"
```

#### 캐러셀 (Carousel)

```yaml
carousel:
  specs:
    slides: "2-10장"
    ratio: "1:1 또는 4:5 (통일)"

  structure:
    slide_1: |
      훅/표지
      - 스크롤 유도하는 강력한 헤드라인
      - "스와이프해서 더 보기" 힌트
    slide_2_9: |
      본문
      - 한 슬라이드 = 한 가지 포인트
      - 시각적 일관성 유지
      - 넘버링 또는 프로그레스 바
    slide_10: |
      CTA/마무리
      - 핵심 요약
      - 저장/공유 유도
      - 댓글 질문

  best_practices:
    - 첫 슬라이드에서 호기심 유발
    - 각 슬라이드가 독립적으로도 가치 있게
    - 마지막에 저장 유도 (저장 = 알고리즘 부스트)
```

### 2. 릴스 (Reels)

```yaml
reels:
  specs:
    duration: "15-90초 (권장: 30초 이하)"
    ratio: "9:16 (1080x1920)"
    format: "MP4"

  structure:
    hook: |
      0-3초
      - 즉각적인 관심 유발
      - 텍스트 오버레이로 주제 명시
      - 얼굴 클로즈업 또는 강렬한 비주얼

    body: |
      3-25초
      - 빠른 편집 (2-3초마다 컷)
      - 자막 필수 (음소거 시청 80%)
      - 트렌딩 오디오 활용

    cta: |
      마지막 5초
      - 팔로우/저장 유도
      - 댓글 질문
      - 프로필 링크 안내

  script_template: |
    [HOOK - 3초]
    "이거 모르면 진짜 손해예요"

    [PROBLEM - 5초]
    "많은 분들이 ~를 어려워하시는데요"

    [SOLUTION - 15초]
    "해결 방법은 바로..."
    [단계별 설명]

    [CTA - 5초]
    "도움이 됐다면 저장해두세요!"
    "더 많은 팁은 프로필 링크에서"
```

### 3. 스토리 (Stories)

```yaml
stories:
  specs:
    duration: "15초/프레임"
    ratio: "9:16"
    expires: "24시간"

  types:
    announcement: "새 포스트/릴스 알림"
    behind_scenes: "비하인드 씬, 일상"
    engagement: "투표, 퀴즈, 질문"
    promotion: "제품/서비스 소개"
    ugc_share: "팔로워 콘텐츠 리포스트"

  interactive_elements:
    - poll: "A vs B 투표"
    - quiz: "정답 맞추기"
    - question: "질문 받기"
    - slider: "감정 표현"
    - countdown: "이벤트 카운트다운"
    - link: "외부 링크 (모든 계정)"

  sequence_example:
    frame_1: "오늘 새 포스트 올렸어요! 👇"
    frame_2: "[포스트 공유]"
    frame_3: "[투표] 어떤 주제가 더 궁금해요?"
    frame_4: "답변은 내일 스토리에서!"
```

## 캡션 작성 공식

### 구조별 템플릿

**1. 스토리텔링형**
```
[상황 설정]
3년 전, 저는 완전히 막막했어요.

[문제/갈등]
아무리 열심히 해도 결과가 안 나왔거든요.

[전환점]
그러다 한 가지를 바꿨는데...

[해결/교훈]
지금은 [결과]를 이룰 수 있었어요.

[CTA]
여러분도 비슷한 경험 있으시면 댓글로 알려주세요 💬
```

**2. 리스티클형**
```
[주제]가 어렵다고요?
이 3가지만 기억하세요 👇

1️⃣ [포인트 1]
→ [간단한 설명]

2️⃣ [포인트 2]
→ [간단한 설명]

3️⃣ [포인트 3]
→ [간단한 설명]

저장해두고 필요할 때 꺼내보세요 📌
```

**3. 질문/참여형**
```
솔직히 말해봐요 🙋

[질문]

저는 [본인 답변]인데,
여러분은 어떠세요?

댓글로 알려주세요!
A: [선택지 1]
B: [선택지 2]
```

**4. 교육/팁형**
```
💡 오늘의 [주제] 팁

[핵심 인사이트 한 문장]

왜 중요하냐면,
[이유 설명]

실천 방법:
✓ [액션 1]
✓ [액션 2]
✓ [액션 3]

이 팁이 도움됐다면 저장 📌
```

## 톤앤매너 가이드

```yaml
instagram_tone:
  voice:
    - 친근하고 대화체
    - 감성적, 스토리텔링
    - 영감을 주는
    - 진정성 있는

  language:
    - "~해요" 체 (존댓말이지만 부드럽게)
    - 이모지 적극 활용
    - 줄바꿈으로 가독성 확보
    - 구어체 표현 OK

  emoji_guide:
    frequency: "문단당 1-2개"
    placement: "문장 끝 또는 리스트 시작"
    avoid: "과도한 사용, 의미 없는 나열"
    recommended:
      - "💡" (팁/인사이트)
      - "👇" (아래 참조)
      - "📌" (저장 유도)
      - "✨" (강조)
      - "🙋" (질문/참여)

  formatting:
    - 첫 줄은 강력하게 (미리보기에 표시)
    - 2-3문장마다 줄바꿈
    - 리스트는 이모지 또는 숫자로
    - 해시태그는 캡션 하단 또는 첫 댓글
```

## 캡션 길이 가이드

| 콘텐츠 유형 | 권장 길이 | 목적 |
|-------------|----------|------|
| 단일 이미지 (일상) | 50-100자 | 빠른 소비 |
| 단일 이미지 (정보) | 200-500자 | 가치 전달 |
| 캐러셀 | 100-300자 | 슬라이드가 주력 |
| 릴스 | 50-150자 | 영상이 주력 |
| 스토리 (텍스트) | 20-50자 | 빠른 인지 |

ctas:

  comment:

    - "이 중에 해본 거 있으면 손 🙋"
