## 프롬프트 조합 예시

### 자동 프롬프트 생성 로직

```python
def generate_prompt(theme, image_type, subject):
    base = THEME_STYLES[theme]

    prompt = f"{base['style_prefix']} of {subject}, "
    prompt += f"{base['color_keywords']}, "
    prompt += f"{base['style_modifiers']}, "
    prompt += f"{CONSTRAINTS['no_text']}, "
    prompt += f"{SIZE_SPECS[image_type]}"

    return prompt
```

### 실제 조합 예시

**Theme:** AI/Tech + **Type:** Concept + **Subject:** 자동화 프로세스

```
"Abstract AI technology visualization of automation workflow process,
dark background with neon blue (#667EEA) and cyan (#00D9FF) glow,
geometric neural network patterns connecting workflow steps,
futuristic tech aesthetic, high contrast,
no realistic faces, no text, no watermarks,
professional developer presentation style,
800x600 resolution, high quality"
```
