## Key Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Target language | Japanese only | Pure Japanese audience targeting |
| Tone | Friendly friend + curator | Not corporate, not influencer. Knowledgeable friend |
| Profile position | Service operator | Transparent about building comparison site |
| Content focus | Procedure comparison & pricing | Data-driven, unique value proposition |
| Posting frequency | Daily 1-2 posts | Consistent presence without burnout |
| DM strategy | Warm outreach for user research | 2-week warm-up, max 15 DMs/day |

## Timeline

| Week | Phase | Focus |

| Week 3 | DM Start | 3-5 DMs/day, user research begins |
