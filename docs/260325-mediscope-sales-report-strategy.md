# MediScope 영업용 PPT 리포트 전략 설계

> 작성일: 2026-03-25
> 목적: 의료관광 병원 대상 SEO/GEO/AEO 컨설팅 영업 자동화 리포트 설계

---

## 1. 리포트 구조 / 목차 설계

### 설계 원칙: "공포 → 희망 → 행동" 3막 구조

B2B 영업 심리학에서 가장 효과적인 구조는 **Nancy Duarte의 Sparkline 패턴**과 **PAS(Problem-Agitate-Solution) 카피라이팅 공식**의 결합이다. 병원장/마케팅 담당자는 "현재 상태가 괜찮다"고 생각하는 경향이 강하므로, 리포트의 첫 번째 임무는 **현상 안주의 위험성을 데이터로 증명**하는 것이다.

### 감정 아크 (Emotion Arc)

```
현실 인식 → 위기감 → 경쟁 불안 → 해결 가능성 → 신뢰 구축 → 행동 촉구
(What Is)    (Pain)    (Fear)      (What Could Be)  (Trust)     (Urgency)
```

### 목차 구조 (전체 12-15 슬라이드)

| 막 | 슬라이드 | 심리 기제 | 목적 |
|:--:|:--------:|:---------:|:----:|
| **1막: 현실 직면** | 1-4 | 인지 편향 깨기 | "생각보다 심각하다" |
| **2막: 기회와 경쟁** | 5-8 | 손실 회피 + FOMO | "안 하면 이만큼 잃는다" |
| **3막: 해결과 행동** | 9-12 | 희망 + 긴급성 | "지금 시작하면 된다" |

### 왜 이 순서인가?

1. **앵커링 효과**: 첫 슬라이드에서 시장 규모(7.5조 원)를 보여주면, 이후 모든 "놓치고 있는 기회"가 크게 느껴진다
2. **손실 회피**: 사람은 같은 크기의 이익보다 손실에 2배 강하게 반응한다. "매월 XX명의 해외 환자를 놓치고 있습니다"가 "XX명을 더 유치할 수 있습니다"보다 강력하다
3. **대비 효과**: 경쟁사와의 직접 비교는 "나만 뒤처져 있다"는 불안을 유발하여 행동을 촉진한다
4. **목표 경사 효과**: Quick Win을 마지막에 배치하여 "이미 80% 왔다, 나머지만 하면 된다"는 느낌을 준다

## 2. PPT 슬라이드별 상세 설계

### 1막: 현실 직면 (슬라이드 1-4)

---

#### 슬라이드 1: 표지 / 후킹
- **제목**: "{병원명} 디지털 가시성 진단 리포트"
- **부제**: "해외 환자가 {진료과}를 검색할 때, {병원명}은 보이고 있을까?"
- **핵심 메시지**: 질문형 후킹으로 호기심 유발
- **시각 요소**:
  - 병원 로고 + MediScope 로고
  - 진단 일시, 대상 URL
  - 전체 종합 점수 대형 게이지 (A-F 등급, 색상 코딩)
- **카피라이팅 공식**: AIDA - Attention 단계
- **동적 데이터**: 병원명, 진료과, URL, 진단 날짜, 종합 등급

---

#### 슬라이드 2: 의료관광 시장 기회
- **제목**: "2024년, 한국 의료관광의 역사적 전환점"
- **핵심 메시지**: "117만 외국인 환자, 7.5조 원 시장 — 당신의 병원은 이 파이에서 얼마를 가져가고 있습니까?"
- **포함할 데이터/차트**:
  - 연도별 외국인 환자 추이 바 차트 (2019→2024: 50만→117만, 특히 코로나 이후 V자 반등)
  - 진료과별 비중 도넛 차트 (피부과 56.6%, 성형외과 11.4% 등)
  - 국적별 비중 (일본·중국 60%, 미국, 대만, 태국)
- **시각적 임팩트**: 117만 → 7.5조 원으로 이어지는 큰 숫자 강조
- **카피라이팅 공식**: Cost of Inaction 셋업 — "이 성장에 탑승하지 않으면?"
- **동적 데이터**: 해당 진료과의 외국인 환자 비중 하이라이트

---

#### 슬라이드 3: 종합 진단 결과 대시보드
- **제목**: "{병원명} 디지털 건강 점수: {종합등급}"
- **핵심 메시지**: "5개 영역 중 {N}개가 개선이 필요합니다"
- **포함할 데이터/차트**:
  - 5개 카테고리 레이더 차트 (기술 SEO, 성능, GEO/AEO, 다국어, 경쟁력)
  - 카테고리별 점수 + 등급 바 차트 (빨강/노랑/초록 색상 코딩)
  - 25개 진단 항목 중 통과/실패 비율 (아이콘: 체크/경고/X)
- **시각적 임팩트**:
  - 레이더 차트에서 "찌그러진 부분" = 약점이 한눈에 보임
  - 빨간색 영역에 시선이 집중되도록 디자인
- **카피라이팅 공식**: PAS - Problem 단계
- **동적 데이터**: 전 항목의 실시간 진단 점수

---

#### 슬라이드 4: 가장 심각한 문제 Top 3
- **제목**: "지금 즉시 주목해야 할 3가지 문제"
- **핵심 메시지**: "이 문제들은 해외 환자가 {병원명}을 찾지 못하게 만들고 있습니다"
- **포함할 데이터/차트**:
  - 심각도 순 카드 레이아웃 (빨간 경고 아이콘)
  - 각 문제별: 무엇이 문제인지 → 왜 문제인지 → 영향도 점수
  - 예시: "구조화 데이터 미적용 → AI 검색엔진이 병원 정보를 읽지 못함 → GEO 점수 0/100"
- **시각적 임팩트**: 빨간 배경의 경고 카드, 숫자로 영향도 표시
- **카피라이팅 공식**: PAS - Agitate 단계
- **동적 데이터**: 진단 결과에서 가장 낮은 점수 3개 항목 자동 추출

---

### 2막: 기회와 경쟁 (슬라이드 5-8)

---

#### 슬라이드 5: 경쟁 병원 대비 포지션
- **제목**: "{진료과} 상위 병원 대비 {병원명}의 위치"
- **핵심 메시지**: "동일 진료과 상위 병원들은 이미 {N}개 항목에서 앞서 있습니다"
- **포함할 데이터/차트**:
  - 레이더 차트: {병원명} vs 업계 평균 vs 상위 그룹 평균 (3개 레이어 오버레이)
  - 항목별 순위 테이블 (당신의 위치를 화살표로 표시)
  - "상위 병원 적용률" 바: "구조화 데이터 — 상위 10개 중 8개 적용" 형태
- **시각적 임팩트**:
  - 자신의 레이더 차트가 경쟁사보다 작게 보이는 시각적 충격
  - 빨간 영역(약점)에 "GAP" 라벨 표시
- **카피라이팅 공식**: Contrast Effect + Social Proof
- **동적 데이터**: 벤치마크 데이터 (있을 경우), 없으면 업계 평균 기준
- **법적 주의**: 경쟁 병원명 비공개, "동일 진료과 상위 그룹"으로 익명 처리

---

#### 슬라이드 6: 놓치고 있는 기회 비용
- **제목**: "현재 상태로 6개월이면, 이만큼의 기회를 잃습니다"
- **핵심 메시지**: "매월 약 {N}명의 잠재 해외 환자가 {병원명} 대신 경쟁 병원을 선택하고 있습니다"
- **포함할 데이터/차트**:
  - 기회비용 계산기 시각화:
    - 월간 예상 검색량 (해당 진료과 + 국가 키워드)
    - 현재 노출률 vs 개선 후 예상 노출률
    - 환자당 평균 진료비 × 예상 유치 증가분 = **월간 기회비용**
  - 6개월/12개월 누적 기회비용 워터폴 차트
- **시각적 임팩트**:
  - 큰 금액(원화)이 "흘러가는" 시각적 은유
  - 타이머/시계 아이콘으로 시간이 지날수록 손실 커지는 것을 표현
- **카피라이팅 공식**: Cost of Inaction + Loss Aversion
- **동적 데이터**: 진료과별 평균 진료비, 검색량 데이터 기반 추정치

---

#### 슬라이드 7: AI 검색 시대의 새로운 게임 (GEO/AEO)
- **제목**: "Google만이 아닙니다 — ChatGPT, Perplexity가 병원을 추천합니다"
- **핵심 메시지**: "AI가 {진료과}를 추천할 때, {병원명}은 언급되지 않고 있습니다"
- **포함할 데이터/차트**:
  - AI 검색 트렌드 그래프: AI 검색 비중 증가 추이
  - {병원명}의 GEO/AEO 점수 vs 업계 기준
  - "AI가 추천하는 병원이 되려면 필요한 것" 체크리스트 (통과/미통과)
  - 참고: HubSpot AEO Grader의 5대 평가 기준 — 감성 분석, 존재 품질, 브랜드 인지도, 점유율, 시장 포지션
- **시각적 임팩트**: ChatGPT/Perplexity/Gemini 로고와 함께 "추천 미포함" 빨간 표시
- **카피라이팅 공식**: Before-After-Bridge (현재 AI 미노출 → AI 추천 병원 → 방법)
- **동적 데이터**: GEO/AEO 세부 진단 5개 항목 점수

---

#### 슬라이드 8: 다국어 접근성 진단
- **제목**: "해외 환자는 영어로 검색합니다 — {병원명}은 준비되어 있나요?"
- **핵심 메시지**: "다국어 최적화 3개 항목 중 {N}개가 미비합니다"
- **포함할 데이터/차트**:
  - 다국어 지원 현황 매트릭스 (한국어/영어/일본어/중국어 × 페이지/메타태그/구조화데이터)
  - hreflang 태그 적용 여부
  - 언어별 검색 노출 시뮬레이션
  - 국적별 환자 비중과 언어 매칭 (일본 35%, 중국 25%, 영어권 15%)
- **시각적 임팩트**: 국기 아이콘 + 체크/X 매트릭스로 직관적 표현
- **카피라이팅 공식**: FAB (다국어 지원 Feature → 해외 노출 Advantage → 환자 유치 Benefit)
- **동적 데이터**: 다국어 진단 3개 항목 결과

---

### 3막: 해결과 행동 (슬라이드 9-12)

---

#### 슬라이드 9: Quick Win — 즉시 개선 가능한 3가지
- **제목**: "이 3가지만 고치면, 2주 내 변화가 시작됩니다"
- **핵심 메시지**: "전체 개선 전에, 지금 당장 실행할 수 있는 항목이 있습니다"
- **포함할 데이터/차트**:
  - 3개 Quick Win 카드:
    - 각각: 문제 → 해결법(1줄) → 예상 효과 → 난이도(쉬움/보통) → 소요 시간
  - 예시:
    - "메타 디스크립션 추가 → Google 검색 CTR 15-25% 향상 → 난이도: 쉬움 → 1일"
    - "의료기관 스키마 마크업 → AI 검색 인식률 향상 → 난이도: 보통 → 3일"
    - "Core Web Vitals 개선 → 이탈률 30% 감소 → 난이도: 보통 → 1주"
  - 개선 전/후 예상 점수 비교 바 (Before: D등급 → After: B등급)
- **시각적 임팩트**: 초록색 체크 카드, "쉬움" 뱃지로 실행 가능성 강조
- **카피라이팅 공식**: Goal-Gradient Effect — "이미 가까이 왔다"
- **동적 데이터**: 진단 결과에서 가장 쉽고 효과 큰 3개 항목 자동 추출

---

#### 슬라이드 10: 개선 로드맵
- **제목**: "3개월 로드맵: {등급} → A등급까지"
- **핵심 메시지**: "체계적인 단계별 개선으로 3개월 내 가시성이 2배 향상됩니다"
- **포함할 데이터/차트**:
  - 3단계 타임라인 간트 차트:
    - **1단계 (1-2주)**: Quick Win 실행 — 메타태그, 스키마, 속도 최적화
    - **2단계 (3-6주)**: 기술 SEO 완성 — 다국어, 사이트맵, 모바일
    - **3단계 (7-12주)**: GEO/AEO 최적화 — AI 가시성, 콘텐츠 전략
  - 각 단계별 예상 점수 향상 곡선 (현재 → Quick Win 후 → 완료 후)
- **시각적 임팩트**: 상승 곡선과 함께 각 마일스톤에 점수 변화 표시
- **카피라이팅 공식**: Before-After-Bridge (현재 → 목표 → MediScope가 연결)
- **동적 데이터**: 현재 점수 기반 예상 개선 곡선

---

#### 슬라이드 11: MediScope 소개 / 왜 우리인가
- **제목**: "MediScope: 의료관광에 특화된 디지털 가시성 파트너"
- **핵심 메시지**: "일반 SEO가 아닌, 의료관광 전문 SEO/GEO/AEO 컨설팅"
- **포함할 데이터/차트**:
  - 차별점 3개 카드:
    - 의료관광 특화: 25개 진단 항목이 모두 의료관광 맥락
    - GEO/AEO 포함: 기존 SEO 업체가 다루지 않는 AI 검색 최적화
    - 자동 모니터링: 월간 자동 리포트로 지속 추적
  - 성공 사례 1-2개 (수치 기반, 익명)
  - 서비스 패키지 개요 (상세는 별도 제안서)
- **시각적 임팩트**: 깔끔한 3컬럼 차별점 + 신뢰 아이콘
- **카피라이팅 공식**: Authority Bias + Social Proof

---

#### 슬라이드 12: CTA / 다음 단계
- **제목**: "다음 단계: 30분 무료 컨설팅"
- **핵심 메시지**: "이 리포트에서 발견된 문제를, 전문가와 함께 구체적인 액션 플랜으로 바꾸세요"
- **포함할 데이터/차트**:
  - 다음 단계 3개:
    1. 30분 무료 전화 컨설팅 예약
    2. 상세 진단 리포트(유료) 확인
    3. 맞춤 개선 제안서 수령
  - 담당자 연락처 + 캘린더 링크
  - 기한 설정: "이 리포트의 데이터는 {진단일} 기준입니다. 검색 환경은 매일 변합니다."
- **시각적 임팩트**: 큰 CTA 버튼 스타일, 캘린더 아이콘
- **카피라이팅 공식**: AIDA - Action 단계 + Scarcity (시간 제한성)
- **동적 데이터**: 담당자 정보, 캘린더 링크

## 3. 셀링 포인트 전략

### 3.1 핵심 심리 기제별 셀링 포인트

#### A. 돈으로 환산할 수 있는 기회 비용 (Loss Aversion)

| 메시지 | 데이터 근거 | 사용 슬라이드 |
|--------|------------|:------------:|
| "현재 상태로 6개월이면 약 {N}명의 해외 환자를 놓칩니다" | 진료과 월간 검색량 × 경쟁사 대비 CTR 차이 × 전환율 | 6 |
| "환자 1명당 평균 진료비 {N}만 원 기준, 연간 {N}억 원의 기회를 놓치고 있습니다" | 보건복지부 진료과별 외국인 환자 평균 진료비 | 6 |
| "AI 검색에서 노출되지 않으면, 2027년까지 전체 검색 트래픽의 40-60%를 잃게 됩니다" | IQVIA 리서치: AI 오버뷰가 유기 CTR 40-60% 감소 | 7 |

**사용법**: 절대 허수를 쓰지 않는다. "약", "추정" 표현을 반드시 포함하되, 계산 로직은 투명하게 보여준다. 신뢰도가 셀링 포인트의 전제 조건이다.

#### B. 경쟁사 직접 비교 (Social Proof + Contrast Effect)

| 메시지 | 데이터 근거 | 사용 슬라이드 |
|--------|------------|:------------:|
| "{진료과} 상위 10개 병원 중 8개가 이미 구조화 데이터를 적용했습니다" | 실제 크롤링 데이터 | 5 |
| "상위 그룹 평균 점수 {N}점 vs {병원명} {N}점" | MediScope 진단 결과 비교 | 5 |
| "경쟁 병원은 이미 영어/일본어/중국어 3개 언어를 지원합니다" | 다국어 크롤링 데이터 | 8 |

**사용법**: 반드시 익명 처리. "동일 진료과 상위 그룹", "업계 평균"으로 표현. 특정 병원명을 절대 언급하지 않는다.

#### C. 즉시 실행 가능한 Quick Win (Goal-Gradient + Activation Energy)

| 메시지 | 데이터 근거 | 사용 슬라이드 |
|--------|------------|:------------:|
| "이 3가지만 고치면 2주 내 Google 검색 노출이 개선됩니다" | 진단 결과 중 난이도 낮고 효과 높은 3개 | 9 |
| "메타 디스크립션 하나만 추가해도 CTR이 15-25% 올라갑니다" | 업계 벤치마크 | 9 |
| "현재 D등급 → Quick Win만으로 B등급 도달 가능" | 점수 시뮬레이션 | 9 |

**사용법**: Quick Win의 핵심은 "쉬움"이다. 기술적 난이도가 아니라 비즈니스 임팩트를 강조한다.

#### D. 시장 성장 / 트렌드 (Bandwagon + FOMO)

| 메시지 | 데이터 근거 | 사용 슬라이드 |
|--------|------------|:------------:|
| "2024년 외국인 환자 117만 명, 전년 대비 93.2% 증가" | 보건복지부 공식 통계 | 2 |
| "2024년 의료관광 경제효과 7조 5,039억 원" | 보건복지부 공식 통계 | 2 |
| "AI 검색 점유율이 2025년 이후 급격히 증가 중 — GEO/AEO가 새로운 표준" | 업계 리서치 | 7 |

### 3.2 의사결정자 유형별 맞춤 메시지

| 의사결정자 | 관심사 | 핵심 메시지 | 강조 슬라이드 |
|-----------|--------|------------|:------------:|
| **병원장/원장** | ROI, 경쟁 우위 | "경쟁 병원은 이미 하고 있다" + 기회비용 | 5, 6 |

| **경영지원팀** | 비용 대비 효과 | "월 {N}만 원 투자로 연 {N}억 원 기회" | 6, 10 |

## 4. 무료 프리뷰 vs 유료 상세 리포트 차별화

### 4.1 게이팅 전략: 3단계 퍼널

참고 모델: SEOptimer(즉시 점수 공개 → 상세는 이메일), HubSpot Website Grader(이메일 입력 후 리포트 제공), HubSpot AEO Grader(이메일 후 5개 차원 상세 분석)

```
[1단계: 무게이팅]        [2단계: 이메일 게이팅]      [3단계: 미팅 게이팅]
즉시 점수 확인            무료 프리뷰 리포트          유료 상세 리포트
(웹에서 바로 확인)        (PDF 다운로드)              (미팅 후 제공)
─────────────────────────────────────────────────────────────────────
URL 입력만으로            이메일 + 병원명 +           30분 컨설팅 미팅 후
종합 등급(A-F) 확인       진료과 입력 후              또는 유료 구독 후
                         8-10 슬라이드 PDF           전체 리포트 제공
```

### 4.2 무료 프리뷰에서 보여주는 것 (8-10 슬라이드)

| 슬라이드 | 포함 | 상세 수준 | "더 보고 싶다" 유발 전략 |
|:--------:|:----:|:---------:|:----------------------:|
| 1. 표지 | O | 전체 | 종합 등급만 — "왜 이 등급인지 알아보세요" |
| 2. 시장 기회 | O | 전체 | 시장은 크다는 것을 인지시키고, "당신의 몫은?" 질문 |
| 3. 종합 대시보드 | O | **카테고리 점수만** | 레이더 차트 보여주되, 개별 항목은 블러 처리 |
| 4. Top 3 문제 | **부분** | **문제만, 해결법 없음** | "이 문제의 해결 방법은 상세 리포트에서" |
| 5. 경쟁 비교 | **부분** | **업계 평균만** | "상위 병원과의 상세 비교는 상세 리포트에서" |
| 6. 기회 비용 | **부분** | **범위만 제시** | "정확한 금액 산출은 상세 리포트에서" |
| 7. Quick Win | **1개만** | 나머지 2개 블러 | "나머지 Quick Win과 실행 가이드는 상세 리포트에서" |
| 8. CTA | O | 전체 | "30분 무료 컨설팅 예약" or "상세 리포트 요청" |

**핵심 원칙**: Zeigarnik Effect (미완결 효과) — 문제는 보여주되, 해결법은 감춘다. "3개 중 1개만 보여줌"으로 나머지에 대한 궁금증을 유발한다.

### 4.3 유료 상세 리포트에서만 제공하는 가치

| 항목 | 무료 | 유료 | 가치 차별화 |
|------|:----:|:----:|:----------:|
| **25개 전체 진단 항목 상세** | 5개 카테고리 점수만 | 항목별 상세 분석 + 스크린샷 | 정확한 문제 파악 |
| **경쟁사 벤치마크 상세** | 업계 평균만 | 상위 5개 병원 익명 비교 | 포지셔닝 인사이트 |
| **기회비용 정밀 산출** | 범위(~억 원) | 월별 정밀 추정치 | 의사결정 근거 |
| **Quick Win 전체 + 실행 가이드** | 1개만 | 3-5개 + 단계별 가이드 | 즉시 실행 가능 |
| **3개월 개선 로드맵** | 없음 | 주 단위 실행 계획 | 체계적 개선 |
| **GEO/AEO 상세 분석** | 점수만 | AI별(ChatGPT/Perplexity/Gemini) 분석 | 미래 대비 |
| **다국어 SEO 상세** | 지원 여부만 | 언어별 최적화 가이드 | 타겟 시장별 전략 |
| **월간 모니터링 리포트** | 없음 | 매월 자동 발송 | 지속적 관리 |
| **전문가 컨설팅** | 없음 | 30분 1:1 리뷰 | 맞춤 전략 |

### 4.4 전환 유도 타이밍

```
[즉시 점수 확인]
    │
    ├─ 등급 A-B: "좋은 상태입니다. 더 나아질 수 있는 부분을 확인하세요" → 상세 리포트 유도
    │
    ├─ 등급 C: "개선 여지가 있습니다. 경쟁사 대비 위치를 확인하세요" → 프리뷰 리포트
    │             └─ 프리뷰 리포트 하단 CTA → 상세 리포트 or 미팅
    │
    └─ 등급 D-F: "즉각적인 개선이 필요합니다" → 프리뷰 리포트 + 즉시 미팅 제안
                   └─ 프리뷰 리포트에 "긴급" 라벨 → 미팅 CTA 강화
```

### 5.2 등급 산출 기준

| 등급 | 점수 범위 | 색상 | 해석 |
|:----:|:---------:|:----:|:----:|
| A | 90-100 | 초록 #22C55E | 우수 — 미세 조정만 필요 |
| B | 75-89 | 연두 #84CC16 | 양호 — 일부 개선 여지 |
| C | 60-74 | 노랑 #EAB308 | 보통 — 체계적 개선 필요 |
| D | 40-59 | 주황 #F97316 | 미흡 — 즉각 개선 필요 |
| F | 0-39 | 빨강 #EF4444 | 심각 — 긴급 대응 필요 |

### 5.3 기회비용 자동 계산 공식

```
월간_기회비용 = 월간_검색량 × (개선_후_CTR - 현재_CTR) × 전환율 × 환자당_평균_진료비

여기서:
- 월간_검색량: 진료과 + "korea" + 주요 언어 키워드 합산
- 현재_CTR: 포지션 기반 (10위 이하 = 1%, 5-10위 = 3%, 1-5위 = 8%)
- 개선_후_CTR: 상위 그룹 평균 CTR
- 전환율: 의료관광 업계 평균 (약 2-5%)
- 환자당_평균_진료비: 보건복지부 통계 기준

6개월_기회비용 = 월간_기회비용 × 6 × 성장_계수(1.05)
12개월_기회비용 = 월간_기회비용 × 12 × 성장_계수(1.10)
```

**주의**: 모든 추정치에는 "약", "추정" 표현을 반드시 포함하며, 계산 전제 조건을 각주로 명시한다.

## 6. 의료광고법 저촉 방지 가이드라인

### 6.1 적용 법령

- **의료법 제56조** (의료광고의 금지 등)
- **의료광고 심의에 관한 규칙** (보건복지부)
- **2024.12.30 개정 의료광고 가이드라인**

### 6.2 리포트 작성 시 절대 금지 사항

| 금지 유형 | 의료법 근거 | MediScope 리포트 적용 |
|----------|-----------|---------------------|
| **비교 광고** | 제56조 제2항 제4호: 다른 의료인의 기능/진료 방법과 비교하는 광고 | 특정 병원명을 절대 언급하지 않음. "동일 진료과 상위 그룹", "업계 평균"으로만 표현 |
| **과장 광고** | 제56조 제2항 제5호: 객관적 사실을 과장하는 광고 | 모든 수치에 "약", "추정" 명시. 출처를 각주로 표기 |
| **치료 효과 보장** | 제56조 제2항 제2호: 치료 효과를 오인하게 할 우려가 있는 광고 | "환자가 늘어납니다" → "검색 노출이 개선됩니다"로 표현. 진료 효과가 아닌 마케팅 효과만 언급 |
| **치료 경험담** | 제56조 제2항 제2호 | 환자 후기, 치료 전후 사진 등 일체 불포함 |
| **전문 병원 표기** | 제3조의5: 지정받지 않은 병원의 "전문" 표기 | "전문", "최고", "최초" 등 표현 사용 금지 |

### 6.3 경쟁사 점수 표현 방법

MediScope 리포트에서 경쟁사 데이터를 표현하는 안전한 방법:

#### 허용되는 표현
```
O "동일 진료과 상위 그룹 평균 점수: 78점"
O "업계 평균 대비 15점 낮습니다"
O "상위 10개 의료기관 중 80%가 구조화 데이터를 적용하고 있습니다"
O "귀 의료기관의 기술 SEO 점수는 업계 평균 이하입니다"
O "Hospital A, Hospital B, Hospital C" (완전 익명 코드)
```

#### 금지되는 표현
```
X "XX병원보다 SEO 점수가 낮습니다"
X "경쟁 병원 OO의 점수는 90점인데 귀 병원은 45점입니다"
X "강남의 유명 피부과 대비 뒤처져 있습니다"
X "최고의 SEO 점수를 달성하겠습니다"
X "이 치료로 외국인 환자가 2배 증가합니다" (치료 효과 암시)
```

### 6.4 안전한 리포트 문구 템플릿

### 6.5 리포트 하단 법적 고지문 (필수 삽입)

```
[법적 고지]
본 리포트는 웹사이트의 기술적 검색 최적화 상태를 분석한 것으로,
의료 서비스의 품질이나 치료 효과에 대한 평가가 아닙니다.
경쟁 분석에 사용된 데이터는 공개적으로 접근 가능한 웹사이트 기술 정보를 기반으로 하며,
특정 의료기관을 비교·비방하려는 의도가 없습니다.
모든 추정치는 업계 벤치마크에 기반한 예상값이며, 실제 결과는 다를 수 있습니다.
```

### 6.6 내부 검수 체크리스트

리포트 자동 생성 후, 아래 항목을 자동으로 검증하는 로직을 포함해야 한다:

- [ ] 특정 병원명이 포함되어 있지 않은가?
- [ ] "최고", "최초", "유일", "전문" 등 과장 표현이 없는가?
- [ ] "치료", "효과", "완치" 등 의료 효과 관련 표현이 없는가?
- [ ] 모든 수치에 "약", "추정", "예상" 한정어가 붙어 있는가?
- [ ] 출처가 명시되어 있는가?
- [ ] 법적 고지문이 포함되어 있는가?
- [ ] "보장", "확실", "반드시" 등 단정 표현이 없는가?
