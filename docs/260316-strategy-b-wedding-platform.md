# 전략 B안: 웨딩 플랫폼 (무료 청첩장 + 마켓플레이스)

> **권고: 단독 실행 비추천. A안으로 시장 검증 후, 월 100건+ 달성 시 전환 검토.**

## 핵심 컨셉

**"모바일 청첩장은 무료. 진짜 수익은 웨딩 생태계에서."**

Zola/The Knot 모델을 한국화. 인터랙티브 모바일 청첩장을 무료로 제공하여
대규모 트래픽(예비부부 + 하객)을 확보하고, 웨딩 관련 서비스로 수익화.

> **현실 체크**: Zola는 $600M+ VC 투자, The Knot은 20년+ 역사.
> 1인 창업자가 Day 1부터 플랫폼을 추구하는 것은 고위험 전략.

## 수익 모델

### 1차 수익: 청첩장 프리미엄 업셀

| 티어 | 가격 | 설명 |
|------|------|------|
| Free | 0원 | 기본 테마 3종, 모든 필수 기능, 하객 무제한, 광고 없음 |
| Pro | 4.9만원 | 프리미엄 인터랙티브 테마, 이스터에그, BGM, 커스텀 도메인 |

### 2차 수익: 웨딩 서비스 연결 (리드젠)

| 수익원 | 모델 | 비고 |
|--------|------|------|
| 스드메 벤더 프로필 | CPA (리드당 과금) | 에스크로 없이 리드 연결만 |
| 답례품 제휴 | 판매 수수료 (10~15%) | 외부 쇼핑몰 링크 |
| 신혼여행 패키지 | 제휴 수수료 | 여행사 링크 |

> **에스크로/결제 중개 배제**: 전자금융거래법 규제 + 개발 복잡도 회피.
> 리드 생성(CPA) + 제휴 링크 모델로 단순화.

## 플라이휠 구조

```
무료 청첩장 (고품질)
    ↓
사용자 유입 (예비부부)
    ↓
하객 트래픽 (1인당 100~200명)
    ↓
벤더 프로필/제휴 링크 노출
    ↓
CPA/제휴 수익
    ↓
더 좋은 무료 청첩장 투자
```

## 수익 시뮬레이션 (보수적)

### Phase 1 (0~6개월): A안으로 시장 검증

이 기간은 A안과 동일. 유료 청첩장으로 수익 확보 + 트래픽 데이터 축적.

### Phase 2 (6~12개월): 무료 전환 + 벤더 연결 시작

| 지표 | 보수적 | 낙관적 |
|------|--------|--------|
| 월 신규 청첩장 | 200건 | 500건 |
| Pro 전환율 | 5% | 10% |
| Pro 매출 | 49만원 | 245만원 |
| 벤더 리드 수익 | 50~100만원 | 200~400만원 |
| **월 합계** | **~100~150만원** | **~445~645만원** |

> 범위 표기: 벤더 리드 수익의 불확실성이 높으므로 하한~상한으로 표기.

> Pro 전환율 5%는 프리미엄 프리미엄 모델 평균치. 20%는 비현실적.

### 비용 구조 (Phase 2 월)

| 항목 | 비용 |
|------|------|
| 인프라 (Free 사용자 증가) | 20~30만원 |
| 마케팅 | 30~50만원 |
| PG 수수료 | 2~10만원 |
| **합계** | **~70만원** |

## Cold Start 해결 전략

### 수요 측 (예비부부)

| 전략 | 실행 방법 |
|------|-----------|
| A안에서 자연 전환 | A안으로 쌓은 사용자/트래픽 기반 |
| Viral 루프 | 하객 노출 → 다음 예비부부 유입 |
| SEO 콘텐츠 | 결혼준비 가이드, 청첩장 트렌드 블로그 |
| 커뮤니티 | 웨딩 카페/커뮤니티 사례 공유 |

### 공급 측 (벤더)

| 전략 | 실행 방법 |
|------|-----------|
| Founding Partner | 5~10개 벤더 12개월 무료 리스팅 |
| 지인 네트워크 | 웨딩 업계 지인 소개 |
| 트래픽 증명 | 월 방문자 데이터로 ROI 제시 |
| 무거래 시작 | 리뷰/디렉토리로 시작, 거래 기능은 나중에 |

## 한국 마켓플레이스 경쟁 분석

| 서비스 | 특징 | 우리 차별점 |
|--------|------|-------------|
| 웨딩북 | 벤더 디렉토리 + 커뮤니티 | 청첩장→마켓 퍼널 없음 |
| 공여사들 | 결혼준비 정보 | 벤더 연결 약함 |
| 더웨딩 | 벤더 매칭 | 청첩장 기능 없음 |
| 네이버 카페 | 후기 커뮤니티 | 플랫폼 아님 |

**우리의 유일한 차별점**: 청첩장 → 하객 트래픽 → 벤더 연결 퍼널. 이것이 없으면 기존 플랫폼과 경쟁 불가.

## 장점

- "무료"로 시장 진입 장벽 제거
- 네트워크 효과 가능성
- 높은 LTV (웨딩 전체 여정)
- Viral 극대화

## 리스크

| 리스크 | 심각도 | 대응 |
|--------|--------|------|
| 초기 무수익 기간 | 🔴 높음 | A안으로 먼저 수익 확보 후 전환 |
| 1인 마켓플레이스 구축 불가 | 🔴 높음 | 에스크로 배제, 리드젠만. Phase 2부터 팀 필요 |
| 대형 플랫폼 진입 | 🔴 높음 | 청첩장→마켓 퍼널이 유일한 해자 |
| 벤더 영업 리소스 | 🟡 중간 | Founding Partner + 인바운드 |
| 법적 이슈 (중개) | 🟡 중간 | 직접 중개 회피, CPA/제휴 링크만 |

## 실행 경로 (A안 → B안 전환)

| 단계 | 기간 | 조건 |
|------|------|------|
| A안 운영 | 0~6개월 | 유료 청첩장 판매 |
| 전환 검토 | 6개월 | 월 100건+ AND 벤더 파트너십 기회 |
| 무료 전환 + 리드젠 | 6~12개월 | Free 티어 확대, 벤더 프로필 시작 |
| 풀 마켓플레이스 | 12개월+ | 팀 구성 후 or 투자 유치 후 |

### Kill Criteria

- A안에서 6개월 후 월 50건 미달 → B안 전환 불가, A안 유지
- B안 전환 후 3개월간 벤더 수익 0 → 리드젠 모델 폐기, A안 복귀
