# 전략 A안: 단일 가격 + AI 블록 에디터 청첩장

## 핵심 컨셉

**"4.9만원 하나로 모든 기능. AI가 원하는 대로 커스터마이징."**

복잡한 티어/코인 없이 단일 가격. 블록 에디터로 섹션 내부 레이아웃까지 자유 편집.
AI 크레딧으로 스타일/디자인 수정을 원하는 만큼 적용.

## 수익 모델

### 전환 메커니즘: Build Free, Pay to Publish

블록 에디터로 무료 제작 → 공유 링크 발행 시 4.9만원 결제.
"다 만들어놨는데, 결제만 하면 바로 카카오톡으로 보낼 수 있어요."

### 가격 구조

| 항목 | 가격 | 설명 |
|------|------|------|
| **청첩장 발행** | **4.9만원** (1회) | 전체 기능 포함, 하객 무제한 |
| AI 크레딧 | 기본 10회 포함 | 스타일/레이아웃 AI 수정 |
| AI 크레딧 추가 | 10회 / 1,900원 | 필요 시 추가 구매 |
| 커스텀 도메인 | 1,900원 | 고객이 직접 도메인을 구매/소유한 경우에만 DNS 연결 지원 (CNAME 안내 + SSL 자동 발급) |

### 4.9만원에 포함되는 모든 것

- 블록 에디터 (섹션 드래그앤드롭 + 내부 레이아웃 편집)
- 전체 테마 선택 (botanical, modern, glitch 등)
- 갤러리 (30장)
- RSVP (하객 무제한)
- 실시간 방명록
- Guest Snap (하객 사진 업로드)
- 계좌 정보 + 카카오페이 연동
- 위치/교통 + 지도 연동
- 파티클 효과 + BGM
- AI 커스터마이징 10회
- 카카오톡 공유 최적화 (동적 OG 이미지)

### AI 크레딧 사용 예시

| 사용 예 | 크레딧 소모 |
|---------|-------------|
| "이 섹션 색감을 따뜻한 톤으로" | 1회 |
| "인트로 애니메이션을 벚꽃 테마로" | 1회 |
| "갤러리를 그리드에서 슬라이드로 변경" | 1회 |
| "전체 폰트를 세리프체로" | 1회 |
| "Hero 이미지 레이아웃을 세로 분할로" | 1회 |

> 대부분의 사용자는 기본 10회로 충분. 디테일 집착형 사용자만 추가 구매.

## 블록 에디터 상세

### 에디터 구조

```
[캔버스] ← 실시간 프리뷰
│
├── [섹션 목록] (드래그앤드롭)
│   ├── Hero          ← ON/OFF, 순서 변경
│   ├── 인사말        ← ON/OFF, 순서 변경
│   ├── 커플소개      ← ...
│   ├── 인터뷰
│   ├── 갤러리
│   ├── 웨딩정보
│   ├── 위치/교통
│   ├── 계좌
│   ├── 방명록
│   ├── Guest Snap
│   ├── 영상
│   └── 푸터
│
├── [섹션 내부 편집]
│   ├── 텍스트 직접 수정
│   ├── 이미지 교체/크롭
│   ├── 레이아웃 변형 (프리셋 선택 or AI)
│   └── 색상/폰트 조절
│
└── [AI 어시스턴트] (크레딧 소모)
    └── 자연어 입력 → 스타일/레이아웃 자동 변경
```

### 블록 에디터 기술 스택 (검토 필요)

| 옵션 | 특징 | 난이도 |
|------|------|--------|
| 직접 구현 (React DnD + 커스텀) | 완전한 제어, 최적화 가능 | 높음 |
| Plate.js / Editor.js 기반 확장 | 블록 에디터 프레임워크 활용 | 중간 |
| GrapeJS 기반 | 웹 빌더 프레임워크, 드래그앤드롭 내장 | 중간 |

> Phase 1 MVP: 섹션 드래그앤드롭 + 기본 편집 (React DnD)
> Phase 2: 섹션 내부 자유 편집 + AI 커스터마이징

## 시장 포지셔닝

```
                      가격
                       ↑
         ~8만원대      │  ● 시중 프리미엄 유료 (기능 제한적)
                       │
       4.9만원 ────────│─────● 우리 (전체 기능 + AI + 블록에디터)
                       │
         ~2만원대      │  ● 시중 저가 유료
                       │
                 0원   │ ● 바른손 무료 (템플릿 고정)
                       └──────────────────────────→ 커스터마이징 자유도
                          낮음              높음
```

**마케팅 포지션**: 바른손 무료보다 확실히 특별하고, 비싼 서비스보다 합리적인 **스위트 스팟**.
