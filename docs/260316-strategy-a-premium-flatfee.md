# 전략 A안: 프리미엄 인터랙티브 청첩장 (1회성 정액제)

## 핵심 컨셉

**"남들과 다른, 특별한 모바일 청첩장을 합리적인 1회 결제로"**

기존 서비스들의 "무료 but 천편일률적" 또는 "코인제 but 불투명한 과금"과 차별화.
투명한 1회 정액제로 하객 수 무제한, 프리미엄 인터랙티브 기능을 제공.

## 수익 모델

### 전환 메커니즘: Build Free, Pay to Publish

사용자가 무료로 청첩장을 완성한 뒤, 공유 링크 발행 시 결제.
"다 만들어놨는데, 결제만 하면 바로 카카오톡으로 보낼 수 있어요."

### 가격 티어

| 티어 | 가격 | 핵심 기능 | 타겟 |
|------|------|-----------|------|
| Free | 0원 | 전체 기능 체험, 프리뷰 가능, 공유 불가 | 체험/제작 |
| Basic | 3.9만원 | 테마 3종, 갤러리 5장, 계좌/지도, RSVP 50명 | 가성비 |
| Premium | 7.9만원 | 인터랙티브 테마, 갤러리 30장, 무제한 RSVP, 방명록, Guest Snap, 파티클, BGM | 핵심 타겟 |

> **2티어 단순화 결정 근거**: Deluxe(14.9만원) 티어는 1차 리뷰에서 가격 정당화 어려움 지적.
> Basic/Premium 2단계로 단순화하고, 커스텀 도메인 등은 개별 애드온으로 제공.

### 애드온

| 옵션 | 가격 | 비고 |
|------|------|------|
| 커스텀 도메인 | 1.9만원 | 선택 옵션 |
| 추가 갤러리 (+20장) | 1.9만원 | Basic 업그레이드용 |
| 보이스오버 | 2.9만원 | Phase 2 |

### 티어 간 업셀 설계

Basic → Premium 전환 동기:
- RSVP 50명 제한 (하객 100명+ 일반적 → 자연스러운 업셀)
- 갤러리 5장 vs 30장 (실질적 용량 차이)
- 방명록/Guest Snap 미포함 (있으면 좋은 기능 → 실사용 후 필요성 체감)
