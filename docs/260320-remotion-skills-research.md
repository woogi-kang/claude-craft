## 1. Remotion Skills 개요

[remotion-dev/skills](https://github.com/remotion-dev/skills)는 **Remotion(React 기반 프로그래매틱 영상 제작 프레임워크)**의 Agent Skills 레포로, AI 코딩 어시스턴트(Claude Code, Codex, Cursor 등)에게 Remotion API의 베스트 프랙티스를 가르치는 **28개+ 모듈형 규칙 파일**이다.

- GitHub Stars: 2,242+
- 2026년 1월 출시, **8주 만에 150K 설치** 달성 (skills.sh 전체 5위, 영상 분야 1위)
- 공식 X 데모 영상이 **600만 뷰** 기록
- 설치: `npx skills add remotion-dev/skills`

### 작동 방식

1. 사용자가 자연어로 영상을 설명
2. AI가 React/TypeScript 코드 생성
3. Remotion이 렌더링하여 영상 파일 출력

### 라이선싱

- 개인/비영리/3인 이하 기업: **무료**
- 4인 이상 기업: **상용 라이선스 필요**
- 외부 클라이언트 대상 영상 생성 서비스: 상용 라이선스 필요

## 5. 공식 리소스

### SaaS 스타터킷

[Prompt to Motion Graphics SaaS Starter Kit](https://www.remotion.dev/docs/ai/ai-saas-template):
- **스택**: Next.js + OpenAI API + Remotion Player + AWS Lambda (옵션)
- **기능**: 채팅 인터페이스, 실시간 프리뷰, 자동 리트라이, 코드 스트리밍
- **설치**: `npx create-video@latest --template prompt-to-motion-graphics`

### Prompt Gallery

[Remotion Prompt Gallery](https://www.remotion.dev/prompts/) — AI 영상 생성 프롬프트 예제 모음

### 경쟁 환경

| 도구 | 특징 | 차별점 |
|------|------|--------|
| **Remotion Skills** | 코드 기반, 결정론적 출력 | 같은 프롬프트 = 같은 결과, B2B 자동화에 적합 |
