## 📱 1. 출발 전 준비 (Pre-Cruise Checklist)

### 앱 & 온라인 체크인
- **Virgin Voyages 앱** 다운로드 → 모든 것(체크인, 룸서비스, 식당 예약, 쇼 예약)에 사용
- 동행자가 다른 캐빈이면 앱 > Messenger > Contacts에서 **예약 연결** (함께 식당 예약 가능)
- **출항 2주 전**까지 온라인 체크인 완료 (여권 사진, 셀피, 도착 시간 선택)
- **출항 24시간 전**: 앱에 Health Check 폼 작성 → 디지털 보딩패스 생성

### 식당 예약 전략
- Virgin Voyages에는 메인 다이닝룸이 없음 → **20개 이상 레스토랑 전부 무료** (뷔페 없음!)
- 예약 오픈 시간: **마이애미 시간 자정**
- 예약 윈도우: RockStar Suite 120일 전 / Premium·Legacy 60일 전 / Essential 45일 전
- 승선 전에는 레스토랑당 1회만 예약 가능
- **The Wake, Razzle Dazzle 브런치**는 디너 예약 제한에 포함 안 됨
- 걱정 마세요: 전체 좌석의 **약 2/3는 선상에서만 예약 가능**하도록 홀딩됨

### 필수 서류
- **여권 필수** (Virgin Voyages는 미국 내 크루즈도 여권 요구, 출생증명서 불가)
- 여권 유효기간 **6개월 이상** 남아야 함
- 비자 필요 여부 확인 (자메이카·케이맨 제도는 대부분 무비자)
- 여권은 **반드시 기내 수하물**에! (위탁하면 객실 배송까지 2시간+ 소요)

## ⚓ 2. 승선일 (Embarkation Day - Miami Terminal V)

### 마이애미 Terminal V (718 N Cruise Blvd)
- **정시 도착**: 앱에서 선택한 Terminal Arrival Time에 맞춰 도착. 일찍 가도 밖에서 대기해야 함
- **주차**: Garage G가 가장 가까움 ($25/박, 예약 불가)
- **짐 드롭**: 오전 9시부터 포터에게 큰 짐 맡김 가능 → 캐빈 번호 적은 메모 준비!
- **RockStar 탑승자**: 우측 "Palm Way" 입구 → VIP 라운지 이용

### 승선 후 첫 2시간 (가장 중요!)
1. **비행기 모드 ON → 선상 Wi-Fi 연결** (앱에서 활동/식당 예약은 선상 Wi-Fi 연결 후에만 가능!)
2. **즉시 예약할 것**:
   - 피트니스 클래스 (번지, 스핀 등 → 무료지만 금방 마감!)
   - 스파 Thermal Suite 패스 (기항일에 더 저렴)
   - 놓친 식당 예약
3. **안전 훈련 (Muster Drill)**: 캐빈 TV 또는 앱에서 안전 영상 시청 → 지정된 집합 장소에서 밴드 스캔
4. **점심**: The Galley 또는 The Pizza Place에서 간단히 식사
5. **Let's Set Sail 파티**: 보통 오후 4:30쯤 풀 데크에서 출항 파티!

## 💰 6. 돈 관리 & 절약 팁

### 결제 시스템
- **The Band** = 룸키 + 신분증 + 지갑 (탭 결제)
- 선상 완전 **무현금** → 출항 전 신용카드 연결 (임시 홀딩 $250)
- 결제 순서: **Bar Tab** → **Sailor Loot** → 신용카드 (자동 전환)

### Bar Tab (음료 크레딧)
- 프리미엄 음료 전용 (알코올, 스페셜티 커피, 주스, 스무디)
- **사전 구매 시 보너스 크레딧** (예: $300 구매 → $50 보너스)
- 다른 사람 음료도 사줄 수 있음
- 비치클럽에서도 사용 가능
- ⚠️ **환불 불가, 남기면 소멸**

### Sailor Loot (일반 크레딧)
- 스파, 엑스커전, 쇼핑, 빙고, 프리미엄 다이닝 업그레이드에 사용
- ⚠️ **환불 불가, 남기면 소멸**

### 절약 해킹
| 팁 | 절약 |
|-----|------|
| 와인 2병 가져가기 (기내 수하물) | 레스토랑 코르키지 $25 절약 |

## 🚢 7. 하선일 (Disembarkation Day)

### 마지막 밤
- **Bar Tab & Sailor Loot 전부 소진** (환불 불가!) → 음료, 쇼핑, 스파에 사용
- 짐 드롭: 큰 짐을 **밤 10시까지** 캐빈 문 앞에 놓기 → 세관 통과 후 Terminal V에서 수령
- 색상별 짐태그 = 하선 시간대
- **셀프 하선**: 짐을 직접 들고 내리면 **오전 7:30~7:45**에 바로 하선 가능

### 당일 아침

- 세관: Terminal V **안면 인식** 시스템으로 빠르게 통과 (여권은 혹시 몰라 소지)
