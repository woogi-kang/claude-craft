# CheckYourHospital AEO 제품 스펙

## 제품 비전

**"AI가 추천하는 병원이 되십시오"**

병원 홈페이지 진단 → AI 추천 시뮬레이션 → AEO 콘텐츠 자동 생성 → 소셜 자동 발행

---

## 서비스 구조

```
진단 (Diagnose) → 생성 (Create) → 확산 (Distribute) → 모니터링 (Monitor)
```

### 1. 진단 엔진

#### 1-1. AI 추천 시뮬레이션 (신규 핵심 기능)

**플로우:**
```
병원 URL 입력
  → 크롤링 + DB 조회 → 주력 시술 3~10개 자동 추출
  → 시술별로 4개 AI에 동시 질문:
    "[지역]에서 [시술] 잘하는 병원 추천해줘"
  → 결과 매트릭스 표시
```

**AI 엔진:**
- ChatGPT (OpenAI API)
- Gemini (Google API) — 기존 구현 확장
- Claude (Anthropic API)
- Perplexity (기존 구현 확장)

**결과 형태:**
```
"강남에서 코성형 잘하는 곳 추천해줘"

  ChatGPT    Gemini     Claude     Perplexity
  1. A성형    1. B성형    1. A성형    1. C성형
  2. B성형    2. C성형    2. D성형    2. A성형
  3. C성형    3. A성형    3. B성형    3. E성형

  원장님 병원: 0/4 AI 추천 ❌
  경쟁사 A성형외과: 3/4 AI 추천 ✅
```

#### 1-2. 기존 24개 항목 진단 (유지)

기술 SEO 10개 + 성능 5개 + GEO/AEO 6개 + 다국어 3개

#### 1-3. 신규 AEO 체크 항목 (3개 추가)

- `check_llms_txt()` — AI 전용 사이트맵 파일 존재 여부
- `check_ai_robots_rules()` — robots.txt AI 크롤러 허용 여부
- `check_ai_meta_tags()` — AI 관련 메타 태그 설정 여부

### 2. 콘텐츠 생성 엔진

#### 2-1. AEO 콘텐츠 자동 생성

**입력:** CYH DB의 시술 정보 + 병원 데이터
**출력:** 병원 홈페이지에 삽입할 수 있는 AEO 최적화 콘텐츠

| 콘텐츠 유형 | 설명 | AEO 효과 |
|------------|------|----------|
| 시술별 FAQ | "코성형 회복 기간은?" Q&A 5~10개 세트 | AI가 답변에 직접 인용 |
| E-E-A-T 프로필 | 의사 경력, 자격증, 전문분야 구조화 | AI 신뢰도 신호 강화 |
| Schema.org 코드 | MedicalClinic, Physician, FAQ 마크업 | AI가 구조화 데이터 우선 참조 |
| 시술 상세 설명 | 소요시간, 회복기간, 가격대, 주의사항 | AI가 비교 정보로 활용 |

#### 2-2. AI 이미지 생성

| 이미지 유형 | 생성 방식 | 용도 |
|------------|----------|------|
| 시술 인포그래픽 | SVG → PNG (Gemini) | 시술 비교, 가격표, 프로세스 |
| 시술 일러스트 | GPT-4o 이미지 생성 | 시술 설명 보조 이미지 |
| 의사 프로필 카드 | 템플릿 기반 | SNS 소개용 |
| 배너/커버 | GPT-4o / Nano Banana | 병원 브랜딩 |

### 3. 소셜 발행 엔진

**기반:** social-outreach 프로젝트 worker 재사용

#### 3-1. 플랫폼별 자동화

| 플랫폼 | 방식 | 포맷 |
|--------|------|------|
| X | Playwright 브라우저 자동화 | 텍스트 280자 + 이미지 |
| Threads | Playwright 브라우저 자동화 | 텍스트 + 이미지 |
| Instagram | Playwright 브라우저 자동화 | 이미지 중심 + 캡션 |

#### 3-2. 콘텐츠 유형별 배분

| 유형 | 비율 | 예시 |
|------|------|------|
| 전문 (Expert) | 50% | "코성형 후 붓기 관리 3단계" |
| FAQ | 20% | "Q. 코성형 회복기간은? A. 평균 2주" |
| 일상/케이스 | 20% | "오늘 상담받은 이야기", 시술 과정 소개 |
| 소개/프로모션 | 10% | "이달의 특가", "신규 장비 도입" |

#### 3-3. 스케줄링

```yaml
default:
  posts_per_day: 2-3
  best_hours: [9, 12, 18, 21]  # 환자 검색 피크 시간
  platforms:
    instagram: 1/day
    threads: 1/day
    x: 1-2/day
```

### 4. 모니터링

- 월 2~4회 자동 재진단
- AI 추천 변동 추적 (진입/이탈 알림)
- 경쟁사 동향 모니터링
- 소셜 포스팅 성과 분석

## 마케팅 플로우 (6단계 설득)

### 1. 공감
> "원장님이 자신 있는 시술, 환자들이 그 시술을 찾을 때 우리 병원이 추천되고 있나요?"

### 2. 위기감
> "경쟁사 A성형외과는 이미 3/4개 AI에서 추천받고 있습니다. 지금 선점하지 않으면 격차는 벌어집니다."

### 3. 현실 진단
> URL만 입력하면 5분 안에 결과가 나옵니다.
> 원장님의 주력 시술로 4개 AI에 직접 검색해서 결과를 보여드립니다.

### 4. 패러다임 전환 (SEO → AEO)
> SEO: 검색 결과 10개 링크 중 하나가 되는 싸움
> AEO: AI가 직접 추천하는 1~3개 안에 드는 싸움

### 5. 대응 방법
> 진단 → 콘텐츠 생성 → 소셜 확산 → 모니터링
> 3단계로 AI가 추천하는 병원이 됩니다.

### 6. 행동 유도
> "지금 무료로 확인하세요. 같은 지역 병원 중 AI 추천 순위를 바로 알려드립니다."

## 결제 유도 포인트

"진단 이후 경쟁사 상황이 바뀌었습니다" → 재방문 유도
