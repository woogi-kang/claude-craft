# Claude Code 2.0 세미나 PPT 구조 설계서

## 개요
- **발표 제목:** Claude Code 2.0 완벽 활용 가이드
- **부제:** AI 시대, 개발 생산성을 폭발시키는 방법
- **청중:** 개발팀 전체 (기술팀)
- **프레임워크:** SCQA + 순차적 구조
- **총 슬라이드:** 25장
- **예상 발표 시간:** 40분

## 청중 분석
- **유형:** 기술팀 (개발자)
- **관심사:** How-to, 구현 방법, 실무 적용
- **선호 구조:** 순차적, 데모/예시 중심
- **슬라이드 특성:** 다이어그램, 코드 예시, 실전 팁

## 핵심 메시지 (3개)
1. **Context는 자산이다** - 모니터링하고 관리하라
2. **병렬 처리로 시간을 사라** - 2-4개가 Sweet Spot
3. **Skills로 팀 지식을 자산화하라** - Git에 영구 보존
